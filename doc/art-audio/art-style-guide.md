# 美术风格指南

> 本文档定义《Immortality》的美术规范，包括 2.5D 混合模式的技术规格、色彩体系、Sprite 规格和比例标准。

## 1. 2.5D 混合模式规范

### 1.1 渲染层级

从摄像机视角由远到近：

```
天空盒 (Skybox)
  ↓
3D 场景（地形、建筑、植被）
  ↓
3D 粒子特效（技能、环境效果）
  ↓
2D Billboard Sprite（角色、怪物、NPC）
  ↓
UGUI Canvas（HUD、界面）
```

### 1.2 摄像机

- **视角**：斜 45° 俯视（Orthographic 或弱 Perspective）
- **旋转**：允许玩家在 ±30° 范围内轻微旋转摄像机
- **缩放**：允许 2 档缩放（近/远）
- **参照**：仙境传说 RO 的经典视角

### 1.3 Billboard Sprite 规格

角色和怪物使用 2D Sprite 贴在 Billboard 上：

| 规格项 | 标准 | 说明 |
|--------|------|------|
| 方向数 | 8 方向 | 上/下/左/右/左上/右上/左下/右下 |
| 帧动画 | 每方向 4-8 帧 | 待机/行走/攻击/受击/死亡 |
| Sprite 尺寸 | 256×256 px（角色）/ 128×128 px（怪物）| 单帧像素尺寸 |
| Sprite Sheet | 横向排列，每行一个方向 | 导入后在 Unity 中切割 |
| 透视处理 | 站立 Sprite 随摄像机轻微倾斜 | 通过 Billboard 脚本控制 |
| 阴影 | 角色脚下投射 Blob Shadow | 不使用真实阴影，保持 RO 风格 |

#### 8 方向命名规范

```
     NW(6)    N(0)    NE(4)
       \       |       /
 W(1) --- 角色 --- E(5)
       /       |       \
     SW(7)    S(2)    SE(3)
```

方向索引 0-7，对应 Sprite Sheet 的行号。

#### 动画状态命名

```
charactername_idle_N      # 待机-北
charactername_walk_N      # 行走-北
charactername_attack_N    # 攻击-北
charactername_hit_N       # 受击-北
charactername_die_N       # 死亡（单方向）
```

### 1.4 3D BOSS 规格

特殊 BOSS 使用真正的 3D 模型：

| 规格项 | 标准 |
|--------|------|
| 面数 | 3,000-10,000 tris |
| 贴图 | 1024×1024，PBR 材质 |
| 动画 | 待机/行走/攻击×3/受击/死亡/特殊技能 |
| 骨骼 | Humanoid Rig（人形）或 Generic（异形） |
| 用途 | 剧情关键 BOSS、世界 BOSS |

### 1.5 3D 场景规格

| 规格项 | 标准 | 说明 |
|--------|------|------|
| 风格 | 低多边形（Low Poly） | 面数控制在合理范围 |
| 地形 | Unity Terrain 或简单 Mesh | 程序化生成 + 手工调整 |
| 建筑 | 低多边形模型 | 可用 TJGenerators 生成或手工搭建 |
| 植被 | Billboard 植被 | 远景用 Billboard，近景用低多边形 |
| 灯光 | Directional Light + 环境光 | 简单烘焙，不做实时全局光 |

## 3. UI 视觉规范

### 3.1 界面尺寸

- **设计分辨率**：1920×1080（横屏）
- **Canvas Scaler**：Scale With Screen Size，参考分辨率 1920×1080
- **安全区域**：边缘留 20px

### 3.2 字体

| 用途 | 字体 | 大小 | 颜色 |
|------|------|------|------|
| 标题 | 思源宋体 Bold | 28-36px | 仙金 `#D4A847` |
| 正文 | 思源黑体 Regular | 16-18px | 灵白 `#F0F0E8` |
| 数值 | 等宽数字 | 14-16px | 灵白 |
| 说明 | 思源黑体 Light | 12-14px | 灰色 `#AAAAAA` |

### 3.3 面板风格

- **背景**：半透明深色（墨青 90% 透明度）+ 圆角 8px
- **边框**：1px 仙金色实线
- **按钮**：墨青底 + 仙金边框 + 悬停变亮
- **图标**：64×64 或 48×48，带品质边框

## 4. 比例标准

### 4.1 角色比例

| 类型 | 头身比 | 说明 |
|------|--------|------|
| 玩家角色 | 2.5-3 头身 | Q版风格，可爱但不幼稚 |
| 普通怪物 | 1-4 头身 | 根据怪物类型变化 |
| NPC | 2.5-3 头身 | 与玩家角色一致 |
| 特殊 BOSS | 4-6 头身 | 偏写实，增强压迫感 |

### 4.2 场景比例

- **1 Tile = 1×1 米**（3D 场景的基本单位）
- **角色高度**：约 1.6 米（2 Tile）
- **建筑高度**：2-4 米
- **树木高度**：3-6 米

## 5. 美术风格参考

> 以下参考用于指导 AI 生成时的提示词方向，最终风格以设计确认为准。

### 5.1 风格关键词

- 国风写意、修仙、仙气缭绕
- 色彩偏冷色调（墨青为主），暖色点缀（仙金）
- 2D 角色精致（类似 RO 但质量更高），3D 场景低多边形
- 特效华丽但不花哨，注重灵气流动感

### 5.2 参考游戏

| 游戏 | 参考点 |
|------|--------|
| 仙境传说 RO | 2.5D 混合模式、Billboard 角色、视角 |
| 原神 | UI 风格、色彩体系、图标设计 |
| 鬼谷八荒 | 修仙题材、境界展示、修炼 UI |
| 太吾绘卷 | 国风写意、色彩感觉 |

### 5.3 不要的风格

- 纯像素风（不是复古像素游戏）
- 写实风格（不适合修仙题材 + AI 生成成本高）
- 日式动漫萌系（保持中国修仙特色）
