# 第7层：世界系统规则书

> 版本：v1.0 | 最后更新：2026-07-16

---

## 概述

第7层定义**这个世界本身怎么运转**。基于前6层已建立的玩家属性（L1）、行动（L2）、战斗（L3）、成长（L4）、经济（L5）、社交（L6）的完整闭环，将地图架构、资源分布、灵气流动、天道法则和阵营系统编织成修仙世界的物理层与规则层。

**设计原则**：
1. **无限深度，有限存储**：三层地图 + 种子驱动，服务器仅存区域元数据（<100KB），地形由客户端 Perlin Noise 生成。
2. **同一世界，不同视野**：五层可见性让筑基修士和化神修士看到的同一片区域完全不同。
3. **稀缺驱动竞争**：灵脉、五行场域、稀有采集点是天然 PVP 热点，但不强制——散修可通过交易行迂回。
4. **天道中立**：天道法则不偏袒任何一方，只定义物理规则，玩家在规则内博弈。

## 一、地图架构

### 1.1 三层地图体系

| 层级 | 用途 | 渲染精度 | 服务器压力 | 可见对象 |
|------|------|---------|-----------|---------|
| **大陆层** | 飞行/传送/高阶俯瞰 | 区域色块 + 地名 + 主要标记 | 无（客户端生成） | 城镇 / NPC宗门 / 地形 |
| **区域层** | 行走/探索/采集 | 地形+资源点+玩家/NPC图标 | 极低（仅区域元数据） | 上述 + 已发现的隐藏点 |
| **场景层** | 城镇/洞府/秘境/宗门内部 | 完整场景+所有实体 | 正常实例负载 | 全部玩家/NPC/物件 |

### 1.2 切换逻辑

```
玩家移动速度 < 飞行阈值 → 区域层（当前区域的地形和资源分布）
          ↓
玩家进入飞行/传送 → 切换大陆层（选目标区域）
          ↓
玩家点击城镇/洞府入口 → 加载场景层实例
```

> 区域层与大陆层之间的切换无加载画面（客户端已缓存区域元数据）；进入场景层有短暂加载（加载完整场景资源）。

### 1.3 种子驱动生成

服务端仅存储每个区域的元数据（约 500 条记录/服务器）：

```
区域元数据 = {
  seed: u64,            // 地形种子（40亿种组合）
  region_level: enum,   // 凡域/灵域/宝域/秘境/绝地
  qi_density: f32,      // 灵气基础浓度
  terrain_type: enum,   // 主导地貌类型
  resources: [f32; 9],  // 九维资源向量
  landmarks: [Landmark],// 主要地标（城镇/NPC宗门位置）
  discovered_hidden: [HiddenSite], // 已被玩家发现的隐藏点列表
}
```

客户端收到 `seed` + `qi_density` + `terrain_type` 后，用 Perlin Noise 在本地生成地形/植被/河流/岩石，保证同一区域所有玩家看到的地形完全一致。

> 元数据总量：约 500 区域 × 200 字节 = 100 KB，数据量不到一张头像图片的大小。

## 二、可见性体系

### 2.1 五层深度发现机制

| 可见层级 | 内容 | 可见条件 | 示例 |
|----------|------|---------|------|
| **L0 公开层** | 城镇、NPC宗门、主要道路、地形轮廓 | 无门槛，所有人可见 | 黄枫谷、天南城、飞仙台 |
| **L1 任务层** | 任务指引的秘境/副本入口位置 | 接到对应任务后标记显示 | "去星海秘境取回灵草" → 秘境入口标出 |
| **L2 事件层** | 随机刷新的上古洞府、灵潮爆发点、天降奇物 | 区域内全服广播事件时显示（限时） | "西北方向灵气异常，似有洞府现世，持续 2 小时" |
| **L3 神识层** | 隐藏洞府、玩家洞府、阵法保护区域、地下遗址 | 满足神识探查条件（见 §2.2） | 化神期修士路过山壁，感知到后方有禁制波动 |
| **L4 归属层** | 自己洞府的内部状态、宗门领地资源、阵法健康度 | 拥有者/宗门成员专属 | 查看洞府阵法耐久、灵脉剩余储量、访客记录 |

### 2.2 神识探查机制

#### 探查公式

```
发现概率 = min( 基准概率 × 距离衰减 × 神识比 , 100% )

基准概率 = 10%（基础值）
距离衰减 = 1.0（100m 内）/ 0.5（100~300m）/ 0.2（300~500m）/ 0（500m 外）
神识比   = 探查者神识 / 目标隐藏阈值
```

#### 不同隐藏对象的隐藏阈值

| 隐藏对象 | 隐藏阈值 | 说明 |
|----------|---------|------|
| 天生秘境 | 30 | 天然形成的秘境入口，不主动隐藏 |
| 上古洞府 | 50~80 | 随机刷新，品阶越高隐藏越深 |
| 地下遗址 | 80~100 | 大机缘所在，只有顶级修士能感知 |
| 玩家洞府 | 布阵者神识 × 阵法隐藏等级系数 | 见 §2.3 |

#### 探查规则

| 条件 | 效果 |
|------|------|
| 探查者神识 < 隐藏阈值 | 无法发现（ 500m 内无感知） |
| 探查者神识 ≥ 隐藏阈值 × 1.0 | 500m 内基准概率发现 |
| 探查者神识 ≥ 隐藏阈值 × 1.5 | 500m 内发现概率 ×2 |
| 探查者神识 ≥ 隐藏阈值 × 2.0 | 1000m 内必发现 + 可穿透 1 层阵法干扰 |
| 物理撞上入口（<10m） | 无论神识比，直接发现 |

> 神识探查是**被动自动**的——玩家在区域层移动时，系统按秒级 tick 检测玩家神识与周边隐藏点的距离和阈值，发现后弹提示"你感知到西北方向有灵力波动"。

### 2.3 玩家洞府隐匿与攻防

#### 隐匿防护

```
洞府被发现概率 = 基础暴露值 / (阵法隐藏等级系数 × 神识屏蔽倍率)

基础暴露值    = 20 + 近期活动热度（进出次数 × 2 + 灵气波动 × 5）
阵法隐藏等级  = 迷踪阵 ×3 / 幻月阵 ×5 / 天隐阵 ×8 / 仙隐大阵 ×10
神识屏蔽倍率  = 洞主神识 / 探查者神识（≥1 时完全不可见）
```

| 阵法 | 隐藏系数 | 布阵材料 | 持续时间 |
|------|---------|---------|---------|
| 迷踪阵（黄阶） | ×3 | 下品灵石 100 + 迷雾草 ×5 | 7 天 |
| 幻月阵（玄阶） | ×5 | 中品灵石 50 + 幻阵石 ×3 | 14 天 |
| 天隐阵（地阶） | ×8 | 上品灵石 20 + 空冥石 ×1 | 30 天 |
| 仙隐大阵（天阶） | ×10 | 极品灵石 5 + 界石碎片 ×1 | 90 天 |

#### 阵法攻防

| 破阵方式 | 消耗 | 成功率 | 风险 |
|----------|------|-------|------|
| 破阵符（一次性） | 灵石购买（市场价）+ 材料消耗 | 黄阶阵 100% / 玄阶 70% / 地阶 40% / 天阶 15% | 符箓消耗，失败不退 |
| 阵法师破阵 | 阵法师小队（2~5 人）+ 时间（10~60 分钟） | 阵法等级 ≤ 阵法师等级 → 100% / 否则 50% | 破阵期间触发阵法反击 |
| 暴力破阵 | 持续攻击阵法（消耗耐久） | 阵法耐久归零即破 | 阵法反击伤害 + 洞主收到警报 |

> 阵法被攻破后，洞府进入 24 小时暴露状态（L0 公开层可见）。洞主可重新布阵，但材料消耗 ×1.5（重建期间的灵气紊乱debuff）。

## 三、区域资源系统

### 3.1 九维资源向量

每个区域存储一组资源向量（`[0~100]` 归一化），不做格子：

| # | 资源维度 | 影响因素 |
|---|----------|---------|
| 1 | **灵气浓度** | 基础值（随机生成）+ NPC宗门灵脉加成 + 玩家宗门灵脉塔加成 + 世界事件 |
| 2 | **金行丰富度** | 矿山/金属性地脉/陨铁坑 |
| 3 | **木行丰富度** | 原始森林/灵草谷/万年古树 |
| 4 | **水行丰富度** | 大湖/寒泉/深海/瀑布 |
| 5 | **火行丰富度** | 火山/地火口/沙漠/熔岩河 |
| 6 | **土行丰富度** | 山脉/灵土盆地/石英矿脉 |
| 7 | **灵草密度** | 影响炼丹材料采集产出（采药） |
| 8 | **矿脉密度** | 影响炼器材料采集产出（采矿） |
| 9 | **灵兽密度** | 影响野生妖兽刷新频率和品阶上限 |

### 3.2 五行场域效应

| 场域类型 | 典型地貌 | 对应灵根修炼加成 | 主要产出 | 灵兽偏向 |
|----------|---------|----------------|---------|---------|
| 金行场域 | 矿山、金属性地脉、陨铁坑 | 金灵根 +30% | 矿石、金属材料、庚金 | 金系妖兽（高攻低血，破甲） |
| 木行场域 | 原始森林、灵草谷、万年古树 | 木灵根 +30% | 灵草、木系材料、乙木精华 | 木系妖兽（回血、放毒、缠绕） |

## 五、灵气与灵脉系统

### 5.1 灵脉模型

灵气不是均匀分布的——它由灵脉驱动，形成修仙世界的地理经济：

```
区域灵气浓度 = 基础值 + Σ(范围内灵脉强度 × 距离衰减系数)

距离衰减系数 = max(0, 1 - 距离灵脉 / 灵脉影响半径)
```

### 5.2 灵脉来源

| 灵脉类型 | 来源 | 影响范围 | 基础强度 | 稳定性 |
|----------|------|---------|---------|-------|
| 天生灵脉 | 区域随机生成（不可移动/不可破坏） | 1~3 个相邻区域 | 0.5x ~ 3.0x | 永久稳定 |
| NPC宗门灵脉塔 | 游戏世界预设（大宗门选址必在灵脉之上） | 当前区域 + 周边 1 区域 | 1.0x ~ 2.0x | 永久稳定 |
| 玩家宗门灵脉塔 | 宗门建筑（见 L6 §2.3），消耗灵石维持 | 当前区域（可随等级扩展） | 0.2x ~ 1.5x | 需持续维护（月费灵石） |
| 灵潮爆发（世界事件） | 随机触发，全服广播 | 3~5 个区域，持续 4 小时 | 临时 +2.0x | 限时 |

### 5.3 灵脉选址竞争

| 选址类型 | 特点 | 竞争烈度 |
|----------|------|---------|
| NPC大宗门周边 | 灵脉最强（×2.0 以上），但 NPC 势力庞大不可侵占 | 零（不可抢占） |
| 天生灵脉区域 | 中高强度灵脉（×1.5~×3.0），先到先得 | 极高——宗门争夺的核心目标 |
| 普通灵域 | 低强度灵脉（×0.8~×1.5），数量充足 | 低——适合中小宗门 |
| 凡域 | 几乎无灵脉（<×0.8），不适合宗门建址 | 零（无人争抢） |

> 优质灵脉是先到先得的稀缺资源。当两个玩家宗门选址在同一区域 → 需要宗主谈判或宗门战解决（宗门战规则见 L6 §2.6 接口预留，具体在第 7 层后半展开）。

### 5.4 灵脉的经济效应

灵脉直接影响以下区域经济指标：
- **修炼速度**（L4 §1.1）：高灵脉区域 = 快速提升境界
- **灵草刷新速度**：灵脉浓度每 +1.0x → 灵草刷新间隔 -20%
- **矿脉品质上限**：灵脉浓度每 +1.0x → 矿石最高品阶上限 +1 级
- **NPC商铺物价**：灵脉越高 → 物价越贵（供给稀缺导致溢价）

## 六、移动与传送

### 6.1 移动方式总览

| 方式 | 速度（相对步行） | 消耗 | 解锁条件 | 使用层级 | 特殊交互 |
|------|---------------|------|---------|---------|---------|
| 步行 | ×1（基准） | 无 | 初始 | 区域层 | 可触发采集/战斗/发现隐藏点 |
| 御剑飞行 | ×5 | 真元持续消耗（每秒 0.5） | 筑基期 | 区域层（加速）→ 跨区域可切大陆层 | 高空奇遇/天罡风/飞行妖兽 |
| 灵兽骑乘 | ×4~×10 | 灵宠体力 | 灵宠 §5.5（见 L4） | 区域层 | 骑乘战斗（L3 §2.6） |
| 传送阵 | 瞬间 | 灵石 = 跨越区域数 × 100 | 城镇/NPC宗门固定传送阵 | 大陆层选目标 | 已发现区域才可传送 |
| 大挪移令 | 瞬间 | 一次性道具（稀有掉落/奇遇/商城） | 获得道具后 | 大陆层任意区域 | 可传送到未探索区域（有风险） |
| 宗门传送阵 | 瞬间 | 宗门贡献 | 宗门 Lv.3 + 宗门传送阵建筑 | 大陆层选宗门领地 | 仅限宗门成员 |

### 6.2 飞行中的世界交互

| 交互 | 触发概率 | 效果 |
|------|---------|------|
| 天罡风区 | 高空区域固定分布 | 持续伤害（每秒 1% 气血），需降低高度绕行 |
| 飞行妖兽袭击 | 每飞行 5 分钟，5% 概率 | 触发空战（飞行中即时战斗，L3 §2.6） |
| 俯瞰奇遇 | 每飞行 10 分钟，2% 概率 | 看到下方有异常灵光——降落后发现 L2 事件层洞府或灵物 |
| 雷暴云 | 特定天气随机生成 | 御剑速度 -50% + 持续雷击伤害，建议落地等待 |

### 6.3 传送阵网络

| 传送阵类型 | 分布 | 费用 | 冷却 |
|-----------|------|------|------|
| 城镇传送阵 | 每个主要城镇 1 座 | 跨越区域数 × 100 灵石 | 无冷却 |
| NPC宗门传送阵 | NPC大宗门各有 1 座 | 跨越区域数 × 200 灵石 | 无冷却 |
| 玩家宗门传送阵 | 需建造宗门传送阵建筑（Lv.3 解锁） | 免费（宗门成员）/ 跨越区域数 × 50（访客） | 无冷却 |

> 传送阵需"发现"才可用——玩家必须曾经到达过该区域或收到过该区域的地图碎片（任务/交易获得）。

## 七、天道法则

### 7.1 灵气守恒定律

```
全服灵气总量 ≈ 常数（波动范围 ±10%）

当某个区域灵气被大量抽取（大量玩家打坐修炼）→ 相邻区域灵气向该区域流动补充
当某个区域灵气长期无人抽取 → 灵气浓度自然上升（回补到区域基础值）
```

| 机制 | 说明 |
|------|------|
| **局部透支** | 单区域内在线修炼玩家 > 50 人时，该区域灵气浓度每 30 分钟 -0.1x（最低 -0.5x），玩家离开后每小时恢复 0.1x |
| **潮汐效应** | 每天 06:00~09:00 和 18:00~21:00（高峰时段）全服灵气浓度 +0.3x，鼓励在黄金时段集中修炼 |
| **灵脉回补** | 灵脉会持续向周边区域辐射灵气——即使某区域被透支，灵脉仍以恒定速率补充 |

> 灵气守恒确保没有某个区域可以无限承载大量玩家集中修炼，同时高峰时段的潮汐效应自然分流玩家到不同时段。

### 7.2 天道感应事件

天道不定时向全服触发随机事件，影响所有区域：

| 事件 | 频率 | 效果 | 持续时间 |
|------|------|------|---------|
| 灵潮爆发 | 每 3~7 天随机触发 | 指定 3~5 个区域灵气 +2.0x，大量上古洞府刷新 | 4 小时 |
| 魔道入侵 | 每 7~14 天随机触发 | 全域灵气 -0.5x，大量魔物刷新（击杀额外修为+掉落） | 2 小时 |
| 天机蒙蔽 | 极低概率（约每月 1 次） | 所有神识探查距离 -50%，隐藏发现概率 -30% | 6 小时 |
| 福缘天降 | 极小概率（约每半月 1 次） | 全域修炼速度 ×1.5，渡劫成功率 +10% | 1 小时 |
| 天道雷劫 | 特定境界玩家达到阈值时触发世界级渡劫 | 全服可见天象，渡劫者附近区域灵气紊乱 | 30 分钟 |

### 7.3 天劫世界化

当某位玩家触发渡劫时（L3 §3.2、L4 §1.5），渡劫事件由个人战斗扩展为世界级事件：

| 天劫阶段 | 世界表现 |
|----------|---------|
| 劫云聚集（前 5 分钟） | 渡劫者所在区域天象变化（所有玩家可见）——乌云密布、雷电闪烁 |
| 天雷落下（渡劫战斗中） | 区域层可见天雷劈下（视觉效果），其他玩家可靠近观礼（禁止 PK 渡劫者，但可偷袭护法者） |
| 渡劫成功 | 全服公告 + 渡劫者获得临时光环（7 天）——附近玩家修炼速度 +5% |
| 渡劫失败 | 区域灵气永久 -0.1x（天道惩罚）+ 渡劫者修为大损（见 L4 §1.5） |

> 大乘及以上境界渡劫 → 全服世界事件。道祖唯一性渡劫 → 全域持续 1 小时的灵潮异常，所有玩家修炼速度翻倍。

## 八、世界事件系统

### 8.1 事件分类

| 事件类型 | 触发方式 | 影响范围 | 典型事件 |
|----------|---------|---------|---------|
| 区域事件 | 区域随机触发 | 1 个区域 | 妖兽潮、灵草爆发、矿脉塌方 |
| 全域事件 | 天道定时触发 | 全服所有区域 | 灵潮爆发、魔道入侵、福缘天降 |
| 连锁事件 | 由前序事件触发 | 逐步扩展 | 魔道入侵→某个 NPC 宗门被攻破→连锁任务线→最终 Boss 现世 |
| 赛季事件 | 按现实时间周期（月/季度） | 全服 | 宗门大比、天下第一斗法大会 |

### 8.2 区域随机事件表

| 事件 | 触发概率（每小时） | 效果 | 持续时间 |
|------|-----------------|------|---------|
| 妖兽潮 | 3% | 区域内大量妖兽刷新，击杀额外掉落 | 30 分钟 |
| 灵草爆发 | 2% | 灵草密度临时 ×3，刷新速度 ×5 | 1 小时 |
| 矿脉塌方 | 1%（矿区概率 ×3） | 矿脉密度临时 ×0.5，但挖出稀有矿石概率 ×5 | 1 小时 |
| 天降奇物 | 0.5% | 区域内随机位置刷新一件稀有道具（先到先得） | 30 分钟或被发现 |
| 路遇高人 | 1%（道路附近概率 ×3） | NPC 高阶修士路过，可请教（获得少量修为）或交易 | 15 分钟 |

### 8.3 连锁事件示例：魔道入侵

```
阶段 1：魔道异动（0:00）
  → 全域灵气 -0.5x，随机 3 个区域出现魔物刷新点

阶段 2：魔潮涌动（1:00）
  → 魔物品阶提升，随机 1 个区域 NPC 小镇被攻击
  → 玩家可前往守卫（成功 → 善业 + 宗门贡献 + 特殊奖励）

阶段 3：魔君降临（2:00）
  → 世界 Boss「魔君」在随机绝地刷新
  → 全服玩家可组队挑战（每队上限 8 人）
  → 击杀奖励：天阶功法残卷 / 仙器材料 / 大量灵石

阶段 4：魔潮退去（Boss 被击杀或 4 小时后）
  → 灵气恢复，魔物停止刷新
  → 按贡献排名发放额外奖励
```

## 九、阵营系统

### 9.1 三大势力

| 势力 | 核心 NPC宗门 | 立场 | 加入条件 | 核心优势 |
|------|------------|------|---------|---------|
| 正道联盟 | 黄枫谷、天剑宗、太一门 | 秩序守序——禁止恶意 PVP，协助 NPC 城镇防御 | 境界 ≥ 筑基 + 善业 ≥ 20 | NPC 商店折扣 -15%，传功阁功法数量 +50% |
| 魔道联盟 | 鬼灵门、合欢宗、血魔殿 | 混乱自由——允许一切 PVP，邪道功法开放 | 境界 ≥ 筑基 + 杀业 ≥ 20 | 击杀玩家额外修为 +20%，邪道功法专属 |
| 散修同盟 | 无固定宗门 | 中立——不参与阵营冲突，可自由切换 | 无限制 | 交易行手续费 -5%（代替商会福利），切换阵营无冷却 |

> 阵营与宗门可同时存在——玩家可以同时是正道联盟成员和某个玩家宗门的成员。阵营是宏观立场，宗门是微观组织。

### 9.2 阵营对立规则

| 规则 | 说明 |
|------|------|
| 战斗标识 | 敌对阵营玩家在野外默认可攻击（不增加 PK 值） |
| 城镇安全区 | 所有城镇为绝对安全区（禁止 PVP），不论阵营 |
| NPC宗门区域 | 正道 NPC宗门对魔道玩家封闭（不可进入），反之亦然 |
| 散修同盟特殊 | 散修不可攻击任何阵营玩家，也不可被攻击（但也不享受阵营福利） |

### 9.3 阵营贡献与商店

| 行为 | 阵营贡献获取 |
|------|------------|
| 击杀敌对阵营玩家（同境界及以上） | +100~+500（按境界差） |
| 完成阵营任务 | +200~+1000 |
| 守卫 NPC宗门（对抗敌对阵营攻击） | +500~+2000 |
| 背叛阵营（切换） | -50% 当前阵营贡献（充公） |

**阵营商店**：消耗阵营贡献兑换阵营专属功法、技能书、丹药配方、装备外观。

### 9.4 阵营战

阵营战是第 7 层的终局 PVP 内容：

| 维度 | 说明 |
|------|------|
| **触发** | 每周六 20:00（固定时间），正道 vs 魔道 |
| **战场** | 指定的中立区域（独立实例，不影响普通地图） |
| **规模** | 大规模（每方上限 200 人，按境界分 3 个战区） |
| **目标** | 占领据点 → 推进战线 → 攻破敌方大本营 |
| **奖励** | 胜方全员：阵营贡献 + 灵潮加成（胜方区域灵气 +1.0x 持续 24h），败方：无惩罚但无增益 |
| **赛季** | 每 3 个月为 1 赛季，赛季末结算阵营总排名，前 100 名玩家获得赛季专属称号和外观 |

### 9.5 NPC宗门外交

| 外交状态 | 条件 | 效果 |
|----------|------|------|
| 中立 | 默认 | NPC宗门对所有同阵营玩家开放，正常交易 |
| 友好 | 玩家完成该宗门 10 个以上任务 | 商店额外折扣 -5%，可学习 NPC宗门专属低级功法 |
| 尊敬 | 玩家完成该宗门 50 个任务或拯救宗门于危难 | 商店折扣 -10%，可学习 NPC宗门专属中级功法，可租用宗门洞府 |
| 敌对 | 玩家攻击该 NPC宗门成员或协助敌方阵营 | 禁止进入，被该宗门 NPC 主动攻击 |
