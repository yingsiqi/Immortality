# 第6层：社交系统规则书

> 版本：v1.0 | 最后更新：2026-07-16

---

## 概述

第6层定义**玩家之间怎么互动**。基于前5层已建立的师徒种子（L4 §7）、宗门种子（L5 §6）、PK与悬赏种子（L3 §5），将师徒、宗门、因果业力、组队四大子系统编织成完整的社交框架。

**设计原则**：
1. **双向互惠**：社交关系必须有来有往，单向付出不可持续。师徒收益非对称但互为闭环。
2. **渐进解锁**：社交功能随境界逐步解锁，防止小号社交号泛滥。
3. **因果真实**：业力不是简单的红名/白名二元，是多维度的行为标记，有真实后果。
4. **纵向预留**：宗门地盘争夺、跨服阵营等大规模社交内容留到第7层（阵营/世界）展开。

## 一、师徒系统

### 1.1 结对规则

| 维度 | 说明 |
|------|------|
| **师父收徒数量** | 随境界逐步解锁：筑基 1 位、结丹 2 位、元婴 3 位、化神 5 位、炼虚 7 位、合体 10 位、大乘 15 位 |
| **徒弟拜师限制** | 同一时间仅可拜一位师父 |
| **境界门槛** | 师父必须比徒弟至少高 1 个大境界（炼气期不可收徒） |
| **结对方式** | 双方在安全区域通过 NPC「师道碑」完成结对仪式，或通过好友列表发起师门邀请 |

### 1.2 关系等级

| 等级 | 达成条件 | 传功倍率 | 突破加成 | 特殊权限 |
|------|---------|---------|---------|---------|
| 记名弟子 | 初始结对 | ×1.0 | 无 | 可接收传功 |
| 入门弟子 | 师徒互动 ≥ 30 天 | ×1.5 | 徒弟修为一阶突破 → 师父突破成功率 +2% | 可参与师门任务 |
| 亲传弟子 | 师徒互动 ≥ 90 天 + 徒弟至少提升 2 个大境界 | ×2.0 | 徒弟修为一阶突破 → 师父突破成功率 +5%、天劫波数 -1 | 继承师父专属功法（玄阶） |
| 恩师 | 徒弟修为超出师父 1 个大境界后自动降级 | ×0.5（仅保留功法库访问） | 无师徒双向加成 | 保留关系不解除 |

> 「师门互动」定义为：传功、助战（协助副本/Boss）、护法（渡劫时在场）、上交材料、师门任务完成——任一项触发即计为当天互动。

### 1.3 师徒双向收益

#### 师父 → 徒弟

| 收益 | 说明 | 限制 |
|------|------|------|
| **传功：功法传授** | 师父将已掌握的黄阶/玄阶功法传授给徒弟 | 仅限入门级功法，天阶/仙阶功法不可传授（防高阶贬值） |
| **传功：技能传授** | 师父将已掌握的黄阶/玄阶技能书传授给徒弟 | 同上，限入门级 |
| **传功：熟练度加速** | 每次传功可小幅提升徒弟某功法熟练度 +5% | 冷却 24 小时，同一功法每 7 天最多接受 1 次 |
| **师门任务** | 师父可发布每日师门任务（如击杀指定怪物、收集材料），徒弟完成后获修为奖励 | 师父每日限发 3 个任务 |
| **助战** | 徒弟可请求师父助战低阶副本/Boss（不消耗师父灵力） | 限比徒弟当前境界低 1 阶及以下的内容 |

#### 徒弟 → 师父

| 收益 | 说明 | 限制 |
|------|------|------|
| **功法突破上限** | 徒弟总修炼进度累积贡献给师父，每达成阈值 → 师父功法等级突破上限永久 +1 | 每位徒弟终身贡献上限 = 师父当前境界系数 × 5 |
| **突破成功率加成** | 徒弟修为一阶突破（如筑基初→筑基中）时 → 师父突破成功率 +2%（亲传 +5%） | 加成累计上限 25%，持续至师父下次突破后清零 |
| **天劫波数减免** | 亲传弟子可为师父渡劫护法 → 天劫波数 -1 | 每次渡劫限 3 名亲传弟子护法（最多 -3 波） |
| **材料上供** | 徒弟可将低阶材料（炼丹/炼器原材料）上供给师父 | 低阶材料在高境界玩家处掉落打折（L5 §9.1），徒弟反成主要来源 |

> 高境界玩家刷低阶副本掉落率衰减（见 L5 §9.1），导致高阶修士反而缺低阶材料。徒弟在低境界产出大量低阶材料，形成天然的物质互补。

### 1.4 解除与降级

| 场景 | 效果 | 惩罚 |
|------|------|------|
| **单方解除** | 7 天内无互动，任意一方可单方面解除 | 无惩罚（冷静退出） |
| **主动解除（未满 30 天）** | 任意一方强制解除 | 双方「师徒动荡」debuff——24h 内不可重新结对 |
| **自动降级为恩师** | 徒弟修为超出师父 1 个大境界 | 关系保留但收益停止 |
| **频繁解除** | 月内解除/重结对 ≥ 3 次 | 「师门动荡」debuff——双方 7 天内师徒收益 -50% |

### 1.5 防止滥用设计

| 策略 | 说明 |
|------|------|
| **解除冷却** | 每次解除后 24 小时内不可重新结对 |
| **频繁惩罚** | 月内 3 次以上解除 → 7 天 debuff，防止师父刷徒弟数量 |
| **传功上限** | 传功熟练度加速 7 天仅 1 次/功法，防止批量化速成 |
| **徒弟终身贡献上限** | 每位徒弟对师父功法突破上限的贡献有上限，防止养小号堆属性 |

## 二、宗门系统

### 2.1 创立条件

| 条件 | 要求 |
|------|------|
| **境界** | 元婴期及以上 |
| **灵石** | 50 万灵石（一次性消耗） |
| **宗门令牌** | 秘境/世界 Boss 掉落的稀有道具 ×1 |
| **初始规模** | 10 人上限，1 座洞府 |

### 2.2 宗门等级与规模

| 宗门等级 | 升级条件 | 人数上限 | 洞府数量 | 建筑上限 |
|----------|---------|---------|---------|---------|
| Lv.1 初立宗门 | 初始 | 10 | 1 | 主殿 |
| Lv.2 小有名气 | 宗门贡献 5000 | 20 | 2 | 主殿 + 2 附属建筑 |
| Lv.3 一方势力 | 宗门贡献 20000 | 50 | 4 | 主殿 + 3 附属建筑 |
| Lv.4 名震一方 | 宗门贡献 80000 | 100 | 8 | 主殿 + 4 附属建筑 |
| Lv.5 仙道大宗 | 宗门贡献 300000 | 200 | 15 | 主殿 + 5 附属建筑 |

> 宗门贡献由成员完成宗门任务、捐献材料/灵石、宗门战胜利累积。

### 2.3 宗门建筑

| 建筑 | 功能 | 解锁条件 | 升级效果 |
|------|------|----------|---------|
| **主殿** | 宗门等级核心，决定人数上限和洞府数量 | 初始（Lv.1） | 每级 +10 人上限 |
| **传功阁** | 宗门功法库：成员可学习宗门专属功法（黄阶~地阶），消耗宗门贡献 | Lv.2 | 等级越高，可存放功法数量越多 |
| **炼丹房** | 宗门炼丹成功率 +N% + 炼丹灵力消耗减免 | Lv.3 | Lv.1：+5%；Lv.3：+15% |
| **炼器坊** | 宗门炼器成功率 +N% + 装备耐久维修折扣 | Lv.3 | Lv.1：+5%；Lv.3：+15% |
| **灵脉塔** | 宗门灵气浓度提升——全员修炼速度加成 | Lv.4 | Lv.1：灵气浓度 ×1.2；Lv.3：×1.5 |
| **藏经阁** | 解锁宗门专属技能树（被动技能，仅宗门成员可学） | Lv.5 | 等级越高，技能树层数越多 |

### 2.4 任职与选举

| 职位 | 人数 | 权限 | 产生方式 |
|------|------|------|---------|
| **宗主** | 1 | 全部宗门管理权（建设/任职/收人/踢人/资源分配） | 宗门创立者；可禅让 |
| **副宗主** | 2 | 建设/任职提名/收人/踢人/部分资源分配 | 宗主任命 |
| **长老** | 4 | 收人/踢人（需宗主或副宗主批准）/洞府批准 | 宗主任命或成员选举（票数前 4） |
| **执事** | 8 | 收人/洞府推荐 | 宗主任命或长老推荐 |

> 选举周期：每 30 天一次。执业成员额外获得宗门资源分配权重（×1.2）。

### 2.5 成员福利

| 福利 | 说明 | 门槛 |
|------|------|------|
| **灵力保底** | 每小时额外恢复 2~10 点灵力（取决于灵脉塔等级） | 加入宗门即享 |
| **每日物资箱** | 每日可领取基础丹药 ×3 + 随机材料 ×2 | 加入宗门即享 |
| **洞府修炼** | 分配洞府 → 额外灵气浓度加成（×1.1~×1.3） | 需长老/宗主批准 |
| **宗门技能** | 藏经阁被动技能（如宗门团结——同宗门组队 +5% 全属性） | 藏经阁解锁 + 消耗宗门贡献 |
| **交易行优惠** | 宗门成员在交易行手续费额外 -0.5% | 加入宗门即享（与商会手续费减免叠加） |

### 2.6 宗门战与地盘争夺（接口预留）

> 完整宗门战规则（地盘争夺、资源点占领、宗门联盟/敌对）属于第 7 层「阵营与世界」的设计范围。第 6 层仅定义以下基础框架：

| 接口 | 预留说明 |
|------|----------|
| **宗门敌对/联盟** | 宗主可声明敌对/联盟关系，目前仅影响 PVP 标识（同宗门不可攻击，敌对宗门可攻），实际收益等第 7 层补齐 |
| **地盘标记** | 宗门可在世界地图上标记领地，目前为纯标识，无资源产出加成 |
| **宗门排行榜** | 按宗门贡献排序，每季度重置一次（与第 7 层赛季联动） |

## 三、因果业力系统

### 3.1 设计哲学

修仙世界的因果业力不是简单的"好人/坏人"二元标签。一位修士可以杀伐果断却在师门中尽心尽责，也可以乐善好施却欠下满地情债。五维业力体系让每个行为在不同维度留下印记，综合影响 NPC 态度、秘境准入和天劫难度。

### 3.2 五维业力

| 业力维度 | 积累方式 | 正业力效果（该维度值较高时） | 负业力效果（该维度值较低时） |
|----------|----------|---------------------------|---------------------------|
| **杀业**（0~100） | PVP 击杀非红名玩家 +5/次、击杀 NPC +1/次 | — | 天劫伤害 +N%（每 10 点杀业 +2%），被悬赏概率 ↑ |
| **善业**（0~100） | 助战 +1/次、复活他人 +2/次、救治 NPC +1/次 | 天劫伤害 -N%（每 10 点善业 -2%），NPC 商店折扣 ↑ | — |
| **情业**（0~100） | 师徒/道侣互动天数累积 | 师徒/道侣收益倍率 +N%（每 20 点 +5%） | 解除关系双方受惩罚 ×1.5 |
| **财业**（0~100） | 交易行成交额累积、赠送物品价值累积 | 交易行手续费减免（每 20 点 -0.5%，上限 -2.5%） | 被偷窃成功率 ↑（树大招风） |
| **名业**（-50~+50） | 悬赏排名 +N、斗法大会排名 +N、宗门战贡献 +N（正面）；被悬赏 -N、红名状态 -N（负面） | 宗门任职门槛降低、NPC 对话解锁额外任务 | 被悬赏金额累加、部分 NPC 拒绝交易 |

> 业力每日 UTC 0 点结算一次，按五维中**绝对值最高**的维度决定当日的「业力标签」（如"杀孽深重""乐善好施""重情重义""富甲一方""名动天下"）。

### 3.3 业力对渡劫的影响

```
天劫伤害倍率 = 基础倍率 × (1 + 杀业加成 - 善业减免)

杀业加成 = floor(杀业值 / 10) × 2%
善业减免 = floor(善业值 / 10) × 2%
```

| 场景 | 示例 |
|------|------|
| 杀业 50、善业 10 | 天劫伤害 +10% -2% = +8% |
| 杀业 0、善业 80 | 天劫伤害 -16% |
| 杀业 100、善业 100 | 天劫伤害 +20% -20% = 0%（"功过相抵"） |

> 天劫难度最高加成 +20%，最高减免 -20%。这意味着极端善人渡劫显著容易，极端杀胚需要更多装备/丹药支撑。

### 3.4 悬赏与追杀

| 机制 | 说明 |
|------|------|
| **自动悬赏** | 杀业 ≥ 30 的玩家被系统自动悬赏，赏金 = 杀业值 × 境界系数 × 100 灵石 |
| **玩家悬赏** | 任何玩家可对通缉榜（杀业 ≥ 30）上的玩家追加悬赏金额（消耗自身灵石） |
| **接取追杀** | 在 NPC「悬赏榜」处接取追杀任务，成功击杀目标后领取赏金 |
| **被悬赏者死亡** | 额外损失修为（见 L5 §1.4 负面行为惩罚表） + 赏金从自身灵石池扣除（不足则负债） |
| **赏金上限** | 单人累计悬赏金额上限为 50 万灵石，防巨额悬赏经济崩坏 |

### 3.5 业力与名声的经济影响

| 名声区间 | NPC 商店折扣 | 特殊效果 |
|----------|-------------|---------|
| 名业 ≥ +30（名扬四海） | -10% | 创建宗门灵石消耗 -20% |
| 名业 +10~+29（小有名气） | -5% | NPC 额外对话选项 |
| 名业 -10~+9（默默无闻） | 0% | 无特殊效果 |
| 名业 ≤ -10（臭名昭著） | +10%（加价） | 创建宗门灵石消耗 +50%，部分 NPC 拒绝交易 |

## 四、组队系统

### 4.1 基础规则

| 维度 | 说明 |
|------|------|
| **队伍规模** | 2~5 人（标准），特殊副本/世界 Boss 可扩容至 8 人 |
| **境界差限制** | 队伍内最高与最低境界差 ≤ 2 个大境界（防带飞） |
| **组队方式** | 好友列表邀请 / 区域喊话组队 / 副本匹配队列（见 §4.3） |
| **队长权限** | 可踢人、邀请、设置分配模式、发起就位确认 |

### 4.2 掉落分配

| 模式 | 规则 | 适用场景 |
|------|------|---------|
| **自由拾取** | 谁先点谁拿（默认） | 亲友队，信任关系 |
| **轮流分配** | 按击杀顺序轮流获得掉落物 | 野队，公平第一 |
| **贡献分配** | 按战斗贡献（伤害 + 治疗 + 承伤综合）权重随机分配 | 混野队，数据说话 |

### 4.3 副本匹配队列

| 维度 | 说明 |
|------|------|
| **触发** | 在副本入口选择「匹配队伍」，系统自动匹配同境界 ±1 小层的 4 名玩家 |
| **匹配优先级** | 同宗门 > 好友 > 最近组队 > 随机 |
| **跨境界组队惩罚** | 高境界带低境界时：高者掉落品质 -20%，低者掉落品质 +30%（鼓励同等水平组队，不鼓励代练） |
| **匹配超时** | 5 分钟未匹配成功 → 允许降低匹配精度（境界差放宽至 ±3 小层） |

### 4.4 队伍增益

| 增益 | 效果 | 条件 |
|------|------|------|
| **同门齐心** | 全队全属性 +5% | 全员同宗门 |
| **手足之情** | 全队经验获取 +10% | 队员互为好友 |
| **师门齐出** | 师父在队时徒弟全属性 +8% | 师徒组队 |
| **临时阵法** | 队伍人数 ≥ 4 → 自动激活小型聚灵阵（灵气浓度 ×1.2） | 在野外区域 |

## 五、社交面板总览

### 5.1 社交关系网

| 关系类型 | 建立方式 | 维持 | 收益 |
|----------|---------|------|------|
| 好友 | 双方确认添加 | 无维持要求 | 组队增益 + 好友交易手续费 -1% |
| 师徒 | 通过师道碑结对 | 7 天内有互动 | 双向收益（见 §1.3） |
| 道侣 | 双方确认 + 消耗「同心结」（稀有道具） | 30 天内有互动 | 双修加成（组队修炼速度 +15%） + 共享洞府 |
| 仇人 | 被击杀后自动添加 | 无 | 击杀仇人额外修为奖励 |
| 宗门 | 申请 + 审批 | 活跃度不低于门槛 | 全面宗门福利（见 §2.5） |

> 道侣系统的完整规则（双修数值/情缘任务/离婚惩罚）留到后续版本细化，第 6 层仅定义关系框架。

## 六、反社交滥用

| 策略 | 说明 |
|------|------|
| **师徒刷号** | 终身贡献上限 + 月内 3 次解除 debuff（见 §1.5） |
| **宗门刷物资** | 每日物资箱绑定不可交易 + 退宗门 48 小时冷却才能加入新宗门 |
| **悬赏洗钱** | 悬赏金额由被杀者自身灵石池扣除（不足则负债），无法通过悬赏转移灵石 |
| **组队代练** | 跨境界组队掉落惩罚（见 §4.3） + 境界差 ≥ 3 个大境界禁止组队匹配 |
| **业力洗白** | 业力值每日自然衰减仅 0.5 点（杀业/名业负值），洗白至少需要 200 天——鼓励用善业对冲而非等待 |
