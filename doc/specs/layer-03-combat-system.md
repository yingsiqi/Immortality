# 第3层：战斗系统规则书

> 版本：v1.0 | 最后更新：2026-07-15

---

## 概述

第3层定义**战斗怎么打**。基于第1层（属性）和第2层（行动原子），将玩家可执行的行动原子编排成完整的战斗体验。

**基调：即时 ARPG**。修仙世界的法宝对轰、飞剑绞杀、灵气护体不适合回合制。但即时战斗不意味着只有"大地图刷怪"一种形态——本章覆盖 15 种战斗/类战斗模式。

## 二、常规即时战斗（模式 1-5）

### 2.1 操作模型

| 决策点 | 方案 |
|--------|------|
| **锁定方式** | **半锁定**：软锁定最近目标，远程技能可手动瞄准。纯无锁定门槛太高，纯锁定失去修仙灵动感 |
| **技能栏** | 8 主动技能槽 + 4 法宝技能槽 + 2 丹药快捷栏 |
| **闪避 / 冲刺** | 消耗真元的短距冲刺，CD 1.5s。身法越高无敌帧越长（上限 0.3s） |
| **防御** | 格挡键（消耗真元持续减伤），无完美格挡判定（与魂系拉开差异） |
| **目标切换** | Tab 键循环切换；点击目标模型直接锁定 |
| **战斗视角** | 第三人称越肩视角，视野距离可调。御剑飞行时自动拉远 |

### 2.2 普攻连段

每个流派 3 段基础连招，最后一段附带流派特殊效果：

| 流派 | 第 1 段 | 第 2 段 | 第 3 段（终结） | 特殊效果 |
|------|---------|---------|-----------------|----------|
| **锻体** | 拳击 | 肘击 | 崩山靠 | 击退 + 目标硬直 0.5s |
| **法修** | 灵气弹 | 双弹 | 爆裂弹 | 小范围爆炸（半径 2m） |
| **剑修** | 横斩 | 竖斩 | 穿心刺 | 破甲（无视 10% 防御） |
| **召唤** | 灵宠扑击 | 灵宠撕咬 | 灵宠猛袭 | 灵宠追击目标 2s |

### 2.3 伤害公式

```
最终伤害 = [技能基础伤害 × (1 + 灵根纯度加成%)] 
          × 五行克制系数(0.75 / 1.0 / 1.25)
          × (1 + 会心倍率)          ← 如果暴击触发
          - 目标护体罡气
          × (1 - 目标减伤%)
```

| 参数 | 来源 | 说明 |
|------|------|------|
| 技能基础伤害 | 技能表（第3层附录） | 每个技能的固定数值 |
| 灵根纯度加成 | 第1层 §1.3 | 灵根纯度 × 对应技能属性系数 |
| 五行克制系数 | 第1层 §1.4 | 克 1.25 / 平 1.0 / 被克 0.75 |
| 会心倍率 | 第1层 §2.1（机缘） | 暴击时触发，倍率 = 1 + 机缘/1000 |
| 护体罡气 | 第1层 §3.1 | 先做减法再乘减伤 |
| 减伤% | 第2层 §3.3 | 格挡 / 灵气护体 / 装备 |

> **克制系数放在乘法中间**，确保 ±25% 的效果足够明显——你不能无视五行克制硬打。PVP 中克制系数放大至 ±30%。

### 2.4 四大流派战斗体验

| 维度 | 锻体 | 法修 | 剑修 | 召唤 |
|------|------|------|------|------|
| **战斗距离** | 近身（0-3m） | 中远（5-20m） | 中距离（3-10m） | 远程（自身在后，灵宠在前） |
| **血量定位** | 高血量（坦克） | 低血量（玻璃炮） | 中等血量 | 低血量（依赖灵宠扛伤） |
| **爆发模式** | 持续压制 | 技能循环爆发 | 破甲后连招 | 灵宠大招 + 本体辅助 |
| **生存手段** | 高减伤 + 高血量 | 护盾 + 闪避 | 机动性 + 格挡 | 灵宠替死 + 控制 |
| **操作关键词** | 拳拳到肉的打击感 | 法术视觉轰炸 | 飞剑拉扯的灵动感 | MOBA 式双线操作 |

### 2.5 战斗节奏预期

| 场景 | 预期时长 | 说明 |
|------|----------|------|
| 普通野怪 | 5-15s | 3-5 个技能循环 |
| 精英怪 | 30-60s | 需要闪避/格挡交互 |
| 副本 Boss | 3-8 分钟 | 多阶段机制 |
| 世界 Boss | 15-30 分钟 | 百人参与 |
| PVP 切磋 | 30-90s | 拼资源管理 |

### 2.6 御剑飞行战斗

修仙的核心幻想不可省略。战斗中飞行规则：

| 状态 | 效果 |
|------|------|
| **地面战斗**（默认） | 全部技能可用，消耗正常 |
| **御剑升空**（消耗真元激活） | 移速 +40%，无法使用普攻连段，只可用飞行专属技能（4 槽），真元持续消耗 |
| **空中施法** | 法修在空中伤害 +10%，但被地面远程攻击命中率 +15%（靶子更大） |
| **俯冲攻击** | 剑修专属：从空中俯冲，伤害 ×1.5，但落地后闪避 CD 翻倍（1T） |
| **坠落** | 真元耗尽自动落地，坠落过程中无法操作，落地受到坠落伤害 |

## 三、特殊战斗模式（模式 6-10）

### 3.1 心魔战 —— 镜像对决

| 维度 | 设计 |
|------|------|
| **触发** | 境界突破时概率触发（小境界 10%，大境界 30%，化神以上 50%） |
| **心魔属性** | 复制玩家当前全部属性 × 1.2；AI 行为固定但高效 |
| **反五行设定** | 心魔的灵根为玩家的反五行（玩家火灵根 90 → 心魔水灵根 90）。双方互相处于被克状态，谁的异属性手段多谁赢 |
| **场地** | 识海空间——无边无际的黑色镜面，无地形障碍 |
| **胜利** | 击杀心魔 → 突破成功，获得「心魔残渣」道具（炼丹材料） |
| **失败** | 被心魔击杀 → 突破失败，修为损失 = `(1 - 毅力减免%) × 20%` |
| **特殊规则** | 心魔战中不可使用丹药（识海中无实物），只能靠自身实力 |

### 3.2 渡劫 —— 生存塔防

| 维度 | 设计 |
|------|------|
| **触发** | 大境界突破必经（小境界突破可选跳过渡劫动画，直接成功率判定） |
| **波次** | 筑基 3 波 / 结丹 5 波 / 元婴 7 波 / 化神 9 波 / 炼虚 13 波 / 合体 17 波 / 大乘 21 波 / 真仙 36 波 / 道祖 81 波 |
| **天雷类型** | 普通雷（直线劈下）/ 追踪雷（追踪玩家须闪避）/ 金雷（无视部分防御）/ 心雷（跳过气血直接削减神识） |
| **玩家操作** | 移动闪避（身法判定）、灵气护体（真元硬抗）、法宝护主（装备法宝自动挡雷，有耐久）、紧急丹药 |
| **禁止项** | 不可攻击天雷 |
| **多人护法** | 可邀请队友在场地边缘帮忙挡雷（队友自身也受伤害），最多 3 名护法 |
| **评分** | 剩余气血比例 × 50% + (1 - 渡劫用时/最大时限) × 50% → SSS / SS / S / A / B。评分影响突破后初始属性加成 |

### 3.3 神识对决 —— 纯数值博弈

核心理念：两个元婴老怪对坐不动，但神识已在虚空中绞杀了三百回合。设计为与 ARPG 完全解耦的独立副玩法。

| 维度 | 设计 |
|------|------|
| **触发** | 主动向同境界或低境界修士发起「威压」；高阶 NPC 也可能主动压迫玩家 |
| **界面** | 双方不移动，画面表现为神识场的视觉化博弈（灵感：拔河 / 精神力的波纹对抗） |
| **操作** | 每轮双方盲选一个象限投入神识值：**攻击**（压制对方）→ **防御**（守住自己）→ **探查**（读取对方本轮选择）→ **佯攻**（虚晃，若对方防御则趁机造成双倍伤害） |
| **克制关系** | 攻击 > 探查 > 佯攻 > 防御 > 攻击（四人择博弈，类似猜拳但有深度） |
| **胜负** | 一方神识归零或主动投降。败方被胜方读取一段记忆/情报（对 NPC 可获取隐藏任务线索） |
| **场景** | 审问俘虏、获取情报、高阶 PVP 前置（先拼神识决定先手优势） |
| **定位** | 独立副玩法，不影响 ARPG 手感。可作为高阶玩家间的"文斗"玩法 |

### 3.4 破阵 / 幻境 —— 战斗 + 解谜

| 维度 | 设计 |
|------|------|
| **阵型分类** | **迷阵**（地图随机变化，走到边缘传送回中心）、**杀阵**（持续掉血 + 定时刷怪）、**困阵**（无法离开，需按顺序破解机关）、**幻阵**（虚假敌人和假宝箱，攻击假宝箱触发陷阱） |
| **破解流程** | 神识探查发现阵眼位置 → 边战斗边移动至阵眼 → 攻击阵眼或按顺序触发机关 → 阵法破除 |
| **干扰** | 破解过程中阵灵 / 守护兽持续干扰玩家。阵灵免疫普通攻击，仅能用对应五行属性技能击破 |
| **产出** | 破阵成功 → 阵旗碎片 + 秘境奖励。集齐碎片可自己布阵 |
| **难度** | 阵法品阶（黄 / 玄 / 地 / 天 / 仙）决定破解时间和阵灵强度 |

### 3.5 夺宝混战 —— 自由 PVP

| 维度 | 设计 |
|------|------|
| **触发** | 世界公告：「XX 秘境中有宝物即将出世」，玩家自行前往 |
| **规则** | 进入秘境后全员强制 PVP。宝物出现后，拾取者获得标记（全图可见），移速 -30%，每秒获得积分 |
| **策略空间** | 不一定是拾取者最强——可能是帮会清完其他人后，轻功最好的队友带宝物跑路 |
| **结算** | 宝物被带出秘境或时间耗尽 → 累计积分最高者获胜 |
| **奖励** | 宝物本身归最终持有者；积分前 3 名额外获得灵石/功勋 |

## 四、其他战斗模式简述（模式 11-15）

| 模式 | 核心设计 | 与常规战斗的差异 |
|------|----------|-----------------|
| **兽潮防守** | 世界事件触发，妖兽波次进攻据点。玩家可建造临时防御工事（类似塔防），同时亲自参战。波次间隔可修复工事、补给丹药 | 有建造元素；注重资源管理和分配 |
| **斗法大会** | 报名参赛，1v1 淘汰制。观众可下注。场地为圆形擂台，出界判负（不能用逃跑流）。冠军获得称号 + 特殊功法 | 有边界限制；禁用丹药（公平性） |
| **埋伏 / 追杀** | PK 值高的玩家可在 NPC 处被悬赏，悬赏者获得目标大致方位。埋伏方可在地图上设伏，进入战斗瞬间先手方获得 1T 先制 | 大地图无缝切入；先手优势机制 |
| **炼丹事故** | 炼丹失败 5% 概率触发。暴走丹灵 = 该丹药属性的元素精怪，击杀后回收部分材料。收服丹灵可使其成为炼丹助手（提高后续成功率） | 可选择的非致命战斗；有收服机制 |
| **奇遇事件战** | 随机参数池：1v1 / 2v2 / 3v3 / 有环境陷阱 / 有 NPC 辅助 / 禁用某属性 / 限时。每次奇遇从参数池中随机组合规则 | 每次体验不同；考验玩家适应力 |

## 五、PVP 专项规则

### 5.1 先手确定

| 场景 | 先手规则 |
|------|----------|
| **切磋** | 发起方先手，但双方均有 1T 准备时间（不开打，只能 buff） |
| **追杀 / 埋伏** | 埋伏方必定先手，持续 2T |
| **夺宝混战** | 无先手规则，谁先打到算谁的 |
| **斗法大会** | 双方神识判定，高者先手；若神识差值不足 20%，随机先手 |

### 5.2 死亡惩罚

| 死亡场景 | 惩罚 |
|----------|------|
| **切磋** | 无惩罚 |
| **野外 PVE 死亡** | 损失 5% 当前境界修为；原地复活（10 分钟 CD）或回城复活 |
| **PVP 被击杀** | 损失 5% 修为 + 背包中随机 1 件非绑定物品掉落 |
| **PK 值惩罚** | 主动击杀非红名玩家累积 PK 值。PK 值越高，被击杀时掉落物品 +1 件，修为损失 +5%。PK 值随在线时间缓慢衰减 |
| **渡劫 / 心魔失败** | 修为损失（见 §3.1、§3.2），无物品损失 |

### 5.3 PVP 平衡策略

- 五行克制在 PVP 中放大至 ±30%（PVE 为 ±25%）
- PVP 中丹药冷却加倍，防止「嗑药流」
- 逃跑成功率 PVP 中减半

## 六、技能体系框架

技能池设计留到第4层（成长系统）详细展开，此处仅定下框架约束：

| 维度 | 约束 |
|------|------|
| **技能数量** | 每个流派约 15-20 个可选技能，玩家最多装备 8 个 |
| **学习方式** | 功法书 + NPC 传授 + 奇遇 + 宗门 |
| **升级方式** | 使用次数累积熟练度 → 自动升级（第4层定义经验曲线） |
| **属性标签** | 每个技能标注五行属性（或异灵根属性/无属性），受灵根纯度加成和五行克制 |
| **冷却体系** | GCD = 1s（所有技能共享），单个技能独立 CD = 3-60s |
| **资源消耗** | 所有主动技能消耗真元（量不同）；神识类技能额外消耗神识值 |
