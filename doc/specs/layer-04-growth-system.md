# 第4层：成长系统规则书

> 版本：v1.0 | 最后更新：2026-07-15

---

## 概述

第4层定义**玩家怎么变强**。基于第1层（角色属性）和第2层（行动原子），将修炼、功法、装备、灵宠、技能、生活技能等子系统编织成完整的成长路径。

**设计原则**：
1. **时间沉淀**：变强需要真实时间（1 秒 = 1 游戏日），拒绝付费即满级。
2. **路径选择**：同一境界下，不同流派的成长曲线有显著差异，无唯一最优解。
3. **纵向衔接**：所有成长子系统使用统一修为值作为底层通货，确保修炼→突破→装备→炼丹形成经济闭环。
4. **横向预留**：本层只定义成长侧的数学关系和心法机制，具体装备属性表、丹方列表、术法参数表留作后续附录或上层细化。

## 一、修为与境界体系

### 1.1 修为值模型

修为值（Cultivation Value, CV）是衡量玩家修炼积累的核心数值，类比传统 MMO 的"经验值"。

| 属性 | 说明 |
|------|------|
| **定义** | 角色修炼积累的总量标度，决定境界是否圆满 |
| **获取来源** | 打坐修炼、战斗击杀、丹药、任务奖励、奇遇 |
| **表达式** | `修为值 = Σ(所有来源获取的修为)` |
| **境界圆满** | 当前境界修为值达到阈值后，可触发突破判定 |

### 1.2 境界体系总表

| 大境界 | 小境界数 | 突破方式 | 天劫触发 |
|--------|---------|----------|---------|
| 炼气 | 13 层 | 自动突破（修为值达标） | 无 |
| 筑基 | 3 层（初/中/后） | 手动突破，无天劫（小境界） | 大境界突破触发 3 波天雷 |

| 元婴 | 3 层（初/中/后） | 手动突破，高概率触发心魔 | 7 波天雷 |
| 化神 | 3 层（初/中/后） | 手动突破，必触心魔 | 9 波天雷 |
| 炼虚 | 3 层（初/中/后） | 手动突破 | 13 波天雷 |
| 合体 | 3 层（初/中/后） | 手动突破 | 17 波天雷 |

### 1.3 境界修为阈值

```
境界 n 的修为阈值 = 基础阈值 × (倍率 ^ (n-1))
```

| 参数 | 说明 | 参考值 |
|------|------|--------|
| 基础阈值（炼气1层 → 炼气2层） | 最低门槛 | 100 修为 |
| 相邻小境界倍率 | 每小层修为需求翻倍 | ×2.0 |
| 大境界突破倍率 | 大境界之间跳变系数 | ×10 |

**示例**（炼气期）：炼气1层→2层需 100，炼气2→3需 200，3→4需 400 ... 炼气12→13需 409,600。筑基期：筑基初期→中期约 100 万，中期→后期约 200 万，筑基→结丹约 1000 万。

> 精确曲线在第4层附录中给出完整数值表。

### 1.4 修为获取效率

```
修为获取速率 = 基础速率 × 灵气浓度系数 × 灵根效率系数 × 功法加成 × (1 + 悟性 × H%)
```

| 系数 | 影响因素 | 范围 |
|------|----------|------|
| 基础速率 | 当前境界 | 境界越高，基础打坐速率越快 |
| 灵气浓度 | 修炼地点（下品 0.5x / 中品 1.0x / 上品 2.0x / 极品 5.0x） | 0.1x - 5.0x |
| 灵根效率 | 天灵根 ×2.0 / 真灵根 ×1.0 / 伪灵根 ×0.5 | 0.5x - 2.0x |
| 功法加成 | 主修功法提供的修炼速度加成 | 1.0x - 3.0x |
| 悟性加成 | H 为常数，悟性越高效率越高 | +0% ~ +50% |

### 1.5 突破判定

#### 小境界突破（非大境界临界）

小境界突破为**自动突破**——修为值达到当前境界圆满阈值后，自动进入下一小层，无需任何操作，无失败惩罚。

#### 大境界突破

| 步骤 | 说明 |
|------|------|
| **前置条件** | 当前大境界最后一个小层修为值达到圆满 |
| **主动触发** | 玩家在安全区域（宗门/洞府/灵脉之上）主动发起突破 |
| **基础成功率** | `基础成功率 = 30% + (悟性 × A%) + (根骨 × B%)`（A、B 为常数，精确定义见附录） |
| **辅助修正** | 服用辅助丹药（+5%~+20%）、布置聚灵阵（+10%）、护法在场（每位 +5%，上限 3 人）、特殊功法加成 |
| **成功率硬上限** | 85%（保留随机性，拒绝 100% 突破） |
| **心魔触发** | 随境界渐进：小境界突破 10% 概率触发，大境界突破 30% 概率触发，化神及以上大境界突破 50% 概率触发（详见 L3 §3.1） |
| **天劫触发** | 大境界突破自动触发渡劫（见 L3 §3.2），渡劫失败 = 突破失败 |

#### 失败惩罚

```
修为损失 = 当前境界总修为 × P% × (1 - 毅力减免%)
```

| 突破场景 | P% | 额外惩罚 |
|----------|----|----------|
| 小境界（自动突破无需判定） | 0% | 无 |
| 大境界（无天劫，筑基以下） | 10% | 无 |
| 大境界（天劫失败） | 20% | 额外损失少量神识上限（临时，持续 N 小时） |
| 大境界（心魔失败） | 20% | 获得 debuff「心魔缠身」——修炼效率 -30%，持续 24 小时 |

> 毅力为隐藏属性（L1 §2.2），可通过特定剧情事件和师承任务提升。

### 1.6 境界压制

```
高境界对低境界的压制 = (高境界战力 - 低境界战力) × 境界压制系数
```

| 境界差 | 压制效果 |
|--------|----------|
| 同大境界（如筑基中期 vs 筑基初期） | 高者伤害 +5%，受伤害 -5% |
| 差 1 个大境界（如结丹 vs 筑基） | 高者伤害 +25%，受伤害 -20% |
| 差 2 个大境界（如元婴 vs 筑基） | 高者伤害 +60%，受伤害 -50%（战力的绝对碾压） |
| 差 3+ 个大境界 | 低者无法造成有效伤害（<5%），硬控技能免疫 |

> 境界压制是 PVE 的核心平衡机制——玩家打不过超 2 个大境界的野怪，需要策略或组队。

## 二、功法体系

### 2.1 功法定位

功法是修仙者的**核心修炼法门**，决定角色的基础成长曲线和战斗风格。不同于术法的"主动释放"，功法通过**周天运转**机制持续提供后台 buff。

| 维度 | 说明 |
|------|------|
| **是否消耗资源** | 不消耗真元/神识，被动生效 |
| **作用方式** | 持续提供属性加成 / 效率加成 / 特殊效果 |
| **是否可更换** | 可自由切换，但切换后有冷却（功法冷却：30 分钟） |
| **学习数量** | 可学习无上限，但**同时生效**的本命功法仅 1 部 |
| **辅修位** | 境界达筑基后解锁 1 个辅修位，元婴解锁第 2 个辅修位 |
| **主/辅关系** | 主修功法 100% 效果，辅修功法 50% 效果 |
| **功法等级** | 黄阶 → 玄阶 → 地阶 → 天阶 → 仙阶，每阶分上/中/下品 |

### 2.2 功法属性模板

每部功法包含以下属性维度：

| 属性 | 说明 | 示例 |
|------|------|------|
| **修炼速度加成** | 打坐/战斗时修为获取倍率 | 1.2x |
| **五维加成** | 对根骨/悟性/神识/身法/机缘的百分比加成 | 根骨 +10% |
| **战斗面板加成** | 对气血/真元/攻击/防御的百分比加成 | 气血 +15% |
| **五行抗性** | 对特定五行属性的伤害减免 | 火抗 +20% |
| **术法修炼速度** | 所属流派术法熟练度提升速率 | +30% |
| **特殊效果** | 独特的周天被动效果 | 每 10s 自动恢复 1% 气血 |

### 2.3 功法等级与属性成长

| 功法品阶 | 属性加成范围 | 修炼速度加成 | 学习条件 |
|---------|-------------|-------------|---------|
| 黄阶下品 | +2%~+5% | ×1.0 | 炼气1层 |
| 黄阶中品 | +3%~+8% | ×1.1 | 炼气4层 |
| 黄阶上品 | +5%~+10% | ×1.2 | 炼气7层 |
| 玄阶下品 | +8%~+15% | ×1.3 | 炼气10层 |
| 玄阶中品 | +10%~+20% | ×1.5 | 筑基初期 |
| 玄阶上品 | +15%~+25% | ×1.7 | 筑基后期 |
| 地阶下品 | +20%~+35% | ×2.0 | 结丹初期 |
| 地阶中品 | +25%~+45% | ×2.3 | 结丹后期 |
| 地阶上品 | +30%~+55% | ×2.7 | 元婴初期 |
| 天阶下品 | +40%~+70% | ×3.0 | 元婴后期 |
| 天阶中品 | +50%~+90% | ×3.5 | 化神期 |
| 天阶上品 | +60%~+120% | ×4.0 | 炼虚期 |
| 仙阶 | +80%~+200% | ×5.0 | 合体期以上，唯一性 |

### 2.4 功法获取方式

| 方式 | 说明 |
|------|------|
| **宗门藏经阁** | 加入宗门后，消耗宗门贡献兑换。获取黄阶~地阶功法的核心途径 |
| **秘境探索** | 秘境 Boss 掉落功法残卷，集齐可合成完整功法。可出地阶~天阶 |
| **师承传授** | 拜师后师父可传授独门功法（不可交易，唯一获取） |
| **奇遇任务** | 特定剧情线奖励，通常附带流派专属效果 |
| **交易市场** | 玩家间自由交易，功法价格由市场决定（L5 经济系统展开） |
| **转世保留** | 转世后保留已修炼的功法，无需重新获取 |

### 2.5 功法熟练度与升级

```
功法升级所需熟练度 = 基础值 × (等级 ^ 1.5)
```

| 属性 | 说明 |
|------|------|
| **熟练度获取** | 功法装备在槽位中时，随时间自动增长；战斗和打坐时额外加成 |
| **等级上限** | 功法品阶决定等级上限（黄阶 10 级，玄阶 20 级，地阶 30 级……） |
| **升级效果** | 每级提升该功法属性的 10%（即 10 级时属性翻倍） |
| **切换保留** | 切换功法不损失已累积的熟练度和等级 |

### 2.6 功法相生相克与搭配策略

#### 功法相生

同时装备主修+辅修功法时，若两者五行属性满足相生关系，触发额外加成：

| 相生组合（主 → 辅） | 触发效果 |
|--------------------|----------|
| 金主+水辅 | 真元恢复速度 +15% |
| 水主+木辅 | 气血恢复速度 +15% |
| 木主+火辅 | 术法威能 +15% |
| 火主+土辅 | 护体减伤 +15% |
| 土主+金辅 | 物理攻击 +15% |

#### 功法相克

若主修+辅修功法五行相克，则辅修功法效果进一步衰减至 30%：

| 相克组合（主 → 辅） | 惩罚效果 |
|--------------------|----------|
| 金主+木辅 | 辅修效果降至 30% |
| 木主+土辅 | 辅修效果降至 30% |
| 土主+水辅 | 辅修效果降至 30% |
| 水主+火辅 | 辅修效果降至 30% |
| 火主+金辅 | 辅修效果降至 30% |

#### 流派推荐搭配

| 流派 | 推荐主修属性 | 推荐辅修属性 | 核心思路 |
|------|------------|-------------|----------|
| 锻体流 | 土系功法（增加气血+防御） | 金系（相生触发攻击加成） | 肉身成圣，硬度优先 |
| 法修流 | 火系功法（增加术法威能） | 木系（相生触发术法再 +15%） | 极致爆发，玻璃大炮 |
| 剑修/御器流 | 金系功法（增加穿透+攻击） | 水系（相生触发真元恢复） | 持续性远程压制 |
| 召唤/阵法流 | 木系功法（增加气血恢复+控制） | 火系（相生触发术法威能+灵宠输出） | 人宠双线，续航拉满 |

### 2.7 原著传承功法

以下为《凡人修仙传》原著核心功法，作为特殊获取的高价值功法保留：

| 功法名称 | 品阶 | 特点 |
|---------|------|------|

| **三转重元功** | 天阶 | 散功重修型功法，每次转修获得额外的属性加成。与转世重修机制挂钩 |

## 三、功法与术法体系

### 3.1 术法分类

| 分类 | 说明 | 示例 |
|------|------|------|
| **主动术法** | 战斗中手动释放，消耗真元/神识 | 小火球术、飞剑穿刺、金刚罩 |
| **被动术法** | 装备后自动生效，不消耗资源 | 灵气亲和（真元恢复 +10%）、铁骨（气血 +5%） |
| **通用术法** | 所有流派均可学习的基础术法 | 基础吐纳（打坐效率 +10%） |
| **流派术法** | 仅特定流派可学习 | 锻体：崩山掌；法修：烈焰风暴 |
| **传承术法** | 师承/宗门/奇遇专属 | 青元剑诀·第一式、大衍诀·神念锁 |
| **五行术法** | 带五行属性标签，受灵根纯度和克制影响 | 三味真火（火）、玄冰诀（水） |

### 3.2 技能获取方式

| 方式 | 说明 |
|------|------|
| **灵根解锁** | 某属性灵根纯度达到阈值后，自动解锁对应属性的基础技能树 |
| **功法附赠** | 学习功法时，随功法附带 1-3 个专属技能 |
| **宗门技能导师** | 消耗宗门贡献学习流派技能 |
| **技能书掉落** | 怪物/Boss 掉落技能书，使用后习得 |
| **师承传授** | 师父直接传授技能（可跳过学习前置条件） |
| **奇遇/任务** | 特定剧情线奖励稀有技能 |

### 3.3 技能树结构

```
流派技能树（以法修·火系为例）
├── 基础层（灵根需求：火≥20）
│   ├── 小火球术         ← 初始自带
│   └── 火焰护盾         ← 消耗 100 修为解锁
├── 进阶层（灵根需求：火≥40，境界≥筑基）
│   ├── 连珠火球         ← 消耗 1000 修为
│   ├── 烈焰风暴（AOE）  ← 消耗 2000 修为
│   └── 火遁术           ← 消耗 1500 修为
├── 专家层（灵根需求：火≥60，境界≥结丹）
│   ├── 地狱火海         ← 消耗 10000 修为
│   ├── 焚天诀·第一式    ← 消耗 15000 修为 + 功法《焚天诀》
│   └── 火灵化身         ← 消耗 20000 修为（召唤火灵辅助战斗）
└── 大师层（灵根需求：火≥80，境界≥元婴）
    ├── 九阳焚天         ← 消耗 100000 修为
    └── 凤凰涅槃         ← 消耗 200000 修为（复活技能，长冷却）
```

### 3.4 术法升级

```
技能等级 N 所需熟练度 = 基础值 × (N ^ 1.8)
```

| 属性 | 说明 |
|------|------|
| **熟练度获取** | 每次释放术法获得固定熟练度 + 命中目标额外加成 |
| **等级上限** | 受境界限制（炼气最高 5 级，筑基 10 级，结丹 15 级……） |
| **升级效果** | 每级提升基础伤害/效果 +10%，减少消耗 -2%（上限 -30%） |
| **术法槽位** | 初始 3 主动 + 1 被动；随境界逐步解锁，总上限 8 主动 + 4 被动（解锁日程见下表） |

### 3.5 流派术法配置策略

| 流派 | 推荐主动术法组合倾向 | 推荐被动术法组合倾向 |
|------|---------------------|---------------------|
| 锻体流 | 单体爆发 + 控制 + 减伤 + 突进 | 气血加成 + 防御加成 + 韧性 |
| 法修流 | AOE + 单体高伤 + 护盾 + 位移 | 真元上限 + 法术穿透 + 暴击 |
| 剑修流 | 远程穿刺 + 破甲 + 位移 + 连招终结 | 神识上限 + 攻击速度 + 破甲 |
| 召唤流 | 灵宠强化 + 控制 + 阵法 + 回血 | 灵宠属性加成 + 召唤数量 + 羁绊 |

## 四、装备成长维度

### 4.1 装备槽位

| 槽位 | 说明 | 受控属性 |
|------|------|----------|
| **武器/法宝** | 主手武器：飞剑/法杖/拳套/灵幡，流派差异 | — |
| **法袍** | 防具主位，提供防御和抗性 | 根骨影响穿戴门槛 |

| **灵宠** | 灵宠槽位 | 神识决定同时可携带数量 |
| **储物袋** | 基础背包容量 + 扩展 | 神识决定基础容量 |

### 4.2 装备品阶体系

```
凡器 → 法器 → 灵器 → 宝器 → 道器 → 仙器
```

| 品阶 | 基础属性倍率 | 强化上限 | 获取途径 |
|------|-------------|---------|----------|
| 凡器 | ×1.0 | +5 | 商店基础售卖、怪物掉落 |
| 法器 | ×1.5 | +8 | 炼器产出、精英怪掉落 |
| 灵器 | ×2.5 | +12 | 炼器产出、Boss 掉落 |
| 宝器 | ×4.0 | +16 | 高阶炼器、秘境 Boss |
| 道器 | ×7.0 | +20 | 世界 Boss、特殊奇遇 |
| 仙器 | ×12.0 | +25 | 转世重修、唯一性任务 |

### 4.3 装备强化

```
强化 N 级属性 = 基础属性 × (1 + N × 单级提升率)
```

| 属性 | 值 |
|------|-----|
| 单级提升率 | +8%（攻击类）/ +6%（防御类） |
| 强化上限 | 装备品阶决定（见 4.2） |
| 强化消耗 | 灵石 + 对应品阶的强化石 |
| 强化成功率 | 每级递减：+1 级 100%，+2 级 95%，+3 级 85%……+10 级 30% |
| 失败惩罚 | +1~+5：无惩罚；+6~+10：强化等级 -1；+11+：强化等级归零或装备损坏 |
| 保护符 | 特殊道具可防止失败降级/损坏（稀有，Boss 掉落/商城） |

### 4.4 装备升阶

升阶是将装备从低品阶提升到高品阶的途径，保留强化等级和特殊属性。

| 属性 | 说明 |
|------|------|
| **前置条件** | 装备强化等级达到当前品阶上限的 80% |
| **消耗** | 升阶石 + 同品阶装备碎片 + 大量灵石 |
| **成功率** | `基础成功率 40% + (机缘 × K%)`，上限 70% |
| **失败惩罚** | 材料全损，装备保留 |
| **升阶后** | 品阶+1，强化等级不变，基础属性使用新品阶的计算公式 |

### 4.5 附魔

| 属性 | 说明 |
|------|------|
| **附魔槽** | 每件装备 1-3 个附魔槽（品阶越高槽位越多） |
| **附魔属性** | 从属性池中随机获取：+N% 某五行攻击 / +N 气血 / +N% 暴击等 |
| **附魔材料** | 妖兽精魂 / 秘境特产 / 特殊掉落 |
| **替换** | 新附魔覆盖旧附魔（不可叠加） |
| **完美附魔** | 极小概率触发「完美附魔」——属性值 ×1.5，有特殊视觉效果 |

### 4.6 法宝培养

法宝作为修仙世界的核心装备，拥有独立的培养体系：

| 功能 | 说明 |
|------|------|
| **温养** | 装备在法宝槽中随时间自动增长契合度，提升法宝技能效果（每级 +2%，上限 30%） |
| **祭炼** | 消耗材料和真元主动提升法宝品阶，同时提升基础属性和解锁新法宝技能 |
| **法宝技能** | 每个法宝自带 1-2 个专属技能，装备后自动加入法宝技能槽（4 槽位） |
| **法宝耐久** | 战斗中法宝受到攻击会损耗耐久，耐久归零后法宝暂时失效，需修复 |
| **法宝升级** | 通过吞噬其他法宝/同属性材料提升等级，等级上限受境界限制 |

## 五、灵宠养成

### 5.1 灵宠基础

| 属性 | 说明 |
|------|------|
| **获取** | 野外捕捉、秘境掉落灵宠蛋、师承赠送、宗门兑换 |
| **品阶** | 凡兽 → 灵兽 → 妖兽 → 荒兽 → 灵兽王 → 神兽 |
| **携带上限** | `基础 1 只 + (神识 / N) 只`，上限 5 只（N 为常数） |
| **出战** | 同时只能出战 1 只灵宠，其余在灵兽袋中待命 |
| **灵宠境界** | 独立境界体系，与主人境界解耦；可通过培养提升 |

### 5.2 灵宠属性

| 属性类别 | 包含属性 | 说明 |
|---------|---------|------|
| **战斗属性** | 气血、攻击、防御、速度、技能 | 独立战斗面板，参与战斗时独立计算 |
| **五行属性** | 金/木/水/火/土 灵根 | 决定灵宠的技能属性和克制关系 |
| **成长资质** | 各项属性的成长倍率（0.5x~2.0x） | 决定每级成长幅度，品阶越高资质越好 |
| **忠诚度** | 0-100 | 影响灵宠是否听从指令、是否会逃跑。低于 30 时可能叛逃 |
| **亲密度** | 0-1000 | 影响合击触发概率、特殊技能解锁、骑乘功能解锁 |

### 5.3 灵宠升级

```
灵宠升级所需修为 = 灵宠等级 × 品阶系数 × 100
```

| 方式 | 说明 |
|------|------|
| **战斗经验** | 出战灵宠参与战斗获得经验（为主人所得修为的 20%） |
| **喂养丹药** | 消耗专用灵宠丹药快速提升等级（丹药由炼丹产出） |
| **灵兽袋修养** | 在灵兽袋中随时间缓慢增长经验（挂机收益） |

### 5.4 灵宠喂养系统

| 喂养类型 | 消耗品 | 效果 |
|---------|--------|------|
| **基础饲料** | 灵谷/灵草（商店购买） | 每日基础喂养，维持忠诚度不下降 |
| **成长丹药** | 启灵丹/培元丹（炼丹产出） | 提升成长资质（上限由品阶决定） |
| **属性丹药** | 五行灵丹（炼丹产出） | 提升特定五行属性的攻击/抗性 |
| **稀有食材** | 妖兽精血/天材地宝（掉落/采集） | 概率解锁隐藏术法或进化 |
| **零食** | 灵果/蜜饯（商店） | 提升亲密度（亲密度 > 500 可解锁骑乘） |

### 5.5 骑乘功能

| 属性 | 说明 |
|------|------|
| **解锁条件** | 灵宠亲密度 ≥ 500，且灵宠体型足够承载玩家 |
| **移动加成** | 大地图移动速度 +40%~+100%（取决于灵宠品阶/速度资质） |
| **战斗状态** | 骑乘时不可使用普攻连段，但可使用特定坐骑技能 |
| **御剑替代** | 非骑乘型灵宠的玩家仍可使用御剑飞行（L3 §2.6），两者互斥 |
| **特殊骑宠** | 某些灵宠拥有特殊地形通行能力（如飞行、涉水、钻地） |

### 5.6 灵宠辅助功能

| 功能 | 解锁条件 | 效果 |
|------|----------|------|
| **自动拾取** | 灵宠等级 ≥ 10 | 自动拾取战利品（可切换过滤规则） |
| **警戒** | 忠诚度 ≥ 60 | 周围有敌对目标时自动预警 |
| **寻宝** | 特定灵宠天赋 | 小范围内显示隐藏宝箱/采集物位置 |
| **炼丹助手** | 亲密度 ≥ 300 + 木属性灵宠 | 炼丹成功率 +5% |
| **护主** | 忠诚度 ≥ 80 | 主人受到致命伤害时，灵宠自动替死（冷却 24 小时） |

### 5.7 灵宠进化

| 条件 | 说明 |
|------|------|
| **进化学** | 达到当前品阶等级上限 + 消耗特定进化材料 + 亲密度 ≥ 800 |
| **进化结果** | 品阶 +1，资质上限提高，解锁新技能/新外形 |
| **进化失败** | 材料消耗，灵宠属性不变，亲密度 -50 |
| **保留** | 进化后强化等级/术法/忠诚度保留 |

### 5.8 灵宠与召唤流联动

灵宠系统与召唤流（L3 §2.5）存在深度联动，形成"人宠双线"战术体系：

| 联动维度 | 说明 |
|----------|------|
| **召唤物槽位** | 召唤物与灵宠共享出战槽位池——神识上限 = 灵宠出战数 + 召唤物数（详见 L3 §2.5） |
| **羁绊加成** | 灵宠与召唤物同时在场时，互相获得属性加成（灵宠+召唤物 各 +10% 全属性） |
| **合体技** | 特定灵宠+特定召唤类型可触发合体技（如「火灵 + 火系召唤物」触发「烈焰双龙」——范围伤害 ×1.5） |
| **协同 AI** | 召唤流功法可让灵宠和召唤物共享仇恨和目标选择，避免各自为战 |
| **流派定位** | 召唤流玩家优先培养高品阶灵宠，灵宠品阶影响召唤物基础属性（品阶每升 1 级，召唤物全属性 +5%） |

## 七、转世重修体系

### 7.1 触发条件

| 方式 | 说明 |
|------|------|
| **主动转世** | 达到当前境界上限后，玩家可在 NPC 处主动发起转世 |
| **寿元耗尽** | 每个境界有寿元上限（炼气 100 年、筑基 200 年……），耗尽后强制转世 |
| **特殊道具** | 「轮回玉」可在任意时刻触发转世（稀有道具） |

### 7.2 保留规则

| 保留项 | 比例/说明 |
|--------|----------|
| **灵根纯度** | 保留 80%，获得额外的「轮回点数」可重新分配 |
| **功法熟练度** | 已修炼的功法全部保留，但等级重置为 Lv.1；转世后功法效果受转世衰减影响（见 §7.2a） |
| **装备** | 全部保留 |
| **灵宠** | 全部保留 |
| **修为值** | 归零（回到炼气1层），但转世后修炼速度永久加成 |
| **术法熟练度** | 全部保留 |

### 7.2a 转世后功法效果衰减

转世并非"无损轮回"——功法的生效效果会随转世次数逐步衰减，需在当世重新修炼以恢复全效：

| 转世次数 | 功法效果衰减 | 恢复条件 |
|----------|------------|----------|
| 1 转（一世） | -10% | 在当世将功法修炼至 Lv.5 恢复全效 |
| 2 转（二世） | -20% | 在当世将功法修炼至 Lv.10 恢复 |
| 3 转（三世） | -30% | 在当世将功法修炼至 Lv.15 恢复 |
| 4 转及以上 | -40%（上限） | 在当世将功法修炼至 Lv.20 恢复 |

> 衰减仅影响功法效果数值（属性加成 / 修炼效率 / 战斗 buff），不影响功法本身的存在和可修炼状态。辅修功法同样受衰减影响。

### 7.3 转世收益

| 收益 | 说明 |
|------|------|
| **转世修炼加成** | 每转世 1 次，全境界修炼速度永久 +5%（上限 +50%） |
| **轮回点数** | 可用于提升灵根纯度上限（突破原 100 点上限）或解锁异灵根 |
| **前世传承** | 转世后可解锁专属传承技能树，仅转世玩家可学 |
| **称号** | 转世次数对应称号：一世散修→二世重修→三世老怪…… |

### 7.4 经济闭环

转世是成长系统的最终循环，也是经济系统（L5）的锚点：

```
修炼 → 突破 → 装备强化 → 渡劫 → 转世 → 保留功法 + 装备
  ↑                                                    |
  └────────── 轮回点数 → 更高灵根 → 更快修炼 ──────────┘
```

转世后玩家回到低境界，但保留功法和装备，形成"高属性低境界"的压制周期。每次转世缩短到达上一世最高境界的时间。

### 8.2 各流派成长节奏差异
