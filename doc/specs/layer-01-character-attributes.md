## 一、灵根系统

### 1.1 原著灵根体系参考

以下为《凡人修仙传》原著灵根分类，作为本项目设计的世界观基线。游戏侧设计在此基础上做了部分差异化（见 1.3），二者不完全一致。

#### 原著灵根分类

| 灵根类型 | 属性构成 | 原著定位 |
|----------|----------|----------|
| **天灵根** | 单一属性（金/木/水/火/土任一） | 修炼速度 2-3 倍，无瓶颈结丹，数百年一遇。灵根中的极品 |
| **真灵根** | 双灵根（二灵根）或三灵根 | 正常修炼速度，大派招收弟子的最低门槛。筑基、结丹均有希望 |
| **伪灵根** | 四灵根或五灵根 | 修炼极慢，五行灵气相互干扰，基本无望筑基。韩立即四灵根伪灵根 |
| **异灵根** | 两种五行属性异变融合 | 修炼速度介于天灵根与真灵根之间，无结丹豁免，但搭配适配功法则实力极强（见 1.5） |

> **原著核心设定**：四灵根/五灵根之所以"废"，是因为五行灵气同时入体互相干扰——"金克木"导致吸纳木灵气时被体内金灵根压制，修炼事倍功半。这正是五行相生相克在角色身上的直接体现。

#### 原著与本项目的核心差异

| 对比维度 | 原著 | 本项目 |
|----------|------|--------|
| 天灵根 | 仅单一属性，完全无其他 | 允许天灵根有少量次属性（如 90火 / 10木），暂不修改 |
| 伪灵根定位 | 基本废材，无后期出路 | 可修炼「大五行混元功」，后期弯道超车 |
| 变异灵根 | 创角时天生异变 | 可在游戏过程中通过特殊机缘触发灵根异变 |

### 1.2 创角分配

玩家创建角色时，拥有 **100 点灵根纯度**，可自由分配给五行属性：

> 【金】【木】【水】【火】【土】

每个属性最低 0 点，最高 100 点。

### 1.3 纯度决定高度

| 分配模式 | 示例 | 效果 |
|----------|------|------|
| **天灵根**（极限偏科） | 90火 / 10木 | 主属性技能学习速度极快，威力有额外百分比加成；但完全无法学习克制属性的技能。适合极致单流派 Build |
| **真灵根**（双修） | 60火 / 40木 | 两个属性均有加成，灵活切换，但单个属性不如天灵根极致 |
| **杂灵根 / 伪灵根**（平均） | 各 20 点 | 所有属性功法学习速度慢、威力平庸；但后期可修炼极为苛刻的 **大五行混元功**，触发五行相生羁绊，实现弯道超车 |

### 1.4 五行相生相克

五行相生相克是本项目的**核心战斗系统**之一。所有五行属性攻击、防御、技能威力均受此制约。

#### 相生关系（生我者母，我生者子）

```
金 → 水 → 木 → 火 → 土 → 金 （循环相生）
```

| 相生方向 | 触发效果 |
|----------|----------|
| **金生水** | 金灵根修士施放水系辅助法术时，效果 +15% |
| **水生木** | 水灵根修士施放木系辅助法术时，效果 +15% |
| **木生火** | 木灵根修士施放火系辅助法术时，效果 +15% |
| **火生土** | 火灵根修士施放土系辅助法术时，效果 +15% |
| **土生金** | 土灵根修士施放金系辅助法术时，效果 +15% |

#### 相克关系（我克者，克制加成；克我者，受制削弱）

```
金 → 木 → 土 → 水 → 火 → 金 （循环相克）
```

| 克制方向 | 口诀 | 攻击方加成 | 被克方削弱 |
|----------|------|-----------|-----------|
| **金克木** | 金断木 | 金系对木系伤害 +25% | 木系对金系伤害 -25% |
| **木克土** | 木破土 | 木系对土系伤害 +25% | 土系对木系伤害 -25% |
| **土克水** | 土掩水 | 土系对水系伤害 +25% | 水系对土系伤害 -25% |
| **水克火** | 水熄火 | 水系对火系伤害 +25% | 火系对水系伤害 -25% |
| **火克金** | 火熔金 | 火系对金系伤害 +25% | 金系对火系伤害 -25% |

> **克制加成 25%** 为 PV 及 BOSS 战核心数值锚点，具体数值可在战斗系统层调参。克制效果作用于最终伤害计算之前。

#### 五行属性与战斗风格对应

| 灵根属性 | 核心特点 | 战斗风格与定位 |
|----------|----------|---------------|
| **金** (Metal) | 极致锋锐，单体爆发 | 擅长御使飞剑法宝，拥有最高的单体物理及法术穿透。PVP 刺客型、单秒爆发型 |
| **木** (Wood) | 生生不息，治疗控制 | 强大生命恢复和缠绕束缚类控制技能。团队治疗者与辅助，生存能力极强 |
| **水** (Water) | 柔中带刚，减速冰冻 | 精通水系与冰系法术，减速、冰冻、控场、净化。群体作战的控场大师，法力雄浑 |
| **火** (Fire) | 焚天灭地，群体灼烧 | 强大范围伤害和持续灼烧效果。清理大量妖兽的效率之王，输出冠绝五行 |
| **土** (Earth) | 厚德载物，坚不可摧 | 天生极高防御和减伤效果。团队最可靠的守护者（坦克），擅长筑墙、护盾、嘲讽 |

### 1.5 异灵根系统

异灵根（原著称"变异灵根"）由两种五行属性在特定条件下异变融合而成，诞生出全新的灵根属性。异灵根修炼速度介于天灵根与真灵根之间，虽无结丹豁免，但搭配适配功法后实力极强——通常能顶 3-4 个同等实力的普通修士。

#### 异灵根生成规则

| 异灵根 | 本源属性 | 核心特点 | 特点 |
|--------|---------|----------|------|
| **雷灵根** | 金 + 水 | 爆发性极强，战斗系顶尖 | 速度与攻击力俱佳，PVP 王者属性 |
| **冰灵根** | 金 + 水（水为主导） | 控制力极强，持久性好 | 冻结、冰封禁锢，冰系控场能力远超普通水灵根 |
| **风灵根** | 木 + 火 | 身法卓绝，难以捉摸 | 御风飞行速度远超同阶，风属性法术极快 |
| **暗灵根** | 土 + 金（隐变） | 隐匿系，擅幻术与诡道 | 潜行、暗杀、幻术，PVP 偷袭与暗杀流核心 |

#### 异灵根相克关系

异灵根之间存在独立的相克循环，仅作用于异灵根之间的对战：

```
暗 → 雷 → 冰 → 风 → 暗 （异灵根四象克制循环）
```

| 克制方向 | 效果 | 备注 |
|----------|------|------|
| **暗克雷** | 暗系对雷系伤害 +20% | 原著设定 |
| **雷克冰** | 雷系对冰系伤害 +20% | 原著设定 |
| **冰克风** | 冰系对风系伤害 +20% | 原著设定 |
| **风克暗** | 风系对暗系伤害 +20% | 风能吹散暗影 |

> 异灵根之间的相克影响上限约 **10-20%**（弱于五行相克的 25%），以体现"异灵根本身已是稀有强者"的设定。

#### 异灵根获取方式

| 方式 | 说明 |
|------|------|
| **创角天赋** | 极小概率（约 5%）在创角时天生异变，随机获得一种异灵根（需至少投入对应两种本源属性各 ≥ 15 点） |
| **游戏内异变** | 修炼到特定境界后，满足两种本源属性纯度 ≥ 30 点时，消耗特殊道具「异灵丹」触发异变 |
| **奇遇 / 秘境** | 特定秘境中极低概率遇到"天雷淬体"、"寒潭洗髓"等事件，直接触发灵根异变 |

> 异灵根一旦获得，原五行纯度将保留（允许继续修炼原属性），但为异灵根属性提供额外技能树和伤害加成。

### 1.6 灵根的影响范围

| 影响维度 | 说明 |
|----------|------|
| **技能学习** | 决定该属性技能的学习速度与可学习上限（天灵根 90 火的玩家学火系技能极快，但完全学不了水系） |
| **技能威力** | 灵根纯度给对应属性技能提供威力百分比加成 |
| **初始技能** | 纯度最高的那一系，开局自动领悟该系 1 个基础技能（如最高火 → 自带《小火球术》） |
| **五行抗性** | 灵根分配影响角色对不同属性攻击的抗性（具体公式详见战斗系统层） |
| **五行相克** | PVP 和 BOSS 战中，属性克制提供 ±25% 伤害修正（见 1.4） |

### 1.7 扩展属性体系

角色基础属性为五行 + 异灵根，游戏中存在超稀有扩展属性：

| 属性类型 | 示例 | 获取方式 |
|----------|------|----------|
| **暗**（非异灵根暗） | 魔物攻击属性 | 后期神器类装备赋予 |
| **神 / 圣** | 特殊攻击属性 | 后期神器类装备赋予 |
| **灵体 / 无属性** | 特殊攻击属性 | 后期神器类装备赋予 |
| **光灵根** | 传说属性，五行合一的极致异变 | 仅在古籍中有记载，无人见过——作为终极隐藏彩蛋保留 |

> 扩展属性不占灵根分配，仅作为后期装备系统的深度拓展。

### 1.8 灵根调整机制

| 方式 | 说明 |
|------|------|
| **RMB 道具**：洗髓丹 / 夺舍丹 | 重置灵根纯度，重新分配（成本高，不轻易使用） |
| **境界突破** | 突破到特定大境界时，获得少量额外灵根点数可分配（模拟"脱胎换骨"） |
| **转世重修** | 保留部分修为 + 部分灵根点数，重新投胎，重新分配（大后期核心消耗系统） |

## 二、核心五维属性

替代传统 MMO 的力量/智力/敏捷/体质/精神，采用修仙世界观重构。

玩家每升一级获得 **自由属性点**，可任意分配到以下五维。

### 2.1 五维定义

| 属性 | 对应传统概念 | 核心影响范围 |
|------|-------------|-------------|
| **根骨** (Physique) | 力量 + 体质 | 气血上限、体修杀伤（锻体技能伤害）、肉身强横度（物理防御）、负重（可穿戴重甲/重法宝的门槛） |
| **悟性** (Comprehension) | 智力 + 法强 | 术法威能（法术伤害）、真元上限（MP）、功法学习速度、突破瓶颈所需经验减免、阵法复杂度与威力 |
| **神识** (Divine Sense) | 精神 + 感知 | 真元上限（MP）、御器类（法宝）操控数量与威力、灵宠携带数量与羁绊强度、神识锁定（命中率） |
| **身法** (Agility) | 敏捷 + 速度 | 出手速度（回合制）/ 移动速度 + 冲刺无敌帧（ARPG）、迷踪幻影（闪避率） |
| **机缘** (Fortune) | 幸运 | 会心一击（暴击率）+ 会心倍率（暴击伤害）、怪物掉落极品装备的概率（上限 2 倍基础掉率，不可无限堆叠）、奇遇事件走向 |

### 2.2 隐藏属性：毅力

| 属性 | 说明 |
|------|------|
| **毅力** (Willpower) | 不显示在面板上的被动天赋属性。决定异常状态抗性（抗晕眩/抗心魔/抗恐惧）、突破失败时保留修为的比例 |

> 毅力可通过特定剧情事件、师承任务获得提升，不可通过升级加点。

## 三、战斗面板（二次换算）

以五维为基数，通过公式计算得出战斗面板数值。以下为映射关系，精确公式在战斗系统层定义。

### 3.1 攻防属性

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 生命值 (HP) | **气血上限** | 根骨 |
| 魔法值 (MP) | **真元上限** | 神识 + 悟性（双属性共同决定） |
| 物理攻击 | **体修杀伤** | 根骨 |
| 魔法攻击 | **术法威能** | 悟性 |
| 物理防御 | **肉身强横度** | 根骨 + 装备 |
| 魔法防御 | **护体罡气** | 真元上限 + 悟性 |

### 3.2 命中与闪避

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 命中率 | **神识锁定** | 神识越高，敌人越难闪避你的飞剑 |
| 闪避率 | **迷踪幻影** | 身法 |

### 3.3 暴击

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 暴击率 | **会心一击** | 机缘（类似 RO 的 LUK 设定） |
| 暴击伤害 | **会心倍率** | 机缘 + 悟性（运气好 + 找准了功法破绽） |

### 3.4 经济属性

| 属性 | 影响范围 | 上限 |
|------|----------|------|
| 掉落加成 | 机缘越高，怪物掉落极品装备概率越高 | **最大 2.0 倍**基础掉率，不可无限堆叠 |

## 四、流派 Build 与加点绑定

### 4.1 四大主流流派

| 流派 | 主加属性 | 辅加属性 | 战斗风格 |
|------|----------|----------|----------|
| **锻体流**（一力降十会） | 根骨 | 身法 | 高血量 + 高体修杀伤，贴身肉搏，拳拳到肉。技能消耗真元极少 |
| **法修流**（狂轰滥炸） | 悟性 | 神识 | 学习全屏 AOE 仙术，身板脆但清怪极快。真元消耗高，需管理资源 |
| **剑修 / 御器流**（飞剑千里） | 神识 | 悟性 | 操控多把飞剑/法宝远程输出，自己不近身。依赖神识操控上限 |
| **召唤 / 阵法流**（安逸挂机） | 神识 | 机缘 | 靠阵法困敌，灵宠输出。培养成本高但操作安逸 |

### 4.2 流派与真元消耗

| 流派 | 真元消耗特征 |
|------|-------------|
| 锻体流 | 技能消耗真元极少，可持续输出，但需近身 |
| 法修流 | 技能消耗真元高，爆发强，需管理资源循环 |
| 剑修 / 御器流 | 中等消耗，平稳输出 |
| 召唤 / 阵法流 | 阵法一次性大消耗，灵宠持续输出不耗真元 |

## 五、真元系统（法力资源）

### 5.1 真元上限

真元上限由 **神识** 和 **悟性** 共同决定（双属性加权）。

### 5.2 真元消耗

所有流派的技能均消耗真元。锻体流消耗极小，法修流消耗大。

### 5.3 真元回复

| 回复方式 | 说明 |
|----------|------|
| **自然回复** | 战斗中自动回复，回复速率由神识决定（每秒 N 点） |
| **打坐** | 非战斗状态主动打坐，回复速率大幅提升 |
| **丹药** | 战斗中可使用回蓝丹药（有冷却） |
| **灵兽技能** | 特定灵兽可提供真元回复光环或主动回蓝技能 |
| **特殊技能** | 类似 WOW 法师"唤醒"的爆发回蓝技能，战斗中可用但有长冷却（法修专属） |
