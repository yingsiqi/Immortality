> 版本：v1.0 | 最后更新：2026-07-14

## 一、灵根系统

### 1.1 创角分配

玩家创建角色时，拥有 **100 点灵根纯度**，可自由分配给五行属性：

> 【金】【木】【水】【火】【土】

每个属性最低 0 点，最高 100 点。

### 1.2 纯度决定高度

| 分配模式 | 示例 | 效果 |
|----------|------|------|
| **天灵根**（极限偏科） | 90火 / 10木 | 主属性技能学习速度极快，威力有额外百分比加成；但完全无法学习克制属性的技能。适合极致单流派 Build |
| **真灵根**（双修） | 60火 / 40木 | 两个属性均有加成，灵活切换，但单个属性不如天灵根极致 |
| **杂灵根 / 伪灵根**（平均） | 各 20 点 | 所有属性功法学习速度慢、威力平庸；但后期可修炼极为苛刻的 **大五行混元功**，触发五行相生羁绊，实现弯道超车 |

### 1.3 灵根的影响范围

| 影响维度 | 说明 |
|----------|------|
| **技能学习** | 决定该属性技能的学习速度与可学习上限（天灵根 90 火的玩家学火系技能极快，但完全学不了水系） |
| **技能威力** | 灵根纯度给对应属性技能提供威力百分比加成 |
| **初始技能** | 纯度最高的那一系，开局自动领悟该系 1 个基础技能（如最高火 → 自带《小火球术》） |
| **五行抗性** | 灵根分配影响角色对不同属性攻击的抗性（具体公式详见战斗系统层） |

### 1.4 扩展属性体系

角色基础属性为五行，游戏中存在扩展属性：

| 属性类型 | 示例 | 获取方式 |
|----------|------|----------|
| 暗 | 魔物攻击属性 | 后期神器类装备赋予 |
| 神 / 圣 | 特殊攻击属性 | 后期神器类装备赋予 |
| 灵体 / 无属性 | 特殊攻击属性 | 后期神器类装备赋予 |

> 扩展属性不占灵根分配，仅作为后期装备系统的深度拓展。属性相克表在第3层战斗系统中详细定义。

### 1.5 灵根调整机制

| 方式 | 说明 |
|------|------|
| ** RMB 道具**：洗髓丹 / 夺舍丹 | 重置灵根纯度，重新分配（成本高，不轻易使用） |
| **境界突破** | 突破到特定大境界时，获得少量额外灵根点数可分配（模拟"脱胎换骨"） |
| **转世重修** | 保留部分修为 + 部分灵根点数，重新投胎，重新分配（大后期核心消耗系统） |

## 二、核心五维属性

替代传统 MMO 的力量/智力/敏捷/体质/精神，采用修仙世界观重构。

玩家每升一级获得 **自由属性点**，可任意分配到以下五维。

### 2.1 五维定义

| 属性 | 对应传统概念 | 核心影响范围 |
|------|-------------|-------------|
| **根骨** (Physique) | 力量 + 体质 | 气血上限、体修杀伤（锻体技能伤害）、肉身强横度（物理防御）、负重（可穿戴重甲/重法宝的门槛） |
| **悟性** (Comprehension) | 智力 + 法强 | 术法威能（法术伤害）、真元上限（MP）、功法学习速度、突破瓶颈所需经验减免、阵法复杂度与威力 |
| **神识** (Divine Sense) | 精神 + 感知 | 真元上限（MP）、御器类（法宝）操控数量与威力、灵宠携带数量与羁绊强度、神识锁定（命中率） |
| **身法** (Agility) | 敏捷 + 速度 | 出手速度（回合制）/ 移动速度 + 冲刺无敌帧（ARPG）、迷踪幻影（闪避率） |
| **机缘** (Fortune) | 幸运 | 会心一击（暴击率）+ 会心倍率（暴击伤害）、怪物掉落极品装备的概率（上限 2 倍基础掉率，不可无限堆叠）、奇遇事件走向 |

### 2.2 隐藏属性：毅力

| 属性 | 说明 |
|------|------|
| **毅力** (Willpower) | 不显示在面板上的被动天赋属性。决定异常状态抗性（抗晕眩/抗心魔/抗恐惧）、突破失败时保留修为的比例 |

> 毅力可通过特定剧情事件、师承任务获得提升，不可通过升级加点。

## 三、战斗面板（二次换算）

以五维为基数，通过公式计算得出战斗面板数值。以下为映射关系，精确公式在战斗系统层定义。

### 3.1 攻防属性

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 生命值 (HP) | **气血上限** | 根骨 |
| 魔法值 (MP) | **真元上限** | 神识 + 悟性（双属性共同决定） |
| 物理攻击 | **体修杀伤** | 根骨 |
| 魔法攻击 | **术法威能** | 悟性 |
| 物理防御 | **肉身强横度** | 根骨 + 装备 |
| 魔法防御 | **护体罡气** | 真元上限 + 悟性 |

### 3.2 命中与闪避

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 命中率 | **神识锁定** | 神识越高，敌人越难闪避你的飞剑 |
| 闪避率 | **迷踪幻影** | 身法 |

### 3.3 暴击

| 战斗属性 | 仙侠名称 | 主要决定因素 |
|----------|----------|-------------|
| 暴击率 | **会心一击** | 机缘（类似 RO 的 LUK 设定） |
| 暴击伤害 | **会心倍率** | 机缘 + 悟性（运气好 + 找准了功法破绽） |

### 3.4 经济属性

| 属性 | 影响范围 | 上限 |
|------|----------|------|
| 掉落加成 | 机缘越高，怪物掉落极品装备概率越高 | **最大 2.0 倍**基础掉率，不可无限堆叠 |

## 四、流派 Build 与加点绑定

### 4.1 四大主流流派

| 流派 | 主加属性 | 辅加属性 | 战斗风格 |
|------|----------|----------|----------|
| **锻体流**（一力降十会） | 根骨 | 身法 | 高血量 + 高体修杀伤，贴身肉搏，拳拳到肉。技能消耗真元极少 |
| **法修流**（狂轰滥炸） | 悟性 | 神识 | 学习全屏 AOE 仙术，身板脆但清怪极快。真元消耗高，需管理资源 |
| **剑修 / 御器流**（飞剑千里） | 神识 | 悟性 | 操控多把飞剑/法宝远程输出，自己不近身。依赖神识操控上限 |
| **召唤 / 阵法流**（安逸挂机） | 神识 | 机缘 | 靠阵法困敌，灵宠输出。培养成本高但操作安逸 |

### 4.2 流派与真元消耗

| 流派 | 真元消耗特征 |
|------|-------------|
| 锻体流 | 技能消耗真元极少，可持续输出，但需近身 |
| 法修流 | 技能消耗真元高，爆发强，需管理资源循环 |
| 剑修 / 御器流 | 中等消耗，平稳输出 |
| 召唤 / 阵法流 | 阵法一次性大消耗，灵宠持续输出不耗真元 |

## 五、真元系统（法力资源）

### 5.1 真元上限

真元上限由 **神识** 和 **悟性** 共同决定（双属性加权）。

### 5.2 真元消耗

所有流派的技能均消耗真元。锻体流消耗极小，法修流消耗大。

### 5.3 真元回复

| 回复方式 | 说明 |
|----------|------|
| **自然回复** | 战斗中自动回复，回复速率由神识决定（每秒 N 点） |
| **打坐** | 非战斗状态主动打坐，回复速率大幅提升 |
| **丹药** | 战斗中可使用回蓝丹药（有冷却） |
| **灵兽技能** | 特定灵兽可提供真元回复光环或主动回蓝技能 |
| **特殊技能** | 类似 WOW 法师"唤醒"的爆发回蓝技能，战斗中可用但有长冷却（法修专属） |
