# 第2层：行动原子规则书

> 版本：v1.0 | 最后更新：2026-07-15

---

## 概述

第2层定义**玩家可以做什么**——每一个行动原子的输入、输出、消耗与判定公式。所有行动原子共享统一的资源模型，不同流派通过参数差异实现差异化体验。

### 设计原则

1. **资源模型统一**：同一类型的行动共享同一套消耗/产出公式，降低后续战斗系统的实现复杂度。
2. **流派可差异化**：四大流派（锻体/法修/剑修/召唤）在执行同一原子时，走不同的参数路径，不新增特殊原子。
3. **五行克制穿透**：所有伤害类原子统一走 `基础伤害 × 灵根纯度加成 × 五行克制系数 × 暴击判定` 流水线。
4. **数值留白**：本章只定义框架和系数方向，精确数值留到第3层战斗系统中敲定。

---

## 一、行动原子分类

行动原子按资源模型分为四类：

| 类型 | 核心消耗 | 核心产出 | 受控属性 | 典型场景 |
|------|----------|----------|----------|----------|
| **瞬发行动** (Instant) | 真元（MP） | 即时伤害/效果 | 悟性 + 神识 | 攻击、技能、道具 |
| **持续行动** (Channeling) | 时间 + 真元 | 累积进度条 | 根骨 + 身法 | 采集、打坐、蓄力 |
| **触发行动** (Reactive) | 条件满足 | 被动效果 | 机缘 + 悟性 | 闪避、反击、护体 |
| **仪式行动** (Ritual) | 材料 + 时间 + 真元 | 产物 / 状态变更 | 悟性 + 机缘 | 炼丹、炼器、突破 |

### 持续时间约定

| 标签 | 含义 | 说明 |
|------|------|------|
| 0T | 即发即完 | 同一帧内结算，不占用行动轮 |
| 1T | 1 个行动轮 | 占用当前行动轮，下轮开始时返还控制权 |
| nT | n 个行动轮 | 持续占用行动轮，可被打断 |

> 具体行动轮时长（回合制）或帧数（实时制）在第3层战斗系统中确定。

## 二、双资源模型

### 2.1 真元（MP）—— 法力池

| 属性 | 说明 |
|------|------|
| **上限** | `真元上限 = 悟性 × α + 神识 × β`（α、β 为系数，精确值待定） |
| **自然恢复** | 每秒 / 每轮恢复 `真元上限 × r%`，打坐时恢复速率翻倍 |
| **消耗来源** | 瞬发行动（攻击/技能/格挡）、持续行动（蓄力/采集） |
| **归零后果** | 无法释放任何消耗真元的行动 |

### 2.2 神识（Divine Sense）—— 战术池

| 属性 | 说明 |
|------|------|
| **上限** | `神识上限 = 神识属性 × γ` |
| **恢复方式** | 仅通过丹药、特定功法或打坐恢复，**不自动恢复** |
| **消耗来源** | 神识锁定、探查、灵宠指令、合击 |
| **归零后果** | 无法执行神识行动；命中率降低 |

### 2.3 设计意图

真元管「量」——持续输出能力强不强；神识管「质」——能不能打出关键战术操作。两者独立恢复形成战术节奏：短期爆发需要神识储备，长期续航看真元管理。

## 三、战斗侧行动原子（12项）

### 3.1 普通攻击

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（1T） |
| **消耗** | 少量真元 |
| **公式** | `伤害 = 流派基础伤害 × 灵根纯度加成 × 五行克制系数` |

**四大流派差异**：

| 流派 | 基础伤害来源 | 特殊效果 |
|------|-------------|----------|
| 锻体 | 根骨 | 攻击不消耗真元，改为消耗微量气血（自残换伤） |
| 法修 | 悟性 | 伤害最高，真元消耗最大，射程最远 |
| 剑修 | 悟性 + 根骨（各 50%） | 自带 5% 破甲（无视目标防御百分比） |
| 召唤 | 神识 | 伤害由灵宠代为执行，自身不参战 |

### 3.2 技能释放

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（1T） |
| **消耗** | `技能基础真元消耗 × 技能等级系数` |
| **冷却** | 每个技能独立冷却（GCD + 技能自身 CD） |
| **公式** | `伤害/效果 = 技能基础值 × (1 + 灵根纯度加成%) × (1 + 五行克制系数) × 暴击判定` |

**五行克制嵌入**：所有五行属性技能的克制系数在第1层 §1.4 已定义（克制方 +25%，被克方 -25%）。异灵根技能走异灵根独立克制循环（暗→雷→冰→风→暗，±20%）。

**技能学习前提**：
- 技能所属五行属性纯度 ≥ 技能要求（如《三味真火》要求火灵根 ≥ 30）
- 天灵根不可学习被克属性的技能
- 杂灵根可学所有属性但学习速度衰减

### 3.3 防御 / 格挡

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T，响应动作） |
| **消耗** | 少量真元 |
| **公式** | `减伤比例 = min(根骨 / (根骨 + 攻击方攻击力 × K), 上限 80%)` |
| **限制** | 每轮只能响应一次；成功格挡后重置普攻冷却 |

### 3.4 闪避

| 字段 | 值 |
|------|-----|
| **类型** | 触发（被动） + 可主动消耗真元强化（瞬发） |
| **被动公式** | `闪避率 = 身法 / (身法 + 攻击方神识 × M)` |
| **主动强化** | 消耗真元，临时翻倍身法值（持续 1T），有冷却 |
| **上限** | 闪避率硬上限 50% |

### 3.5 使用道具

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 道具本身被消耗，无额外真元消耗 |
| **限制** | 公共冷却 1T；战斗中可用道具受种类限制（丹药/符箓/一次性法宝） |
| **丹药耐药性** | 同类丹药效果递减公式见 §3.18 |

### 3.6 蓄力 / 吟唱

| 字段 | 值 |
|------|-----|
| **类型** | 持续（nT） |
| **消耗** | 每轮消耗 `真元上限 × c%` |
| **效果** | 完成后伤害/效果 × `(1 + n × 提升系数)` |
| **打断** | 受到超过气血上限 15% 的伤害时打断，消耗的真元不返还 |

### 3.7 切换法宝

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 无真元消耗 |
| **冷却** | 3T |
| **效果** | 切换当前装备的法宝，获得对应法宝的被动效果和可用技能组 |
| **限制** | 神识属性决定同时可装备的法宝数量上限 |

### 3.8 逃跑 / 脱离

| 字段 | 值 |
|------|-----|
| **类型** | 持续（2T） |
| **消耗** | 每轮消耗真元 |
| **判定** | `成功率 = 身法 / (身法 + 敌方平均身法)` |
| **失败惩罚** | 下轮受到追击伤害（1.5 倍普攻），且闪避率 -20%（1T） |
| **PVP 限制** | PVP 中成功率减半 |

### 3.9 神识锁定

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 神识值 |
| **效果** | 目标闪避率 -X%（X 由神识差值决定），持续 3T |
| **免疫** | 神识远高于锁定者可免疫（阈值：目标神识 > 锁定者 × 3） |

### 3.10 灵气护体

| 字段 | 值 |
|------|-----|
| **类型** | 触发（被动） + 可主动强化 |
| **被动效果** | `常驻减伤 = 根骨 × D%` |
| **主动强化** | 消耗大量真元，减伤翻倍持续 3T，冷却 10T |
| **流派差异** | 土灵根专注的锻体修士护体效果 ×1.5 |

### 3.11 灵宠指令

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 神识值 |
| **效果** | 对灵宠下达一条指令（攻击/技能/召回/自爆等），灵宠独立执行 |
| **携带数量** | 神识属性决定同时可携带灵宠数量上限 |
| **自主行动** | 无指令时灵宠按 AI 自主行动（攻击最近目标或保护主人） |

### 3.12 合击 / 连携

| 字段 | 值 |
|------|-----|
| **类型** | 触发 |
| **消耗** | 参与双方各消耗神识值 |
| **触发条件** | 双方在同一行动轮内对同一目标释放同属性技能 |
| **效果** | `合击伤害 = (A 技能伤害 + B 技能伤害) × 合击系数(1.3~1.5)` |
| **冷却** | 每对组合冷却 5T |

## 四、非战斗侧行动原子（10项）

### 4.1 采集

| 字段 | 值 |
|------|-----|
| **类型** | 持续（nT） |
| **消耗** | 真元（少量持续） |
| **耗时公式** | `采集时间 = 基础时长 / (身法系数 × 工具加成)` |
| **产出判定** | 基础产出 100%；机缘触发「稀有产出」概率 = `机缘 / (机缘 + E)` |

### 4.2 炼丹

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **消耗** | 材料 + 丹方 + 真元 + 时间 |
| **成功率** | `基础成功率 + 悟性 × F%`，上限 90% |
| **品质判定** | 成功后多段判定：普通（默认）→ 精良（机缘触发）→ 完美（机缘二次触发，极低概率） |
| **流派加成** | 木灵根修士炼丹成功率 +10%，品质判定额外 +1 次 |

### 4.3 炼器

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **消耗** | 材料 + 图纸 + 真元 + 时间 |
| **成功率** | `基础成功率 + (悟性 + 根骨) / 2 × G%` |
| **品质判定** | 同炼丹的多段判定模型 |
| **流派加成** | 金灵根修士炼器成功率 +10%，锻造武器有概率附加「锋锐」属性 |

### 4.4 打坐 / 修炼

| 字段 | 值 |
|------|-----|
| **类型** | 持续 |
| **消耗** | 时间 |
| **产出** | 经验值 / 法力上限提升 |
| **效率公式** | `修炼效率 = 灵根纯度系数 × 灵地灵气浓度 × 功法加成 × (1 + 悟性 × H%)` |
| **灵气浓度** | 灵脉等级决定（下品 0.5x / 中品 1.0x / 上品 2.0x / 极品 5.0x） |
| **干扰** | 受攻击打断，灵气稀薄区域效率衰减 |

### 4.5 交易

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发 |
| **消耗** | 无 |
| **议价** | `成交价 = 基准价 × (1 + (买卖双方的机缘差值造成的偏移))` |
| **上限** | 议价幅度 ±20% |

### 4.6 传送

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（短距离）/ 仪式（跨区域/跨界） |
| **消耗** | 短距离：真元；跨区域：灵石；跨界：特殊道具（传送符 / 跨界符） |
| **耗时** | 短距离即时，跨区域读条 nT，跨界需完整仪式 |
| **限制** | 跨界传送有境界门槛 |

### 4.7 神识探查

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 神识值 |
| **范围** | `探查半径 = 神识属性 × R 米` |
| **效果** | 发现范围内隐藏目标、陷阱、秘境入口；敌方信息（境界 / 血量百分比 / 灵根属性） |
| **反制** | 目标神识高于探查者 ×3 时可完全隐藏信息 |

### 4.8 布阵

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **消耗** | 阵旗 / 阵盘 + 材料 + 真元 + 时间 |
| **复杂度上限** | 悟性决定可布置的阵法复杂度等级 |
| **效果** | 阵法范围内提供 buff / debuff / 伤害 / 传送等效果，持续直到被破除 |
| **破除** | 敌方可用神识探查发现阵眼，攻击阵眼破坏 |

### 4.9 服药

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发（0T） |
| **消耗** | 丹药本身 |
| **耐药性公式** | `Δx = x × (1 - x/X)^(n-1)` |
|  | `x` = 丹药基础法力增量，`X` = 当前境界最大基础法力值，`n` = 已服用次数 |
| **现实含义** | 同种丹药效果递减，迫使玩家追求更高级丹药或切换种类 |

### 4.10 突破境界

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **消耗** | 无（但失败有惩罚） |
| **前置条件** | 当前境界法力/修炼值达到圆满 |
| **成功率** | `基础成功率 + 五维加权判定 + 辅助道具加成` |
| **失败惩罚** | 损失 `(1 - 毅力减免%) × 当前境界总修为的 P%` |
| **毅力** | 隐藏属性，不可加点；通过特定剧情/师承任务获得 |

## 五、社交侧行动原子（7项）

### 5.1 组队

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发 |
| **消耗** | 无 |
| **人数上限** | `基础 4 人 + 神识属性加成（每 N 点神识 +1 名额）` |
| **经验分配** | 均分 × 组队加成（2 人 ×1.1，3 人 ×1.15，4 人 ×1.2，5 人以上 ×1.25） |

### 5.2 拜师

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **前置条件** | 师父境界 > 徒弟至少 2 个大境界；好感度达标 |
| **收益** | 解锁师承技能树；修炼该师父传承功法时获得效率加成 |
| **限制** | 每名玩家终身最多拜 3 位师父 |

### 5.3 PK / 切磋

| 字段 | 值 |
|------|-----|
| **类型** | - |
| **消耗** | 复用战斗原子 |
| **模式** | 切磋（不致死，气血归零判负）/ 生死斗（致死，死亡惩罚适用） |
| **先手规则** | 发起方占据先手，但切磋模式双方均有 1T 准备时间 |

### 5.4 宗门贡献

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发 |
| **消耗** | 上交材料 / 灵石 / 丹药 |
| **产出** | 宗门贡献点（用于兑换宗门功法 / 丹药 / 权限） |
| **阶梯** | 贡献越高宗门地位越高，解锁更多兑换项 |

### 5.5 双修 / 论道

| 字段 | 值 |
|------|-----|
| **类型** | 持续 |
| **消耗** | 双方时间 + 少量真元 |
| **产出** | 双方获得「修炼加速」buff（持续 n 小时） |
| **效率** | `加速比例 = 双方悟性之和 × S%` |
| **限制** | 同一对象冷却 24 小时（游戏时间） |

### 5.6 传功

| 字段 | 值 |
|------|-----|
| **类型** | 仪式 |
| **消耗** | 传功者永久损失一定修为值 |
| **产出** | 受功者直接获得修为；若传授技能，受功者免学习条件直接习得 |
| **限制** | 仅限师父→徒弟；每次传功后有长冷却 |

### 5.7 悬赏 / 仇杀

| 字段 | 值 |
|------|-----|
| **类型** | 瞬发 |
| **消耗** | 灵石（发布悬赏） |
| **死亡惩罚** | 生死斗中死亡 → 掉落背包中的随机物品（装备绑定除外）+ 损失修为 |
| **PK 值** | 主动击杀非红名玩家累计 PK 值，PK 值高者被击杀时惩罚加重 |

## 六、通用数值流水线

所有产出/伤害/效果类行动原子，统一走以下流水线：

```
输入属性
  → 基础值计算（查表/公式）
  → 灵根纯度加成（第1层 §1.3）
  → 五行克制系数（第1层 §1.4，±25%）
  → 流派差异化修正（§3.1 四大流派差异）
  → 暴击判定（机缘 → 会心一击）
  → 目标抗性/防御减伤
  → 最终结果
```

这一步流水线确保第3层战斗系统设计时，所有原子有统一的数值入口和出口。
