# 第5层：经济系统规则书

> 版本：v1.0 | 最后更新：2026-07-15

---

## 概述

第5层定义**什么东西从哪产出、在哪流通、在哪消耗**。基于第1~4层（属性→行动→战斗→成长）已定义的产出入口和消耗出口，构建完整的修仙世界经济模型。

**设计原则**：
1. **玩家驱动**：除体验提升类、体力类、外观类道具外，所有道具必须由游戏内产出（玩家采集/打怪掉落/生活技能制作），系统不直接刷钱。
2. **自由流通**：全部道具可自由交易，走世界交易行；RMB 道具仅限交易行流通，系统不出售。
3. **时间沉淀**：修为增长以真实时间为底层计量，拒绝付费直升。氪金只能加速/便利，不能跳过。
4. **多源平衡**：修为产出来自自然增长、主动获取、离线补偿三路分流，单路不可垄断。
5. **防通胀闭环**：每个产出入口必须有对应的消耗出口，形成"产出→流通→消耗→回收"的完整经济循环。

---

## 一、修为产出体系（多源分流）

修为值（CV）是游戏的**底层硬通货**——所有成长系统（L4）以修为值为核心驱动力。修为产出分散在多个来源，确保没有单一来源可垄断。

### 1.1 自然增长（挂机收益）

```
每秒修为获取 = 基础速率 × 灵气浓度系数 × 灵根效率系数 × 功法加成 × 在线状态系数
```

| 系数 | 影响因素 | 范围 |
|------|----------|------|
| 基础速率 | 当前境界（炼气 0.1/s，筑基 0.5/s，结丹 2/s……） | 境界越高越快 |
| 灵气浓度 | 地图灵气等级（见 §1.2） | 0.1x ~ 5.0x |
| 灵根效率 | 天灵根 ×2.0 / 真灵根 ×1.0 / 伪灵根 ×0.5 | 0.5x ~ 2.0x |
| 功法加成 | 主修功法提供的修炼速度加成 | 1.0x ~ 3.0x |
| 在线状态 | 在线 ×1.0 / 离线 ×0.1 / 离线（洞府） ×0.15 / 离线（门派） ×0.12 | 0.1x ~ 1.0x |

#### 离线补偿说明

| 状态 | 补偿倍率 | 说明 |
|------|---------|------|
| 普通离线 | ×0.1 | 在任意地点下线，保持最低限度的自然增长 |
| 洞府离线 | ×0.15 | 拥有个人洞府并在此下线，灵气浓度略高于野地 |
| 门派/宗门离线 | ×0.12 | 加入门派后在门派范围内下线 |
| 商会特殊区域 | ×0.18 | 商会区域离线摆摊时（见 §6），额外+0.03 |

> **离线收益上限**：离线收益累计上限 = 24 小时 × 当前在线速率。超出后停止增长，需上线领取已累计收益。

### 1.2 地图灵气浓度分级

| 灵气等级 | 浓度系数 | 代表区域 | 说明 |
|---------|---------|---------|------|
| 稀薄 | 0.1x | 凡人城镇外围、荒野险地 | 不适合修炼，仅路过 |
| 普通 | 0.5x | 凡人城镇、普通野外 | 散修玩家的主要活动区 |
| 中等 | 1.0x | 门派外围、灵脉支脉 | 大多数玩家的日常活动区 |
| 浓郁 | 2.0x | 门派核心、灵脉主脉 | 有门派归属的玩家专属区域 |
| 极品 | 5.0x | 秘境洞天、宗门圣地 | 限时/限次进入，高阶玩家的争夺点 |

> 灵气浓度不仅影响修为速度，还影响炼丹成功率、法宝温养效率等（详见 L4 §6）。

### 1.3 主动修为获取

| 来源 | 修为产出 | 占总修为获取比例（估算） | 与 L3/L4 衔接 |
|------|---------|----------------------|--------------|
| **打怪掉落** | 击杀怪物获得修为（基于怪物境界 × 难度系数） | 30%~40% | L3 §2：野外遇敌、副本清图 |
| **副本通关奖励** | 通关副本后一次性奖励大量修为 | 15%~20% | L3 §1：秘境、Boss 战 |
| **任务奖励** | 宗门/城镇/奇遇任务奖励修为 | 15%~20% | L2 §5.4：宗门贡献 |
| **丹药服用** | 修为丹直接增加修为数值 | 5%~15% | L4 §6.1：炼丹产出 |
| **秘境探索** | 秘境中特殊节点奖励修为 | 5%~10% | L3 §3.4：破阵/幻境 |
| **奇遇事件** | 随机触发的修为增减事件 | <5% | L3 §4：奇遇事件战 |

> **前期权重**（炼气~筑基）：任务 + 打怪占主导（~60%），丹药和秘境为辅。
> **后期权重**（结丹+）：自然增长 + 副本 + 丹药占比上升，任务降至配角。

### 1.4 负面行为惩罚

| 行为 | 修为影响 | 说明 |
|------|---------|------|
| **恶意 PVP（击杀非红名玩家）** | 立即损失 `当前境界总修为 × 0.5%` + PK 值累积（详见 L3 §5.2） | 每次击杀非红名玩家触发 |
| **偷窃/盗窃（玩家）** | 立即损失 `当前境界总修为 × 0.2%` | 偷窃失败或被通缉后触发 |
| **修行走火入魔** | 每小时损失 `自然增长量 × 50%` | 突破失败后 debuff 期间持续扣减 |
| **邪派行为（屠村/破坏）** | 名声系统降低，间接影响修为效率 | 邪派声望导致 NPC 商店加价、任务不可接 |
| **心魔缠身** | 修炼效率 -30%，持续 24 小时 | L4 §1.5 突破失败的附加惩罚 |

> 负面行为造成的修为损失**不回收进经济系统**（直接抹除），防止刷修为。

### 1.5 特殊事件/奇遇

| 事件类型 | 修为影响 | 触发条件 |
|---------|---------|----------|
| **天降福缘** | 一次性获得大量修为（当前境界 5%~20%） | 极低概率，大地图随机触发 |
| **秘境灵气潮汐** | 临时提升地图灵气浓度（×3，持续 4 小时） | 定期世界事件 |
| **魔道入侵** | 灵气浓度全域下降（×0.5），击杀入侵魔物获额外修为 | 不定期世界事件 |
| **论道/讲法** | 听高阶 NPC 讲法，被动获得修为增长（持续 1 小时） | 宗门定期活动 |
| **天道感应** | 修为获取速率 ×2（持续 30 分钟） | 极小概率，突破成功后概率触发 |

## 二、体力/资源池系统

### 2.1 体力值定义

体力值是限制玩家高频收益行为的资源池，类比传统手游的"体力/活力"。

| 属性 | 说明 |
|------|------|
| **名称** | 灵力（Spiritual Power, SP） |
| **上限** | `基础 120 点 + (根骨 × 0.5)`，上限 200 点 |
| **恢复速率** | 每小时恢复 10 点（在线 + 离线均恢复） |
| **满恢复时间** | 从 0 到满约 12~20 小时 |
| **恢复加速** | 特定丹药/洞府加成可提升至每小时 15 点 |

### 2.2 灵力消耗场景

| 玩法 | 消耗灵力 | 说明 |
|------|---------|------|
| **普通副本** | 10 点/次 | 普通难度，掉落基础材料和修为 |
| **精英副本** | 20 点/次 | 高难度，掉落装备和稀有材料 |
| **秘境探索** | 30 点/次 | 限时秘境，掉落功法残卷和特殊道具 |
| **世界 Boss** | 15 点/次 | 参与即有奖励，最后一次伤害加成 |
| **采集（高级资源点）** | 5 点/次 | 稀有矿石/灵草采集点 |
| **炼丹/炼器** | 不消耗灵力 | 生活技能走独立熟练度体系 |

### 2.3 战令/月卡双倍特权

| 特权类型 | 效果 | 说明 |
|---------|------|------|
| **基础月卡**（RMB） | 灵力恢复速度 +50%（每小时 15 点） | 月付，价值约 30 元 |
| **高级战令**（RMB） | 每日额外领取 60 点灵力（一次性） | 赛季制，约 68 元 |
| **双倍券** | 指定玩法消耗灵力不变但产出翻倍 | 可由战令赠送，也可交易行流通 |
| **灵力药剂**（可交易） | 立即恢复 30 点灵力，冷却 2 小时 | 炼丹产出，可在交易行自由买卖 |

> **关键设计**：灵力药剂由炼丹产出（玩家制作），可在交易行自由交易。这意味着零氪玩家可以通过炼丹→出售灵力药剂→赚灵石→购买其他玩家出售的 RMB 道具，形成搬砖闭环。

### 2.4 资源池设计原则

1. **不卡死**：灵力只影响额外收益玩法（副本、秘境），不影响基础修为自然增长和在线挂机。
2. **可溢出**：灵力满后不浪费——溢出部分按 20% 效率转化为微量修为（聊胜于无）。
3. **可交易**：灵力药剂可自由交易，避免"不氪金就玩不了"的体验。
4. **上限保护**：灵力上限有天花板（200 点），防囤积。

### 2.5 灵力系统与其他层的联动

| 联动方向 | 说明 |
|----------|------|
| **与 L3 战斗系统** | 副本、秘境、世界 Boss 副本由 L3 §4 战斗系统承载，灵力作为入场门票 |
| **与 L4 成长系统** | 灵力药剂由 L4 §6.1 炼丹产出，技能栏消耗灵力进入副本获取技能书 |
| **与经济闭环** | 灵力→副本收益→交易行→灵石→灵力药剂，形成"肝→产出→交易→恢复→再肝"闭环 |
| **月度战令** | 战令灵力包是月卡/战令的核心付费价值锚点，设计上不破坏搬砖生态 |

## 三、货币体系

### 3.1 货币层级总览

| 货币 | 获取方式 | 主要用途 | 是否可交易 | 是否绑定 |
|------|---------|---------|-----------|---------|
| **灵石** | 打怪掉落/任务奖励/玩家交易/副本产出 | 核心交易货币，所有玩家间流通 | ✅ 自由交易 | ❌ 非绑定 |
| **灵石碎片** | 同上（小额） | 找零/小額交易/ NPC 小额支付 | ✅ 自由交易 | ❌ 非绑定 |
| **绑定灵石** | NPC 任务/绑定道具出售 | NPC 商店购买基础补给 | ❌ 不可交易 | ✅ 角色绑定 |
| **宗门贡献** | 宗门任务/上交材料/宗门战 | 宗门商店兑换功法/丹药/装备 | ❌ 不可交易 | ✅ 宗门绑定 |
| **功勋** | PVP 玩法/宗门战/世界 Boss | 功勋商店兑换 PVP 装备/特殊道具 | ❌ 不可交易 | ✅ 角色绑定 |

### 3.2 灵石经济

灵石是本游戏的**唯一自由交易货币**。一切自由市场的流通以灵石为计价单位。

#### 灵石面额

| 面额 | 换算 | 说明 |
|------|------|------|
| 灵石碎片 | 1000 碎片 = 1 灵石 | 小面额，用于找零和小額交易 |
| 灵石 | 1 灵石 = 1000 碎片 | 标准交易单位 |
| 上品灵石 | 1 上品 = 1000 灵石 | 大额交易使用，减少背包压力 |
| 极品灵石 | 1 极品 = 1000 上品 | 超大型交易/高级 NPC 交易使用 |

#### 灵石产出

| 来源 | 产出量占比（估算） | 说明 |
|------|-----------------|------|
| 怪物掉落 | 40% | 怪物掉落灵石碎片，精英/Boss 掉落整块灵石 |
| 副本奖励 | 20% | 通关奖励 + 额外掉落 |
| 任务奖励 | 15% | 宗门/城镇/奇遇任务 |
| 玩家交易手续费回收 | 10% | 系统回收的灵石（税收）再分配 |
| 生活技能出售给 NPC | 10% | 炼丹/炼器产物卖店 |
| 采集 | 5% | 灵脉矿石可直接卖店 |

#### 灵石消耗锚点

| 消耗项 | 消耗占比（估算） | 说明 |
|------|----------------|------|
| **装备强化** | 30% | 每次强化消耗灵石，高等级消耗呈指数增长 |
| **装备升阶** | 15% | 升阶消耗大量灵石 + 材料 |
| **交易行手续费** | 15% | 系统回收（见 §5.3 税收设计） |
| **传送费用** | 10% | 跨区域/跨界传送消耗灵石 |
| **NPC 商店购买** | 10% | 基础补给、药水等 |
| **炼丹/炼器材料** | 10% | 拍卖行购买材料 |
| **宗门捐献** | 5% | 上交灵石换宗门贡献 |
| **特殊功能** | 5% | 改名、重置属性等 |

> **产出-消耗平衡**：上述消耗锚点确保灵石总量不会无限膨胀。系统通过税收（交易行手续费）和强化消耗持续回收灵石。

### 3.3 绑定灵石

绑定灵石是**系统控制的基础物价锚点**，防止通货膨胀影响零氪玩家基本生存。

| 属性 | 说明 |
|------|------|
| **获取** | NPC 任务奖励、出售绑定道具给 NPC、签到奖励 |
| **用途** | 仅在 NPC 商店购买基础补给（基础丹药、基础材料、修理装备） |
| **不可交易** | 无法在其他玩家之间流通 |
| **比例** | 绑定灵石的 NPC 物价固定，不受市场波动影响 |

> **设计意图**：确保零氪玩家即使不参与自由市场交易，也能通过任务和离线挂机获得足够的绑定灵石维持基本游戏体验。

## 四、道具产出-流通-消耗生命周期

### 4.1 产出源头全景

| 产出源头 | 产出物 | 占市场供给比例（估算） |
|---------|-------|---------------------|
| **怪物掉落** | 灵石碎片、装备（凡器~灵器）、技能书、材料、灵宠蛋 | 30% |
| **采集（矿/草/木）** | 矿石、灵草、木材——炼器/炼丹的基础材料 | 20% |
| **炼丹** | 各类丹药（修为/回血/回蓝/突破/属性/灵宠） | 20% |
| **炼器** | 装备（法器~宝器）、法宝 | 15% |
| **副本** | 装备、灵石、功法残卷、特殊材料 | 10% |
| **任务** | 绑定灵石、装备、丹药（绑定） | 5% |

### 4.2 流通渠道

| 渠道 | 说明 | 手续费 | 适合场景 |
|------|------|-------|---------|
| **世界交易行** | 全服统一市场，玩家挂单/购买 | 成交价的 5% | 标准化道具（材料/丹药/装备） |
| **商会区域摆摊** | 加入商会的玩家可在商会区域离线摆摊 | 商会等级决定（最低 2%） | 个性化装备/批量达 |
| **玩家直接交易** | 面对面交易，无需上架 | 无手续费 | 熟人/朋友间小额交易 |
| **拍卖行** | 稀有道具竞拍（Boss 首杀/仙器/特殊道具） | 成交价 8% | 稀有/唯一性道具 |

### 4.3 消耗出口全景

| 消耗类型 | 消耗路径 | 对经济的影响 |
|---------|---------|------------|
| **装备强化** | 灵石 + 强化石 → 属性提升（强化石由炼器产出/怪物掉落） | 回收灵石 + 稳定材料需求 |
| **装备升阶** | 灵石 + 升阶石 + 装备碎片 → 品阶提升 | 回收灵石 + 消化低阶装备 |
| **装备修理** | 灵石 → 耐久恢复 | 持续性小额灵石回收 |
| **丹药消耗** | 丹药 → 一次效果（战斗/修炼/突破） | 消耗品持续需求，支撑炼丹市场 |
| **传送** | 灵石 → 跨区域移动 | 小额灵石回收，高频 |
| **交易行手续费** | 灵石 → 系统回收 | 经济调控核心手段 |
| **宗门捐献** | 灵石/材料 → 宗门贡献 | 内部流通，不向系统外流失 |
| **玩家间交易** | 灵石 → 道具 | 货币在玩家间流动，不增加系统总量 |

### 4.4 经济循环图示

```
                    ┌──────────────┐
                    │  系统产出入口  │
                    │ (怪/采集/副本) │
                    └──────┬───────┘
                           │
                           ▼
                    ┌──────────────┐
                    │  灵石 + 材料   │  ← 炼丹/炼器加工后价值提升
                    │  (玩家背包)    │
                    └──────┬───────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
```

## 五、交易行系统

### 5.1 交易行定位

世界交易行是本游戏自由经济的**核心流通枢纽**。所有非绑定的道具均可在此上架交易。

| 特性 | 说明 |
|------|------|
| **全服统一** | 所有玩家共享同一个交易行市场，不分线不分服 |
| **全天候** | 24 小时在线，离线玩家也可收取出售收益 |
| **品类全面** | 装备/丹药/材料/技能书/灵宠/灵石/灵力药剂/部分 RMB 道具 |
| **上架数量限制** | 基础 10 个上架位，随境界提升（元婴 +5，化神 +5） |

### 5.2 上架与定价规则

| 规则 | 说明 |
|------|------|
| **推荐价** | 系统根据近期成交均价自动生成推荐价格区间（-20%~+20%） |
| **自由定价** | 玩家可自由定价，超出推荐价区间需额外手续费（见 §5.3） |
| **一口价** | 标准模式，谁先买谁得 |
| **竞价模式** | 稀有道具可开启竞价模式，设定底价和竞拍时长（12/24/48 小时） |
| **手续费扣除** | 上架时立即扣除上架费（少量灵石，防恶意占位）。成交后扣除成交手续费 |
| **流拍处理** | 未售出道具自动下架，上架费不退。可重新上架 |

### 5.3 税收设计

| 税率类型 | 税率 | 说明 |
|---------|------|------|
| **标准手续费** | 成交价的 5% | 所有上架道具统一征收 |
| **溢价税** | 超出推荐价区间部分 × 10% | 超出推荐价 +20% 以上的部分，额外征收 10% 溢价税 |
| **竞价手续费** | 成交价的 8% | 竞价模式更高，因系统撮合成本更高 |
| **商会减免** | 最低降至 2% | 加入商会后根据商会等级减免（见 §6.2） |

> **溢价税设计意图**：防止恶意囤货炒价。如果你想卖天价（如 10 倍市价），系统通过溢价税拿走大部分超额收益，让炒货变得无利可图。

### 5.4 防通胀/防炒作机制

#### 价格保护机制

| 机制 | 说明 |
|------|------|
| **每日价格涨跌幅限制** | 单品价格单日涨幅不超过昨日均价的 30%。防止短期爆炒 |
| **成交量加权均价** | 交易行显示的价格为过去 7 天成交量加权均价（VWAP），非最后一笔成交价 |
| **冷却期** | 同一玩家买入某道具后，24 小时内不可再上架同种道具（防左手倒右手） |

#### 防工作室批量行为

| 策略 | 说明 |
|------|------|
| **产出衰减** | 同一玩家持续刷同一怪物超过 2 小时，掉落率逐级衰减至 20% |
| **交易限制** | 新注册账号 7 天内不可使用交易行（防批量小号） |
| **灵石流动监控** | 异常灵石流动（短时间内大量同 IP 账号向同一账号转移灵石）触发人工审核 |
| **采集点竞争** | 高级采集点采用"抢点"模式，单点同时只能一人采集，工作室无法批量挂机 |
| **修为产出上限** | 每日在线修为自然增长有软上限（超过 12 小时后效率减半），遏制无限制挂机 |

#### 经济调控储备

| 手段 | 说明 |
|------|------|
| **系统回收团** | 当某种基础材料价格跌破成本线时，系统以保底价限量收购（类似央行 QE） |
| **限时活动** | 灵石消耗类活动（强化周/升阶折扣）可短时回收大量灵石 |
| **灵石购买力锚定** | NPC 商店的绑定灵石物价作为锚点，灵石对绑定灵石长期汇率保持稳定 |
| **调控不干预** | 以上仅为应急手段，正常情况下不干预自由市场定价 |

## 六、商会系统

### 6.1 商会定位

商会是玩家组织的经济合作体，提供交易优惠和离线经营能力。不同于宗门的战斗/社交定位，商会专注于经济利益。

| 维度 | 商会 | 宗门 |
|------|------|------|
| **核心定位** | 经济合作体 | 战斗/修炼/社交 |
| **加入条件** | 缴纳会费 | 境界/推荐 |
| **核心功能** | 手续费减免、离线摆摊、商会任务 | 功法传承、修炼加成、宗门战、师承任务 |
| **产出侧重** | 灵石、道具流通效率 | 修为加速、功法获取、PVP 内容 |
| **升级驱动** | 商会资金（捐献+任务） | 宗门贡献（任务+捐献+宗门战） |
| **成员数量** | 上限 50 人 | 无上限 |
| **冲突处理** | 商业竞争（竞价/商路争夺） | 宗门战（大规模 PVP） |
| **重叠** | 可同时加入 | 可同时加入 |

### 6.2 商会等级与特权

| 商会等级 | 升级条件 | 手续费减免 | 离线摆摊位 | 商会技能 |
|---------|---------|-----------|-----------|---------|
| Lv.1 初创商会 | 初始 | 手续费 4% | 2 个 | 无 |
| Lv.2 发展商会 | 10000 商会资金 | 手续费 3.5% | 3 个 | 低级商会技能 |
| Lv.3 兴盛商会 | 50000 商会资金 | 手续费 3% | 4 个 | 中级商会技能 |
| Lv.4 繁荣商会 | 200000 商会资金 | 手续费 2.5% | 5 个 | 高级商会技能 |
| Lv.5 传奇商会 | 500000 商会资金 | 手续费 2% | 6 个 | 专属商会技能 |

> 商会资金由成员捐献灵石/材料、商会任务奖励、商会税收（内部交易部分抽成）构成。

### 6.3 离线摆摊

| 特性 | 说明 |
|------|------|
| **功能** | 在商会专属区域（商会驻地）设置离线摊位，即使玩家不在线也能持续售出道具 |
| **摊位数量** | 商会等级决定（Lv.1 可设 2 个，Lv.5 可设 6 个） |
| **商品上架** | 离线摆摊的商品同时也在世界交易行可见（双通道） |
| **收益收取** | 离线期间售出道具的收益自动存入商会钱庄，上线后领取 |
| **摆摊外观** | 高等级商会可自定义摊位外观（不影响功能） |
| **离线灵气** | 在商会区域离线获得额外灵气浓度加成（×0.18，见 §1.1） |

### 6.4 商会任务

| 任务类型 | 内容 | 奖励 |
|---------|------|------|
| **物资收集** | 上交指定材料/丹药 | 商会资金 + 个人贡献 + 灵石 |
| **护送商队** | 护送 NPC 商队往返 | 商会资金 + 个人贡献 + 随机道具 |
| **商业竞争** | 与其他商会的限时竞价任务 | 大量商会资金 + 专属称号 |
| **通商开拓** | 在特定地图建立临时商路 | 商会等级经验 + 永久通行折扣 |

### 6.5 商会技能树

商会技能消耗个人贡献解锁，提供经济侧被动加成：

| 技能 | 效果 | 最大等级 |

| **议价术** | NPC 商店购买折扣 +2%/级 | 5 级（共 -10%） |

| **货源感知** | 采集稀有材料概率 +2%/级 | 3 级（共 +6%） |

## 七、战令/月卡与自由经济平衡

### 7.1 付费内容边界

| 付费类型 | 是否可交易 | 是否破坏平衡 | 说明 |
|---------|-----------|------------|------|
| **月卡**（灵力恢复加速） | ❌ 不可交易 | 否（只有效率差异） | 加速型付费 |
| **战令**（每日灵力包+赛季奖励） | ❌ 不可交易 | 否（上限可控） | 加速型付费 |
| **灵力药剂** | ✅ 可交易 | 否（丹药可替代） | 玩家制作+交易 |
| **外观/时装** | ✅ 可交易 | 否（无属性） | 完全装饰 |
| **背包扩展** | ❌ 不可交易 | 否（便利功能） | 体验提升类 |
| **保护符**（强化不掉级） | ✅ 可交易 | 否（不增加战力上限） | 容错型道具 |
| **双倍券**（副本产出翻倍） | ✅ 可交易 | 否（不增加次数上限） | 效率型道具 |
| **经验/修为丹** | ❌ 不可交易 | 否（有耐药上限） | 加速型道具 |

> **核心原则**：可交易的 RMB 道具只限于"效率提升"和"外观"两类，不产出任何"属性上限突破"类道具。付费只能让你更快到达上限，不能让你突破上限。

### 7.2 可交易 RMB 道具清单

以下道具为战令/月卡赠送或商城购买后，可挂入交易行出售的道具：

| 道具名 | 效果 | 获取来源 | 交易行定价参考 |
|-------|------|---------|-------------|
| 灵力药剂 ×5 礼包 | 立即恢复 150 灵力 | 月卡每日福利 | 市场价（玩家定价） |
| 双倍修为符（1h） | 修为获取 ×2，持续 1 小时 | 战令奖励 | 市场价 |
| 保护符·凡 | 强化 +1~+5 失败不掉级 | 战令/商城 | 市场价 |
| 保护符·灵 | 强化 +6~+10 失败不掉级 | 高等级战令 | 市场价 |
| 外观兑换券 | 兑换商城任意外观 | 赛季战令最终奖励 | 市场价 |
| 改名符 | 修改角色名 | 商城 | 市场价 |

### 7.3 搬砖党经济模型

```
搬砖循环：零氪/微氪玩家
┌────────────────────────────────────────┐
│  挂机采集/打怪 → 获得基础材料/灵石       │
│     ↓                                   │
│  炼丹/炼器加工 → 产出丹药/装备           │
│     ↓                                   │
│  挂交易行出售 → 获得灵石                 │
│     ↓                                   │
│  用灵石购买 RMB 道具（保护符/灵力药剂等）  │
│     ↓                                   │
│  装备强化/提升战力 → 打更难的副本         │
│     → 获得更好的材料 → 回到第一步         │
└────────────────────────────────────────┘
```

**估算效率**：一个每天在线 4-6 小时的活跃搬砖党，通过炼丹/采集/打怪→交易行出售，月收益约可购买 1-2 张月卡等值的 RMB 道具。

### 7.4 付费不破坏平衡的三道防线

1. **所有付费道具可交易**：零氪玩家可通过搬砖获得几乎全部付费道具（外观除外）。
2. **不产出属性突破道具**：付费只能加速成长，不能提升属性上限。
3. **战令灵力上限可控**：战令赠送的灵力药剂虽可交易，但受丹药冷却和灵力上限限制，无法无限制肝。

## 八、设备耐久与维修经济

### 8.1 装备耐久度

| 属性 | 说明 |
|------|------|
| **耐久值** | 每件装备有独立耐久值（100/100） |
| **耐久损耗** | 每次受到攻击 -1~-3；每次攻击 -1；死亡 -10 |
| **耐久归零** | 装备属性失效（不消失），需修理后恢复 |
| **特殊装备** | 仙器/神器耐久为 200，损耗减半 |

### 8.2 维修体系

| 维修方式 | 消耗 | 效果 |
|---------|------|------|
| **基础维修**（NPC 铁匠） | 绑定灵石 | 恢复全部耐久（价格固定，不与市场挂钩） |
| **精修**（玩家炼器师） | 灵石 + 材料 | 恢复全部耐久 + 额外临时属性加成（+2% 持续 24 小时） |
| **自我修复** | 灵石（更贵） | 野外无 NPC 时使用，费用加倍 |
| **法宝修复** | 灵石 + 同属性材料 | 法宝专属，损耗比普通装备慢 50% |

> **设计意图**：维修经济是灵石回收的小型锚点。绑定灵石维修确保零氪玩家基础生存，精修系统给炼器玩家提供额外服务收入渠道。

### 8.2a 耐久与炼器技能联动

装备耐久系统与 L4 §6.3（炼器）存在以下深度联动：

| 联动维度 | 说明 |
|----------|------|
| **炼器品质影响耐久上限** | 同等部位下，品阶越高基础耐久越高（凡品 100→下品 120→中品 140→上品 160→极品 180→仙器 200） |
| **炼器师维修加成** | 玩家炼器师进行精修时，根据自身炼器等级提供额外加成：药师 +1%、大师 +2%、宗师 +3%、器圣 +5%（与精修自带 +2% 叠加） |
| **炼器制作耐久保护** | 玩家炼器师制作装备时，可消耗额外材料附魔「耐久保护」——磨损速率 -20%（与仙器/神器的固有减半效果乘算） |
| **耐久素材专属** | 炼器师可采集/产出「矿石/稀有金属」类材料，此类材料的品质影响制作成品的耐久上限 |
| **耐久附魔** | 化神以上炼器师可对装备进行耐久附魔（如「不磨损」——耐久损耗 -30%），附魔消耗独立灵石流 |

## 九、防工作室/防通胀综合策略

### 9.1 产出端限制

| 策略 | 说明 |
|------|------|
| **怪物掉落衰减** | 连续刷同一怪物超过 2 小时，掉落率每 30 分钟衰减 10%（至最低 20%） |
| **采集点刷新** | 采集点非定时刷新，最后采集后的 5~15 分钟随机刷新（防脚本定时采集） |
| **修为产出软上限** | 每日自然增长超出 12 小时后效率减半 |
| **副本掉落绑定** | 副本中掉落的未鉴定装备为绑定状态，鉴定后变为可交易（防止脚本无限刷本） |
| **等级压制掉落** | 低境界玩家刷高境界怪物掉落惩罚；高境界玩家刷低境界怪物无掉落 |

### 9.2 流通端限制

| 策略 | 说明 |
|------|------|
| **交易行门槛** | 新注册账号 7 天 / 当前境界 ≥ 炼气 5 层 才可使用交易行 |
| **同 IP 交易限制** | 同 IP 下多个账号之间交易触发风控（72 小时冻结） |
| **大额交易审核** | 单笔 > 1000 灵石的交易触发系统审核（自动/人工） |
| **批量上架限制** | 同种道具单次上架最多 20 个，超量需分多次（增加脚本成本） |

### 9.3 行为端监测

| 监测项 | 判定标准 | 处罚措施 |
|-------|---------|---------|
| **同 IP 多开** | 同 IP > 3 个活跃账号 | 第 1 次警告，第 2 次限制交易 7 天 |
| **非正常移动模式** | 定点重复路径持续 4 小时以上 | 强制验证（图形验证码） |
| **异常交易** | 长期单向资金转移（A→B，A 不购买任何道具） | 冻结资金，人工核查 |
| **批量注册** | 同设备/同 IP 短时间内注册多个账号 | 封禁新注册账号 |

### 9.4 经济健康度指标

运营团队需持续监控以下指标，作为经济调控依据：

| 指标 | 健康范围 | 预警阈值 | 触发措施 |
|------|---------|---------|---------|
| **灵石总量增长率** | 月增长 < 10% | > 20% | 启动限时灵石消耗活动 |
| **基础材料价格指数** | 月波动 < ±20% | > ±40% | 启动系统回收/释放 |
| **日活玩家灵石持有量分布** | 基尼系数 0.4~0.6 | > 0.7 | 评估经济两极分化程度 |
| **交易行日均成交量** | 日均 > 10 万笔 | < 5 万笔 | 检查是否有系统故障或流动性危机 |
| **搬砖党 vs 付费玩家比例** | 3:1 ~ 5:1 | > 10:1 | 检查工作室渗透率 |

## 十、与 L4 成长系统的数值衔接

### 10.1 修为产出与境界需求

| 境界 | 单小层修为需求 | 日均自然增长（离线挂机） | 日均主动获取（活跃玩家） | 修炼至圆满天数 |

| 炼气 1~13 层 | 100~40 万（累计约 80 万） | 200~800 | 2000~8000 | 数小时 |
| 筑基初期 | 100 万 | 1,000 | 10,000 | ~3 天 |
| 筑基后期 | 200 万 | 1,500 | 15,000 | ~7 天 |
| 结丹初期 | 1,000 万 | 3,000 | 30,000 | ~15 天 |
| 结丹后期 | 2,000 万 | 5,000 | 50,000 | ~1 个月 |
