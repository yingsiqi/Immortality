## §1 四大流派定义

### 1.1 锻体（根骨核心）—— 坦克 / 战士

| 项目 | 说明 |
|------|------|
| 核心属性 | 根骨（主）、身法（副）、神识（辅） |
| 定位 | 高血量、高物理防御、近战前排、团队守护 |
| 战斗风格 | 肉身硬抗、近身爆发、控制技 |
| 加点倾向 | 根骨 ≥ 60，身法适应团队节奏 |
| 强度窗口 | 炼气~元婴前排统治力最强，大乘期后受百分比穿透克制 |

前排基石，根骨提供气血与物理防御的双重保障。身法保证不会被远程职业风筝，神识提供基础命中与法宝防御手段。

### 1.2 法修（悟性核心）—— 法师 / 炮台

| 项目 | 说明 |
|------|------|
| 核心属性 | 悟性（主）、神识（副）、悟性继续堆 |
| 定位 | 超高术法威能、中远距离输出、AOE 清场 |
| 战斗风格 | 读条吟唱、范围轰炸、护盾自保 |
| 加点倾向 | 悟性 ≥ 60，神识 ≥ 40 |
| 强度窗口 | 金丹~化神 AOE 清场效率登顶，前期身板脆弱需保护 |

悟性是术法威能的唯一线性来源，每点悟性带来的伤害成长极高。神识保障真元池深度与持续作战能力，避免"一套技能空蓝"的窘境。

### 1.3 剑修（均衡偏身法）—— 刺客 / 战士

| 项目 | 说明 |
|------|------|
| 核心属性 | 身法（主）、根骨（副）、悟性（副） |
| 定位 | 高频率出手、中等伤害、单体点杀 + 小型 AOE |
| 战斗风格 | 身法压制、连招爆发、拔剑术 |
| 加点倾向 | 身法 ≥ 50，悟性 ≥ 40 |
| 强度窗口 | 全周期平滑，元婴~化神单体点杀峰值，大乘期群战略乏力 |

身法决定出手频率与先手权，悟性提供剑诀技的附加伤害系数，根骨保障近战生存。剑修无显著弱势期，也无绝对统治期——均衡的代价。

### 1.4 召唤（神识核心）—— 控场 / 辅助

| 项目 | 说明 |
|------|------|
| 核心属性 | 神识（主）、悟性（副）、机缘（辅） |
| 定位 | 多单位操控、宠物大军、控场辅助、PVE 效率极高 |
| 战斗风格 | 法宝遥控、灵兽群战、异常状态流 |
| 加点倾向 | 神识 ≥ 60，悟性理解功法需求 |
| 强度窗口 | 元婴~炼虚 PVE 效率碾压其他流派，PVP 依赖操作与宠物质量 |

神识决定法宝操控数量与灵兽携带上限。悟性保障御物诀/驭兽诀的学习速度与技能等级。机缘影响宠物领悟稀有技能的触发概率。

## §3 真元系统

### 3.1 真元上限公式

真元上限公式由 `attribute-system.md §5.2` 统一定义，build 系统直接引用：

```
真元上限 = (神识 + 悟性 × 0.6) × 15 × 境界系数
```

其中境界系数 = 2^(境界等级-1)，详见下表。

### 3.2 境界真元系数

| 境界 | 境界系数 | 炼气=1x 基准值 | 法修典型值（悟性65/神识60） |
|------|:--------:|:-------------:|:-------------------------:|
| 炼气 | 1.0 | 510 | (60+65×0.6)×15 = 1,485 |
| 筑基 | 2.0 | 1,020 | 2,970 |
| 金丹 | 4.0 | 2,040 | 5,940 |
| 元婴 | 8.0 | 4,080 | 11,880 |
| 化神 | 16.0 | 8,160 | 23,760 |
| 炼虚 | 32.0 | 16,320 | 47,520 |
| 合体 | 64.0 | 32,640 | 95,040 |
| 大乘 | 128.0 | 65,280 | 190,080 |

> 基准值以炼气期属性（神识22/悟性20）计算：`(22 + 20×0.6) × 15 = 510`。法修典型值以成长期加点（悟性65/神识60）计算。
> 数值与 attribute-system.md §5.2 完全对齐。build 加点时直接引用该公式，无需另行定义。如需在代码中实现，调用 `attribute_system.calc_max_mp(...)` 即可。

> 上表真元上限按各流派成长期典型属性计算，实际值因个体加点差异浮动 ±15%。

### 3.3 真元用途

| 消耗项 | 消耗规则 | 说明 |
|--------|----------|------|
| 施放术法 | 技能基础消耗 × 境界系数 | 法修核心消耗项，高悟性可降低法力消耗百分比 |
| 驱动法宝 | 法宝每秒消耗 × 品质系数 | 召唤/锻体常驻消耗，品质越高消耗越大 |
| 维持灵宠 | 每只灵宠每秒消耗基础值 | 召唤流派核心持续消耗，灵兽越多负担越大 |
| 护体罡气 | 开启后每秒消耗真元上限 × 1% | 通用防御技，锻体/剑修在面对高爆发时开启 |

### 3.4 真元恢复

| 恢复方式 | 恢复公式 | 说明 |
|----------|----------|------|
| 自然恢复 | 每 tick 恢复 `max_mp × 0.01 × realm_coefficient` | 基础恢复，战斗中也生效 |
| 丹药恢复 | 固定值 + 百分比混合（如回元丹 +200 +5%） | 战斗外/战斗间隙使用 |
| 打坐加速 | 自然恢复速度 × 3，且额外增加 `神识 × 0.5` 每 tick | 脱战专用，不可在战斗中使用 |

```python
# 真元恢复 tick 计算（每 tick ≈ 2 秒）
def mana_regen_tick(player, is_meditating=False):
    base_regen = player.max_mana * 0.01 * player.realm_coefficient()
    if is_meditating:
        return base_regen * 3 + player.divine_sense * 0.5
    return base_regen
```

## §4 流派可切换说明

### 4.1 非锁定机制

流派不由创建角色时的选择决定，而是由**天资点分配方案**自然定义。天资点是自由分配的，同一角色可以在不同等级阶段切换打法：

| 切换方式 | 限制 | 代价 |
|----------|------|------|
| 同角色内切换 | 无系统限制 | 已投入天资点不可撤回，只能通过后续升级获取的新点数逐渐调整 |
| 转世后切换 | 无限制 | 所有属性点返还，可按新流派自由分配（详见 §5） |

**示例**：一名炼气期锻体玩家（根骨 40），在筑基期想转为法修，无法将根骨的 40 点撤回加到悟性上。他只能将后续升级所得的约 30 点全加悟性，形成"根骨 40 + 悟性 30"的混合架构——强度不如从头专注法修，但生存能力更强。

### 4.2 非转世切换的推荐场景

| 原流派 → 目标流派 | 可行性 | 推荐理由 |
|-------------------|--------|----------|
| 锻体 → 剑修 | ★★★★☆ | 根骨可转化为近战生存，补充身法即可 |
| 剑修 → 锻体 | ★★★★☆ | 身法对前排也有用，根骨天花板略低但可接受 |
| 法修 → 召唤 | ★★★☆☆ | 神识共用，悟性对召唤也有帮助，但错过根骨成长 |
| 召唤 → 法修 | ★★★★★ | 神识+悟性直接转化，最佳切换路线 |
| 锻体 → 法修 | ★★☆☆☆ | 根骨对法修收益极低，浪费大量点数，不推荐 |
| 法修 → 锻体 | ★☆☆☆☆ | 严重不推荐，悟性对体修收益极低 |

## §5 转世重置（简要）

### 5.1 转世重置规则

| 项目 | 规则 |
|------|------|
| 属性重置 | 所有已分配天资点全部返还（249 点全额），可按新流派重新分配 |

| 保留项 | 流派经验、称号、成就、已解锁功法（等级归零需重新修炼） |
| 清除项 | 当前等级、境界、属性点分配方案、装备绑定（部分） |
