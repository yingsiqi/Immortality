## §1 文档定位

### 角色系统在游戏架构中的位置

角色系统是整个游戏最底层的 **"玩家数据容器"**——所有战斗、修炼、装备等系统的输入输出都围绕角色属性展开。

| 层级 | 系统 | 关系说明 |
|------|------|----------|
| **L1 规范** | §1 灵根 · §2 属性 · §3 战斗面板 | 角色系统是 L1 规范在**系统层面**的完整展开与落地实现 |
| **L3 游戏系统** | combat | 读取角色战斗面板（气血/真元/杀伤/威能/命中/闪避/暴击）进行计算 |
| | cultivation | 读取境界/灵根判定突破条件，突破成功后回写属性 |
| | equipment | 装备属性加成叠加到角色面板，法宝受神识上限约束 |
| **自身** | character | 定义属性的**数据结构、分配规则、增长曲线、状态管理** |

### 核心职责

1. **数据定义**：灵根纯度、五维属性、战斗面板、境界等级的数据结构
2. **分配规则**：RO 式消耗体系（249 天资点自由分配）
3. **增长曲线**：属性随境界的自然增长与突破跃迁
4. **状态管理**：16 项角色状态的触发/持续/解除全生命周期
5. **创角流程**：PV Demo 核心——玩家进入游戏后的第一个交互体验

## §3 关键概念速查

| # | 概念 | 一句话定义 |
|---|------|-----------|
| 1 | **灵根纯度** | 100 点五行自由分配（单项上限 90，纯 100 需转世），决定灵根类型（天/真/伪/异）和技能限制 |
| 2 | **天资点** | 每级 3 点，83 级共 249 点，RO 式消耗递增（0~20 区段每点 1 天资，100+ 区段每点 6 天资） |
| 3 | **五维属性** | 根骨/悟性/神识/身法/机缘，决定战斗面板的五大基础维度，另有隐藏属性毅力 |
| 4 | **战斗面板** | 由五维属性换算的二次属性：气血上限、真元上限、体修杀伤、术法威能、命中率、闪避率、暴击率、暴击伤害 |
| 5 | **境** | 83 级 = 炼气 13 级 + 筑基~大乘 7 大境界各 10 级（前期 3+中期 3+后期 3+圆满 1） |
| 6 | **流派** | 锻体/法修/剑修/召唤，非锁定自由加点体系，点数有限自然精一系 |
| 7 | **突破** | 小阶段（丹药+任务，无惩罚）→ 圆满（+心魔判定，失败 -10% 修为）→ 大境界（渡劫+心魔，失败 -20% 修为 + debuff） |
| 8 | **心魔** | 突破失败的负面状态：悟性 -20%、突破成功率 -10%、修为自动流失 0.1%/min，3 层可叠，需清心丹或试炼任务消除 |
| 9 | **真元** | 由神识×悟性驱动的能量货币，消耗于施放术法/驱动法宝/维持灵宠/护体罡气 |
| 10 | **转世** | 属性重置 + 灵根上限突破（+10/转，最多 3 转）+ 涅槃印记永久 buff（技能学习 +25%、经验 +25%） |

## §4 设计约束与边界

| 约束项 | 具体规则 |
|--------|----------|
| **种族** | 暂仅**人族**，无种族系统设计 |
| **流派** | **非锁定**加点，自由切换但已分配点数不可撤回（非转世），只能通过后续加点调整方向 |
| **灵根上限** | 单属上限 **90**，纯 100 需转世突破；转世后每转 +10 上限 |
| **等级结构** | 炼气 **13** 级 + 7 大境界各 **10** 级 = **83** 级 |
| **天资点** | 每级 **3** 点，共计 **249** 点 |
| **RO 消耗** | 6 档递增：0~20→1, 21~40→2, 41~60→3, 61~80→4, 81~99→5, **100+→6**（仅转世可达） |
| **境界显示** | **公开**仅显示大境界名称；**探查**显示前中后期；**自己**显示完整层级（如炼气七层） |
| **异灵根** | 雷/冰/风/暗，由两种五行异变融合，创角 5% 概率触发或游戏内事件获得 |
