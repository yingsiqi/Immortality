# 角色创建流程

> 本文档是角色系统拆分子文档之一，聚焦玩家从零创建角色的完整交互流程，覆盖首次创角与转世创角两种场景，对齐 L1 规范。创角体验是 PV Demo 的核心首次交互，设计目标是**在 2~3 分钟内完成沉浸式角色构建**。

---

## §1 创角流程总览

### 1.1 六步创角流程

```text
┌─────────────────────────────────────────────────────┐
│                    创角六步流程                        │
├─────────────────────────────────────────────────────┤
│                                                       │
│  Step 1 ──── 选择性别/外观                            │
│             (8 套预设模板，种族仅"人族")              │
│               │                                       │
│               ▼                                       │
│  Step 2 ──── 灵根纯度分配                              │
│             (100 点分配界面，5 行滑块)                 │
│               │                                       │
│               ▼                                       │
│  Step 3 ──── 灵根判定动画                              │
│             (天灵根/真灵根/伪灵根/5%异灵根)           │
│               │                                       │
│               ▼                                       │
│  Step 4 ──── 流派倾向选择                              │
│             (推荐但不锁定，系统高亮推荐)               │
│               │                                       │
│               ▼                                       │
│  Step 5 ──── 初始属性预分配                            │
│             (24 天资点，3 套预设 / 手动微调)           │
│               │                                       │
│               ▼                                       │
│  Step 6 ──── 命名 + 确认创建                          │
│             (创角预览 → 确认 → 动画 → 进入场景)       │
│                                                       │
└─────────────────────────────────────────────────────┘
```

### 1.2 流程概览表

| 步骤 | 界面名称 | 核心交互 | 耗时参考 | 可回退 |
|------|----------|----------|----------|--------|
| Step 1 | 凡尘之相 | 选择性别 + 外观模板 | ~15s | 可回退 |
| Step 2 | 灵根初现 | 5 行滑块分配 100 点纯度 | ~30s | 可回退至 Step 1 |
| Step 3 | 灵根觉醒（动画） | 自动播放判定动画 | ~5s | 不可交互 |
| Step 4 | 问道择途 | 选择流派倾向，参考推荐 | ~20s | 可回退至 Step 2 |
| Step 5 | 天资淬炼 | 选择预设或手动加点 | ~40s | 可回退至 Step 4 |
| Step 6 | 命定之名 | 输入姓名 + 预览确认 | ~15s | 可回退至 Step 1 |

**PV Demo 策略**：每个步骤在首次进入时显示简短引导气泡（2s 自动消失），速通玩家可直接操作。若玩家连续 15s 无操作，触发 NPC 语音提示。

## §2 灵根分配界面设计

### 2.1 交互元素清单

| 元素 | 类型 | 说明 |
|------|------|------|
| 五行滑块 × 5 | 拖动滑块（0~90，步长 1） | 金/木/水/火/土 各一，实时数值显示 |
| 剩余点数指示器 | 文本（剩余 X 点 / 总计 100 点） | 分配总点数 ≤100 时绿色，>100 时红色 |
| 纯度百分比 | 环状进度条 | (已分配点数 / 100) × 100%，绿色 < 100%，红色 > 100% |
| 确认按钮 | 按钮 | 纯度 = 100 点且无单项 > 90 时可点击；>100 或 <100 时禁用 |
| 异灵根触发进度条 | 模糊进度条（无精确数值） | 显示"引动天地灵气中..."动画效果 |
| 重置按钮 | 按钮 | 一键将所有滑块归零 |
| 随机分配按钮 | 按钮 | 系统自动生成一套随机分配方案 |

### 2.2 交互规则

| 规则 | 说明 |
|------|------|
| 滑块范围 | 每项 0~90，拖动至 90 时出现提示：**"天灵根之资，然大道独行"** |
| 实时校验 | 总分配点数实时核算，>100 时确认按钮锁定，滑块变红 |
| 确认条件 | 总分配点数 **恰好等于 100**，且无单项超过 90 |
| 视觉反馈 | 非当前操作的滑块颜色渐变透明度，当前滑块高亮发光 |
| 引导提示 | 首次进入时显示"将 100 点灵根纯度分配至五行属性"引导气泡 |

#### 2.2.1 异灵根触发机制

灵根分配确认后、判定动画播放前，有 **5% 概率** 触发异灵根检测流程：

```text
分配确认 → [5% 概率] → "引动天地灵气中..." 进度条动画 (2~3s)
                       → 判定异灵根类型 → 播放异灵根觉醒动画
           [95% 概率] → 进入常规灵根判定
```

异灵根由两种五行属性融合生成，融合规则：

| 异灵根 | 融合条件 | 纯度要求 |
|--------|----------|----------|
| 雷灵根 | 金 + 木 | 金 ≥ 30 且 木 ≥ 30 |
| 冰灵根 | 水 + 金 | 水 ≥ 30 且 金 ≥ 30 |
| 风灵根 | 木 + 火 | 木 ≥ 30 且 火 ≥ 30 |
| 暗灵根 | 水 + 火 | 水 ≥ 30 且 火 ≥ 30 |

触发后，两种源属性的纯度按 7:3 权重折算为异灵根纯度，作为后续异灵根功法修炼的基础属性。详见 [attribute-system.md §1.1 异灵根](#attribute-system.md)。

### 2.3 灵根判定反馈动画

| 灵根类型 | 判定条件 | 视觉特效 | 文字反馈 |
|----------|----------|----------|----------|
| 天灵根 | 单属性 ≥ 90 | 金色光柱冲天 + 屏幕震动 | **"天降奇才！XX 灵根纯度惊人！"** |
| 真灵根 | 双属性合计 ≥ 100 且单属性 ≥ 30 | 蓝绿色光效环绕 | **"资质上佳，XX/YY 双灵根共鸣！"** |
| 伪灵根 | 各属性 15~25 | 灰色光效渐隐 | **"资质平平，然天道酬勤。"** |
| 异灵根 | 5% 触发 | 紫色雷光炸裂 + 电弧闪烁 | **"天地异变！灵根异化为 XX 灵根！"** |

> 判定逻辑详情参见 [attribute-system.md §1.1 灵根纯度分配](#attribute-system.md)。

## §3 流派倾向推荐逻辑

### 3.1 推荐规则

流派倾向选择界面的上下文：玩家已完成灵根分配和判定，系统根据灵根属性和初始五维倾向给出流派推荐。

| 判定条件 | 推荐流派 | 推荐理由 | 推荐强度 |
|----------|----------|----------|----------|
| 根骨 ≥ 50 | 锻体 | 天生神力，适合近战肉搏 | ⭐⭐⭐ 强推荐 |
| 悟性 ≥ 50 | 法修 | 悟性超群，术法威能惊人 | ⭐⭐⭐ 强推荐 |
| 身法 + 悟性 ≥ 80 | 剑修 | 身法迅捷，剑道可期 | ⭐⭐ 推荐 |
| 神识 ≥ 50 | 召唤 | 神识强大，可驭万灵 | ⭐⭐⭐ 强推荐 |
| 以上均未达到 | 自选 | 五维均衡，万法随心 | 无推荐，四流派平铺 |

### 3.2 界面交互规则

| 规则 | 说明 |
|------|------|
| 推荐高亮 | 满足条件的流派卡片边框发光 + 右上角显示"推荐"标签 |
| 多流派满足 | 同时满足多个条件时，按优先级排序高亮，但均标注推荐标签 |
| 流派简介 | 点击流派卡片展开详情：核心属性、战斗风格、加点倾向、适用人群 |
| 不锁定机制 | 选择仅为倾向建议，不锁定流派——后续加点自由分配 |
| 跳过选项 | 可跳过直接进入 Step 5（以无流派倾向状态进入属性分配） |

> 流派定位与战斗风格详见 [build-system.md §1 四大流派定义](#build-system.md)。

### 3.3 流派卡片展示内容

每个流派卡片包含：

| 字段 | 示例（锻体） |
|------|-------------|
| 流派名称 | 锻体 |
| 图标 | 💪（游戏内使用定制图标） |
| 核心属性 | 根骨（主）、身法（副）、神识（辅） |
| 一句话定位 | 坦克/战士——高血量近战前排 |
| 战斗风格 | 肉身硬抗、近身爆发、控制技 |
| 难度评级 | ★★☆☆☆（上手简单，精通需理解仇恨机制） |

## §4 初始属性预分配

### 4.1 基础规则

- **初始天资**：8 级 × 3 点 = **24 点**

- RO 式消耗与完整 249 天资体系对齐，详见 [attribute-system.md §4 RO 式属性消耗表](#attribute-system.md)

| 21~40 | 2 |
| 41~60 | 3 |

## §6 非首次创角（转世创角）

### 6.1 检测与入口

| 规则 | 说明 |
|------|------|
| 检测时机 | 点击"新建角色"时，系统检测账户是否有转世角色 |
| 有转世角色 | 弹出提示："检测到前世道痕，是否继承前世之力？" ⇒ 进入转世创角流程 |
| 无转世角色 | 正常进入首次创角流程（§1~§5） |
| 转世角色定义 | 已完成至少一次大轮回（飞升/兵解）的角色 |

### 6.2 转世创角差异

| 项目 | 首次创角 | 转世创角 |
|------|----------|----------|
| 灵根上限 | 100 点，单项 ≤ 90 | 基础 100 + 转世次数 × 10，单项 ≤ 最低(90 + 转世次数 × 10, 100) |
| 初始天资 | 24 点 | 24 点（重置后重新分配，不继承前世的加点） |
| 外观选择 | 仅首次选择 | 可"沿用前世外观"或"重新选择" |
| 名称 | 全新输入 | 可"沿用前世道号"或"重新命名" |
| 流派推荐 | 基于灵根分配 | 基于灵根分配 + **基于前世流派经验推荐** |
| 额外界面 | 无 | 涅槃印记层数展示 + 特性确认 |
| 初始场景 | 凡人村落 | XX 山脚（根据前世最高境界决定） |
