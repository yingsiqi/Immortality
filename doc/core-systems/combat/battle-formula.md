# 战斗数值公式

## 一、公式总览

### 1.1 完整伤害公式

```
基础伤害 = 攻击力 × 境界系数 × 技能系数 × (1 + 灵根纯度加成) × 五行克制系数

暴击判定 → 若暴击：基础伤害 = 基础伤害 × 会心倍率

破防伤害 = MAX(基础伤害 - 目标护体罡气, 基础伤害 × 0.10)

最终伤害 = 破防伤害 × (1 - 目标减伤%)
```

> **境界系数**：玩家与怪物各使用自己的境界系数。攻击方用攻击者境界系数计算基础伤害；防御方的境界系数通过护体罡气和减伤%间接体现（护体罡气依赖真元上限，真元上限已含境界系数）。详见 §1.3。

### 1.2 最低伤害保底

护体罡气不会让伤害归零。最低伤害 = 基础伤害 × 10%。

### 1.3 境界系数

> 来源 attribute-system.md §6.2。境界系数 = 2^(境界等级-1)，炼气=1 为基准。

| 境界 | 系数 | 倍率说明 |
|------|:----:|---------|
| 炼气期 | 1.0 | 基准值，伤害无放大 |
| 筑基期 | 2.0 | 2 倍 |
| 金丹期 | 4.0 | 4 倍 |
| 元婴期 | 8.0 | 8 倍 |
| 化神期 | 16.0 | 16 倍 |
| 炼虚期 | 32.0 | 32 倍 |
| 合体期 | 64.0 | 64 倍 |
| 大乘期 | 128.0 | 128 倍 |

> 示例：金丹期剑修攻击力120 × 境界系数4.0 = 等效480 基础攻击力。元婴期怪物防御时，护体罡气 = 真元上限×0.3 + 悟性×5，其中真元上限已包含其自身的境界系数×8.0。

#### 跨境界战斗

| 场景 | 效果 |
|------|------|
| 玩家 vs 高 1 境怪物 | 玩家系数1.0 vs 怪物系数2.0，基础伤害差距 2 倍 |
| 玩家 vs 低 2 境怪物 | 玩家系数2.0 vs 怪物系数0.5（未列出，通过真元环比折算），碾压效果明显 |
| PVP 同境界 | 双方系数相等，回到纯粹的属性与操作比拼 |

## 二、参数定义

### 2.1 攻击力

| 流派 | 攻击力公式 | 说明 |
|------|-----------|------|
| 锻体流 | `根骨 × 2.0` | 体修杀伤 |
| 法修流 | `悟性 × 2.0` | 术法威能 |
| 剑修流 | `(神识 × 1.5 + 悟性 × 0.5)` | 御器+术法混合 |
| 召唤流 | `神识 × 1.8` | 灵宠伤害继承本体神识 |

### 2.2 灵根纯度加成

> 来源 attribute-system.md §1.2。

```
灵根纯度加成 = 灵根纯度 × 0.0025
```

- 上限 0.25（100 纯度）
- 天灵根典型值：90 纯度 → +22.5%
- 双修典型值：60 纯度 → +15%
- 杂灵根典型值：20 纯度 → +5%
- 技能无五行属性（如部分物理技能）时加成 = 0

### 2.3 五行克制系数

| 关系 | 系数 | 触发条件 |
|------|------|----------|
| 克制 | **1.25** | 攻击属性克目标属性（如火→金） |
| 平 | **1.00** | 无克制关系或目标无属性 |
| 被克 | **0.75** | 攻击属性被目标属性克（如火→水） |
| PVP 放大 | **1.30 / 0.70** | PVP 场景克制系数 ±30% |

> 怪物无属性（五行=无）时，永远走 1.00 平线。

### 2.4 会心系统

> 来源 attribute-system.md §5.9-5.10。会心系统为线性成长模型，无需阈值即可产生收益。

| 参数 | 公式 | 说明 |
|------|------|------|
| 基础暴伤 | `150%` | 暴击时最低伤害倍率（零投入时） |
| 会心率 | `0.02 + 机缘 × 0.004` | 上限 50%。机缘=100 时 → 42%；机缘=0 时 2% 保底 |
| 会心倍率 | `1.5 + 机缘 × 0.01 + 悟性 × 0.005` | 上限 3.0。机缘=100/悟性=20 时 → 1.5+1.0+0.1 = 2.6 |
| 会心判定 | 每次攻击独立随机 | 暴击时伤害数字加特效标记 |

### 2.5 护体罡气（防御减伤前）

> 来源 attribute-system.md §5.6。护体罡气由真元上限和悟性共同决定，法修/剑修收益最高。

| 对象 | 公式 | 说明 |
|------|------|------|
| 玩家 | `真元上限 × 0.3 + 悟性 × 5` | 真元上限越高、悟性越高则护体越强。炼气期约 167-338 |
| 怪物 | `怪物 防御值` | 直接使用怪物表中的防御数值 |

### 2.6 减伤%

| 来源 | 数值 | 叠加方式 |
|------|------|----------|
| 格挡中 | 40% | 与被动减伤乘算 |
| 金刚罩等护盾技能 | 技能描述值 | 同类取最高 |
| 装备词条 | 各装备独立 | 乘算叠加（1 - Π(1 - xᵢ)） |
| 心魔战/渡劫 | 0% | 特殊模式禁用减伤 |

> Alpha 阶段默认 减伤% = 0，仅格挡时启用。

## 三、气血与真元

### 3.1 气血上限

```
玩家气血 = 根骨 × 20 + 装备加成
怪物气血 = 怪物表中「气血」值
```

### 3.2 真元上限

```
真元上限 = (神识 + 悟性 × 0.6) × 15 × 境界系数
```

境界系数表（来源 attribute-system.md §6.2）：

| 境界 | 境界系数（2^n） |
|------|:--------------:|
| 炼气期 | 1.0 |
| 筑基期 | 2.0 |
| 金丹期 | 4.0 |
| 元婴期 | 8.0 |
| 化神期 | 16.0 |
| 炼虚期 | 32.0 |
| 合体期 | 64.0 |
| 大乘期 | 128.0 |

> 炼气期示例：神识22 + 悟性20×0.6 = 34, ×15 = 510, 境界系数1.0 → 真元上限510。

### 3.3 真元回复

| 状态 | 速率 | 说明 |
|------|------|------|
| 战斗中自然回复 | `神识 × 0.1` 点/秒 | 仅神识影响速率 |
| 非战斗打坐 | `真元上限 × 5%` 点/秒 | 20 秒回满 |
| 丹药回复 | 丹药表数值 | 战斗中可用，共享丹药 CD |
