# 战斗数值公式

> 版本：v1.0-alpha | 最后更新：2026-07-18 | 基于 L1 §1-3、L3 §2.3 定稿

---

## 一、公式总览

### 1.1 完整伤害公式

```
基础伤害 = 攻击力 × 技能系数 × (1 + 灵根纯度加成) × 五行克制系数

暴击判定 → 若暴击：基础伤害 = 基础伤害 × 会心倍率

破防伤害 = MAX(基础伤害 - 目标护体罡气, 基础伤害 × 0.10)

最终伤害 = 破防伤害 × (1 - 目标减伤%)
```

### 1.2 最低伤害保底

护体罡气不会让伤害归零。最低伤害 = 基础伤害 × 10%。

## 二、参数定义

### 2.1 攻击力

| 流派 | 攻击力公式 | 说明 |
|------|-----------|------|
| 锻体流 | `根骨 × 2.0` | 体修杀伤 |
| 法修流 | `悟性 × 2.0` | 术法威能 |
| 剑修流 | `(神识 × 1.5 + 悟性 × 0.5)` | 御器+术法混合 |
| 召唤流 | `神识 × 1.8` | 灵宠伤害继承本体神识 |

### 2.2 灵根纯度加成

```
灵根纯度加成 = 对应属性灵根纯度 / 200
```

- 上限 0.50（100 纯度）
- 天灵根典型值：90 纯度 → +45%
- 双修典型值：60 纯度 → +30%
- 杂灵根典型值：20 纯度 → +10%
- 技能无五行属性（如部分物理技能）时加成 = 0

### 2.3 五行克制系数

| 关系 | 系数 | 触发条件 |
|------|------|----------|
| 克制 | **1.25** | 攻击属性克目标属性（如火→金） |
| 平 | **1.00** | 无克制关系或目标无属性 |
| 被克 | **0.75** | 攻击属性被目标属性克（如火→水） |
| PVP 放大 | **1.30 / 0.70** | PVP 场景克制系数 ±30% |

> 怪物无属性（五行=无）时，永远走 1.00 平线。

### 2.4 会心系统

| 参数 | 公式 | 说明 |
|------|------|------|
| 会心率 | `机缘 / 300` | 上限 33%（机缘=100 时） |
| 会心倍率 | `1 + 机缘 / 1000` | 来源 L3 §2.3。上限 1.10（机缘=100 时），初始倍率 1.0 |
| 会心判定 | 每次攻击独立随机 | 暴击时伤害数字加特效标记 |

### 2.5 护体罡气（防御减伤前）

| 对象 | 公式 | 说明 |
|------|------|------|
| 玩家 | `真元上限 × 0.05 + 悟性 × 0.8` | 来源 L1 §3.1：护体罡气由真元上限+悟性共同决定。法修/剑修收益最高 |
| 怪物 | `怪物 防御值` | 直接使用怪物表中的防御数值 |

### 2.6 减伤%

| 来源 | 数值 | 叠加方式 |
|------|------|----------|
| 格挡中 | 40% | 与被动减伤乘算 |
| 金刚罩等护盾技能 | 技能描述值 | 同类取最高 |
| 装备词条 | 各装备独立 | 乘算叠加（1 - Π(1 - xᵢ)） |
| 心魔战/渡劫 | 0% | 特殊模式禁用减伤 |

> Alpha 阶段默认 减伤% = 0，仅格挡时启用。

### 3.1 气血上限

```
玩家气血 = 根骨 × 20 + 装备加成
怪物气血 = 怪物表中「气血」值
```

### 3.2 真元上限

```
真元上限 = (神识 + 悟性) × 10
```

### 3.3 真元回复
