# 技能系统

> 基于 L4 §3（术法体系）、data/skills/shufa.md（术法数据库）和 L3 §2.3（伤害公式）。

## 一、技能分类体系

### 1.1 按流派分类

基于 L4 §3.5，六个战斗流派各有独立的技能树：

| 流派 | 属性倾向 | 定位 | 武器 | 普攻终结效果 |
|------|----------|------|------|-------------|
| **锻体流** | 土·体修 | 近战坦克 / 单体爆发 | 拳套 / 重斧 | 击退 + 硬直 0.5s |
| **剑修流** | 金·剑修 | 远程穿刺 / 破甲 DPS | 飞剑 | 破甲（无视 10% 防御） |
| **火烈流** | 火·法修 | AOE 爆发 / 暴击 | 法杖 / 灵幡 | 小范围爆炸（半径 2m） |
| **水月流** | 水·法修 | 控制 / 防御 / 隐匿 | 法珠 / 玉笛 | 减速 20%，持续 2s |
| **木灵流** | 木·法修 | 持续伤害 / 治疗 / 召唤辅助 | 灵木杖 / 药鼎 | 恢复自身 2% 最大生命 |
| **召唤流** | 无·御灵 | 灵宠大军 / 阵法 / 傀儡 | 灵幡 / 虫笛 | 灵宠追击目标 2s |

### 1.2 按技能类型分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **主动术法** | 战斗中手动释放，消耗真元 | 崩山拳、三味真火、飞剑穿刺 |
| **被动术法** | 装备后自动生效，不消耗资源 | 铁骨（防御+5%）、灵气亲和（真元恢复+10%） |
| **通用术法** | 所有流派均可学习 | 基础吐纳（打坐效率+10%） |
| **流派术法** | 仅特定流派可学习 | 锻体：金刚体；火烈：烈焰风暴 |
| **传承术法** | 师承/宗门/奇遇专属 | 青元剑诀·第一式、大衍诀·神念锁 |
| **五行术法** | 带五行属性标签，受灵根纯度和克制影响 | 三味真火（火）、玄冰诀（水） |

### 1.3 按五行属性分类

每个术法标注五行属性（或异灵根属性/无属性），受 L1 §1.4 五行克制影响：

| 属性 | 核心特点 | 技能侧重 |
|------|----------|---------|
| **金** | 极致锋锐，单体爆发 | 穿透、破甲、飞剑技 |
| **木** | 生生不息，治疗控制 | 回血、缠绕、召唤 |
| **水** | 柔中带刚，减速冰冻 | 减速、冰冻、净化、护盾 |
| **火** | 焚天灭地，群体灼烧 | 灼烧、AOE、爆发 |
| **土** | 厚德载物，坚不可摧 | 护甲、减伤、嘲讽 |
| **无** | 无五行属性 | 物理伤害、通用技，不受克制影响 |

## 二、技能释放机制

### 2.1 施法系统（前摇/后摇）

```
前摇时间 = 基础前摇 × (1 - 施法速度加成) × 技能系数
```

| 影响因素 | 说明 |
|----------|------|
| 身法 | 提升施法速度（每 10 点身法 -3% 前摇） |
| 装备 | 施法速度加成词条 |
| 技能熟练度 | 每级 -2% 前摇（上限 -20%） |
| 特殊状态 | 如「顿悟」buff 下前摇减半 |

#### 施法打断条件

- 受到伤害超过当前气血的 10%
- 被控制技能命中（眩晕/冰冻/沉默）
- 主动取消施法
- 目标超出射程

**打断惩罚**：消耗已支付真元的 50%，进入 1 秒技能锁定（无法释放任何技能）。

### 2.2 普攻连段

每个流派 3 段基础连招（来源 L3 §2.2）：

| 流派 | 第 1 段 | 第 2 段 | 第 3 段（终结） | 特殊效果 |
|------|---------|---------|-----------------|----------|
| **锻体** | 拳击 | 肘击 | 崩山靠 | 击退 + 目标硬直 0.5s |
| **法修** | 灵气弹 | 双弹 | 爆裂弹 | 小范围爆炸（半径 2m） |
| **剑修** | 横斩 | 竖斩 | 穿心刺 | 破甲（无视 10% 防御） |
| **召唤** | 灵宠扑击 | 灵宠撕咬 | 灵宠猛袭 | 灵宠追击目标 2s |

> 水月流与火烈流共用法修普攻模板，木灵流普攻终结改为恢复自身 2% 最大生命。

### 2.3 术法槽位（L4 §3.4a）

| 境界 | 主动槽位 | 被动槽位 |
|------|---------|---------|
| 炼气 | 3 | 1 |
| 筑基 | 4 | 1 |
| 结丹 | 5 | 2 |
| 元婴 | 6 | 3 |
| 化神 | 7 | 3 |
| 大乘 | 8 | 4 |

> 法宝技能槽独立计算（4 槽位），不受此表限制。

## 三、技能冷却体系

### 3.1 全局冷却（GCD）

| 规则 | 值 |
|------|-----|
| GCD | 1.0 秒，所有主动技能共享 |
| 移动技能 GCD | 0.5 秒 |
| 瞬发技能 GCD | 0.3 秒 |

### 3.2 独立冷却

```
实际冷却 = 基础冷却 × (1 - 冷却缩减%) × 技能等级系数
```

| 冷却缩减来源 | 上限 |
|-------------|------|
| 装备属性 | 40% |
| 天赋/被动技能 | 25% |
| 临时 BUFF | 20% |
| **总计上限** | **70%** |

### 3.3 技能冷却范围（参考 shufa.md 数值基准）

| 层级 | CD 范围 |
|------|---------|
| 基础层 | 5 ~ 15s |
| 进阶层 | 10 ~ 30s |
| 专家层 | 20 ~ 60s |
| 大师层 | 40 ~ 120s |
| 隐藏层 | 120 ~ 600s |

## 四、技能树系统

### 4.1 技能树层级结构

每个流派的技能树按境界划分为 5 层（来源 shufa.md §2.1）：

```
基础层（炼气期自动解锁）
  ├── 进阶层（筑基期 + 基础层已解锁 ≥ 3 节点）
  │     ├── 专家层（结丹期 + 进阶层已解锁 ≥ 3 节点）
  │     │     ├── 大师层（元婴期 + 专家层已解锁 ≥ 2 节点）
  │     │     │     └── 隐藏层（化神期 + 特定奇遇/条件）
```

### 4.2 数值基准（来源 shufa.md §2.4）

所有伤害数值以「攻击力 × 系数」表示，随装备/境界动态缩放：

| 层级 | 伤害系数范围 | CD 范围 |
|------|------------|---------|
| 基础 | 0.8 ~ 1.5 | 5 ~ 15s |
| 进阶 | 1.2 ~ 2.5 | 10 ~ 30s |
| 专家 | 2.0 ~ 4.0 | 20 ~ 60s |
| 大师 | 3.0 ~ 6.0 | 40 ~ 120s |
| 隐藏 | 5.0 ~ 10.0 | 120 ~ 600s |

### 4.3 技能获取方式

| 方式 | 说明 |
|------|------|
| **灵根解锁** | 某属性灵根纯度达到阈值后自动解锁对应属性的基础技能树 |
| **流派自动习得** | 选择流派时自动获得该流派前 3 个基础技能 |
| **功法附赠** | 学习功法时随功法附带 1-3 个专属技能 |
| **技能书掉落** | 怪物/Boss 掉落技能书，使用后习得 |
| **宗门技能导师** | 消耗宗门贡献学习流派进阶技能 |
| **师承传授** | 师父直接传授技能（可跳过学习前置条件） |
| **奇遇/任务** | 特定剧情线奖励稀有技能 |

## 五、技能升级系统

### 5.1 熟练度机制

```
技能等级 N 所需熟练度 = 100 × (N ^ 1.8)
```

| 属性 | 说明 |
|------|------|
| **熟练度获取** | 每次释放术法获得固定熟练度 + 命中目标额外加成 |
| **等级上限** | 受境界限制（见下表） |
| **升级效果** | 每级提升基础伤害/效果 +10%，真元消耗 -2%（上限 -30%） |

#### 术法等级上限

| 境界 | 术法等级上限 |
|------|------------|
| 炼气 | 5 |
| 筑基 | 10 |
| 结丹 | 15 |
| 元婴 | 20 |
| 化神 | 25 |
| 大乘及以上 | 30 |

### 5.2 技能突破

| 条件 | 说明 |
|------|------|
| **触发条件** | 技能熟练度达到当前等级上限 |
| **消耗** | 特定突破材料 + 灵石 |
| **突破效果** | 等级上限提升 + 解锁新的技能效果 |
| **技能变异** | 突破时概率触发——威力/范围/效果/消耗/冷却变异 |

### 5.3 技能变异类型

| 变异类型 | 效果 |
|---------|------|
| **威力变异** | 伤害系数 +20% |
| **范围变异** | 作用范围 +30% |
| **效果变异** | 附加特殊效果（如额外减速/灼烧） |
| **消耗变异** | 真元消耗 -20% |
| **冷却变异** | 独立冷却 -20% |

## 六、技能平衡性设计

### 6.1 伤害平衡原则

| 原则 | 说明 |
|------|------|
| 单体 DPS > 群体 DPS | 单体技能应有更高的单目标伤害 |
| 长冷却 DPS > 短冷却 DPS | 大招型技能单位时间伤害更高 |
| 高消耗 DPS > 低消耗 DPS | 高真元消耗对应更高收益 |
| 难操作 DPS > 易操作 DPS | 需要精准时机/连招的技能收益更高 |

### 6.2 控制技能平衡

| 类型 | 说明 |
|------|------|
| 硬控制 | 眩晕、冰冻等，持续时间 1-3s |
| 软控制 | 减速、沉默等，持续时间 3-8s |
| 控制免疫 | 精英/Boss 具有递减抗性 |
| 控制递减 | 连续控制效果每次递减 50% |

### 6.3 真元消耗设计

| 技能类型 | 消耗特征 |
|---------|---------|
| 基础技能 | 低消耗（20-50 真元），可持续使用 |
| 进阶技能 | 中等消耗（50-120 真元），需规划使用 |
| 终极技能 | 高消耗（120-300 真元），关键时刻使用 |

### 6.4 五行克制对技能的影响

所有带五行属性的技能受 L1 §1.4 五行克制影响：

| 关系 | 伤害修正 |
|------|---------|
| 克制（如火→金） | ×1.25（PVE）/ ×1.30（PVP） |
| 平 | ×1.00 |
| 被克（如火→水） | ×0.75（PVE）/ ×0.70（PVP） |
