# P1 资产需求清单

> 本文档列出 Alpha P1 阶段（MVP 核心纵切）所需的全部资产。
> P1 阶段默认使用 Unity 原语占位，不阻塞功能开发。美术风格确认后可替换为 AI 生成资产。

## 资产获取策略

按 `agent-execution-protocol.md` 第 5.2 节资产获取优先级执行：

1. **P0** — 复用现有资产（当前 `game/Assets/` 仅有 `.gitkeep`，无可用资产）
2. **P1** — Unity 原语占位（**P1 阶段默认**）
3. **P2** — AI 生成（需美术风格确认，见 `design-review-queue.md` A-01~A-03）
4. **P3** — 外部导入

## 资产目录约定

```
game/Assets/
├── Art/
│   ├── Sprites/          # 2D 精灵
│   │   ├── Icons/        # 物品/技能/境界图标
│   │   ├── UI/           # UI 背景和元素
│   │   └── Characters/   # 角色肖像
│   ├── Animations/       # 动画文件
│   └── Materials/        # 材质文件
├── Prefabs/
│   ├── Characters/       # 角色 Prefab
│   ├── Effects/          # 特效 Prefab
│   └── UI/               # UI Prefab
├── Scenes/               # 场景文件
└── AddressableAssetsData/ # 可寻址资源配置
```
