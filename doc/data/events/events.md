# 事件数据表

> 基于《凡人修仙传》原著世界观与 MMORPG 项目需求，涵蓋 5 大类、38 条事件，覆盖全等级段（Lv1-10）。区域 ID 引用自 data/world/regions.md，奖励 ID 引用自 data/equipment/equipment.md、data/alchemy/pills.md、data/pets/pets.md。

---

## 字段说明

| 字段 | 说明 |
|------|------|
| id | `event_xxx` |
| 名称 | 事件中文名 |
| 分类 | 随机奇遇/天道事件/任务链/连锁事件/赛季事件 |
| 触发区域 | region_xxx（引用 regions.md 真实 ID） |
| 触发条件 | 如「境界≥筑基，击杀赤血蟒精英后10%概率」「每日12:00-14:00」 |
| 触发方式 | 条件触发/概率触发/时间触发/行为触发/任务前置 |
| 推荐等级 | Lv 范围 |
| 人数限制 | 单人/队伍(1-4人)/团队(5-20人)/无限制 |
| 事件类型 | 战斗/收集/护送/解谜/竞速/防守/交易/选择分支 |
| 事件描述 | 2-3 句修仙小说质感描述（触发时的叙事文本） |
| 奖励 | 经验/灵石/装备 ID/材料 ID/丹方 ID/称号/声望 |
| 失败惩罚 | 无/经验减少/物品损失/声望降低 |
| 重复性 | 一次性/每日/每周/可重复/赛季限定 |
| 冷却时间 | 如「72小时」「无冷却」 |
| 分支选择 | 有/无；若有，简述选项及后果 |
| 与其他事件关联 | 任务链 ID 或前置事件 ID（无则「—」） |
| 备注 | 原著来源/原著扩展/游戏原创 |
