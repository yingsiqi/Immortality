| 特征 | 说明 |
|------|------|
| 地理 | 大陆型，越国+大晋+元武国+幕兰草原等广阔地域 |
| 推荐等级 | 越国 Lv1-3（新手区）→ 大晋 Lv6-8（高阶区） |
| 灵气浓度 | 越国稀薄→大晋一般 |
| 主要势力 | 越国七派、大晋仙朝、阴魂宗、天魔宗 |
| 主线定位 | 玩家出生地、新手教学区 → 中期过渡 → 高阶挑战 |

### 乱星海（独立海域大陆）

| 特征 | 说明 |
|------|------|
| 地理 | 群岛海域型，星宫主岛+各岛屿+广阔海域 |

### 关联关系

玩家到达越国后期（Lv3+）可通过**越国-乱星海传送阵**或**无边海航行**前往乱星海。两个大陆的传送节点彼此独立但部分秘境互通。
