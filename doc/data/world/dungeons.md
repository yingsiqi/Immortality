# 副本数据表

> 基于《凡人修仙传》原著世界观，定义 MMORPG 中的 20 个副本（天南 10 + 乱星海 10），覆盖剧情/挑战/材料/团队/秘境五种类型。

---

## 字段说明

| 字段 | 说明 |
|------|------|
| id | `dungeon_xxx` |
| 名称 | 副本名 |
| 所属区域 | region_xxx |
| 层级 | 场景/副本 |
| 类型 | 剧情/挑战/材料/团队/秘境 |
| 推荐等级 | Lv 范围 |
| 人数限制 | 1~20 人 |
| Boss | monster_id（引用 monsters.md 真实 ID） |
| 掉落 | 装备 ID + 材料 ID |
| 进入条件 | 境界/等级/道具/任务 |
| 进入次数限制 | 每日 N 次/每周 N 次/无限 |
| 副本机制 | 机关/谜题/Boss 阶段/限时 |
| 五行场域 | 金/木/水/火/土/均衡/混合 |
| 备注 | 原著来源/游戏原创 |
