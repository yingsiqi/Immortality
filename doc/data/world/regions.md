# 区域地图数据表

> 基于《凡人修仙传》原著世界观，定义 MMORPG 中天南（15 个）与乱星海（15 个）的区域数据。

---

## 字段说明

| 字段 | 说明 |
|------|------|
| id | `region_xxx` |
| 名称 | 区域名 |
| 所属大陆 | continent_tiannan / continent_luanxinghai |
| 层级 | 区域 |
| 区域类型 | 山脉/平原/沙漠/海域/沼泽/城市/宗门 |
| 推荐等级 | Lv 范围 |
| 五行场域 | 金/木/水/火/土/均衡（L7 §3） |
| 子场景数量 | 约数 |
| 安全区 | 是/否/部分 |
| 传送阵 | 有/无 |
| 主要资源 | 采集材料 ID 列表（L7 §3 九维资源向量） |
| 备注 | 原著来源/游戏原创 |
