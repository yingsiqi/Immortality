| 维度 | 天南 | 乱星海 |

| **类型** | 大陆型（山川平原） | 海域型（群岛海域） |
| **新手友好度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| **灵气整体浓度** | 越国稀薄 → 大晋一般 | 外海一般 → 内海浓郁 |
| **主要敌人** | 妖兽 + 魔道修士 | 妖兽 + 海兽 + 散修仇杀 |
| **核心玩法** | 宗门任务、新手引导 | 海上冒险、深海探索 |
| **最危险区域** | 大晋皇陵（Lv8-10） | 星海核心（Lv8-10） |
| **特色资源** | 铁精、千年药材、符纸 | 炼晶、妖丹、深海灵物 |
