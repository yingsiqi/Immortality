## 与 L4 成长系统的衔接

| L4 系统 | 装备侧对应 |
|---------|-----------|
| §4 装备系统 | 装备品阶与玩家境界绑定（装备境界下限）；炼器师等级决定可炼制品阶上限；可成长装备随主人境界自动解锁属性 |
| §4.1 强化体系 | 可强化装备支持 +0~+15 强化，强化等级影响基础属性与特殊效果的触发概率 |
| §4.2 打造系统 | 所有可炼制装备均有固定配方（主要材料 2-4 种），材料来源涉及采矿、狩猎、采集三条产业链 |
| §4.3 法宝槽位 | 法宝类装备占用独立法宝槽（与防具/饰品槽不冲突），本命法宝位唯一 |

## 与 L3 战斗系统的衔接

| L3 系统 | 装备侧对应 |
|---------|-----------|
| §2.1 操作模型 | 4 法宝技能槽对应法宝类装备的主动法宝技能；武器类型影响普攻连段与特殊效果（如剑修破甲） |
| §2.3 伤害公式 | 武器攻击属性影响技能基础伤害；防具防御属性影响护体罡气计算；五行属性加成受装备五行影响 |
| §2.4 四大流派 | 剑修偏向飞剑类武器；法修偏向法珠/古宝；锻体偏向护甲/护盾；召唤偏向法宝控制类装备 |
| §2.5 战斗节奏 | 道器级装备的爆发属性可改变 PVP 战斗节奏预期（缩短击杀时间） |
