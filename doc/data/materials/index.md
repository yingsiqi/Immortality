# 采集物/材料表

> 基于 7 层规则书与原著世界观，定义游戏中所有采集物和基础材料的分类体系、品阶标准与具体数据。

---

## 分类总览

```
采集物/材料
├── 一、灵草植物    ← 炼丹主要来源
├── 二、矿石金属    ← 炼器主要来源
├── 三、灵兽材料    ← 妖兽掉落，炼器 + 炼丹兼用
└── 四、天地灵物    ← 灵气结晶 / 天地生成 / 特殊液体
```

### 分类对比

| 分类 | 获取方式 | 主要用途 | 可再生 | 典型示例 |
|------|---------|---------|-------|---------|
| 灵草植物 | 野外采集 / 洞府种植 | 炼丹药材 | 是（刷新 / 种植） | 黄精芝、霓裳草、九曲灵参 |
| 矿石金属 | 矿脉采集 / 拆解 | 炼器主料 | 否（储量制） | 铁精、庚精、魔髓钻 |
| 灵兽材料 | 击杀妖兽掉落 | 炼丹 + 炼器 + 特殊 | 是（妖兽刷新） | 妖丹、本命鳞片、碧鸠毒液 |
| 天地灵物 | 特殊环境 / 灵脉 / 奇遇 | 通用 + 货币（灵石） | 极慢或一次性 | 灵石、天雷子、醇液、化一神泥 |

## 品阶体系

全材料统一五阶制，与 L4 装备品阶、L7 区域等级对齐：

| 品阶 | 名称 | 对应区域等级 | 对应功法品阶 | 示例 |
|------|------|------------|------------|------|
| 1 | 凡品 | 凡域（0.3x~0.8x） | 黄级 | 止血草、铁精、鼠妖皮 |
| 2 | 下品 | 灵域（0.8x~1.5x） | 玄级 | 霓裳草、火熔晶、筑基期妖丹 |
| 3 | 中品 | 宝域（1.5x~3.0x） | 地级 | 黄精芝、紫精铜、结丹期妖丹 |
| 4 | 上品 | 秘境（3.0x~5.0x） | 天级 | 九曲灵参、庚精、元婴期妖丹 |
| 5 | 极品 | 绝地（5.0x±3.0x） | 圣/仙级 | 天雷竹、天晶碑、化神期妖丹 |

## 与规则层的关联

### L5 经济系统

| L5 规则 | 材料侧对应 |
|----------|-----------|
| §2 产出体系 | 灵草/矿石的自然刷新速率受灵脉浓度影响 |
| §3 灵石货币 | 灵石本身是第四类「天地灵物」，也是采集+消耗双用途 |
| §5 交易行 | 所有材料可上架交易行（基础物资类） |
| §6 商会 | 商会可发布材料收购任务，引导玩家到特定区域采集 |

### L7 世界系统

| L7 规则 | 材料侧对应 |
|----------|-----------|
| §3.1 九维资源向量 | 灵草密度、矿脉密度、灵兽密度三个维度直接决定材料产出 |
| §3.2 五行场域 | 每种场域倾向特定五行材料（火行场域多火熔晶/金阳芝；水行场域多寒髓/玄冰花） |
| §3.3 区域等级 | 凡域→绝地五级对应材料品阶上限（凡域最高产出凡品，绝地可出极品） |
| §5 灵脉 | 灵脉浓度每 +1.0x → 灵草刷新间隔 -20%、矿脉品质上限 +1 级 |

### L4 生活技能

| L4 技能 | 消耗材料 |
|----------|---------|
| 炼丹 | 灵草植物（主）+ 灵兽材料（辅）+ 天地灵物（催化） |
| 炼器 | 矿石金属（主）+ 灵兽材料（辅）+ 天地灵物（核心） |
| 灵宠培育 | 灵兽材料（食物/进化素材） |

## 数据表结构

### 统一字段规范

每项材料共用以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识，小写+下划线 |
| 名称 | string | 中文名 |
| 分类 | enum | 灵草/矿石/兽材/灵物 |
| 品阶 | int | 1~5 对应凡品~极品 |
| 五行 | enum | 金/木/水/火/土/无/异（注明具体） |
| 产地 | string | 区域等级 + 地貌类型（引用 L7 区域分类） |
| 产出密度 | int | 1~10，该材料在产地的丰富程度 |
| 刷新时间 | string | 采集后重新可采集时间（灵草）/ 不适用（矿石储量制） |
| 用途 | list | 炼丹/炼器/直接使用/任务/货币 |
| 关联配方 | list | 对应的丹药 id / 装备 id |
| 描述 | string | 原著背景 + 游戏内说明 |

## 编写原则

- **先易后难**：灵草植物最多但分类最清晰，优先写；天地灵物最杂，放最后
- **对标技能表**：每表独立 .md，以数据表格为主体，辅以分类说明
- **与原著对齐**：优先录入原著明确提及的材料，再补游戏原创（原创需标注）
- **留扩展空间**：品阶覆盖 1~5 阶但不填满，为后续版本新增留白
