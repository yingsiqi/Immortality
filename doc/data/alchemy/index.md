## 品阶体系

全丹药统一五阶制，与材料表、L4 装备品阶、L7 区域等级对齐：

| 品阶 | 名称 | 对应材料品阶 | 对应炼丹等级 | 丹药示例 |
|------|------|-------------|-------------|---------|
| 1 | 凡品 | 凡品 | 学徒 | 止血散、回气散、壮骨丹 |
| 2 | 下品 | 下品 | 初级 | 小还丹、聚气丹、火精丹 |
| 3 | 中品 | 中品 | 中级 | 黄龙丹、益气丹、铁骨丹 |

| L4 系统 | 丹药侧对应 |
|---------|-----------|
| §2 灵根与五行 | 五系 Buff 丹（火精丹/金锋丹等）受灵根纯度影响效果；突破丹受灵根品阶影响成功率 |

| §5 转世重修 | 天元丹、延寿丹等寿元类丹药为重修玩家提供时间窗口；洗髓丹可重置术法点 |
| §6 生活技能·炼丹 | 全部丹药的炼制工艺对应 L4 炼丹技能等级，宗师级可炼制极品丹药 |

| L5 规则 | 丹药侧对应 |
|---------|-----------|

| §6 商会与物价 | 商会定期发布丹药收购任务，引导炼丹师玩家定向生产；丹方价格受商会调控 |

## 编写原则

- **与原著对齐**：优先录入原著明确涉及的丹药（黄龙丹、大还丹、筑基丹等），标注「原著来源」或「原著扩展」
- **MMORPG 数值梯度**：高品丹药效果明显优于低品，但材料成本与炼制难度更高，形成合理的投入产出比
- **与材料表严格对齐**：所有材料引用使用材料表中真实存在的 ID 和名称
- **三文件互链**：本索引链向 pills.md 和 recipes.md；pills.md 和 recipes.md 通过产物 ID 互链
- **留扩展空间**：品阶覆盖 1~5 阶但不填满每阶数量，为后续版本新增丹药留白
