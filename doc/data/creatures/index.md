## 与规则书衔接说明

### 与 L7 §4（妖兽等级体系）的衔接

L7 §4 定义了妖兽的通用等级框架——一级妖兽对应炼气期，八级妖兽对应大乘期。本表中的怪物等级（1-10 级）直接映射至该体系：

| 怪物等级 | 对应妖兽等级 | 对应修士境界 |
|----------|-------------|-------------|
| 1-2 | 一~二级妖兽 | 炼气期 |
| 3 | 三级妖兽 | 结丹期 |
| 4-5 | 四~五级妖兽 | 元婴~化神期 |
| 6 | 六级妖兽 | 炼虚期 |
| 7-8 | 七~八级妖兽 | 合体~大乘期 |
| 9-10 | 超越八级 / 特殊存在 | 渡劫~真仙级 |

### 与 L3 §1（战斗规则）的衔接

L3 §1.2（操作模型）定义了战斗的实时 ARPG 框架。本表怪物行为 AI 字段直接对应 L3 §1.5（怪物行为）的状态机设计：

- **主动**：进入索敌范围即攻击 → 对应 L3 的「野外遇敌」模式
- **被动**：受击后反击 → 触发 L3 的仇恨系统
- **巡逻**：沿固定路线移动 → 可被玩家伏击（L3 §1「埋伏/追杀」模式）
- **伏击**：潜伏状态下首次攻击必暴击 → 联动 L3 §2.3 伤害公式中的会心倍率
- **召唤同伴**：血量低于 30% 时召唤 → 触发 L3 的增援机制
- **逃跑**：血量低于 10% 时脱离战斗 → 联动 L3 的追击/逃逸判定

怪物五行属性与 L3 §2.3 伤害公式中的五行克制系数（克 1.25 / 平 1.0 / 被克 0.75）直接联动。
