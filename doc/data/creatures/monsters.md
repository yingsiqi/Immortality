# 怪物数据表

> MMORPG 怪物数据定义，共 43 种怪物，6 大类，覆盖 1-10 级（对应炼气~大乘以上）。

---

## 字段说明

| 字段 | 说明 |
|------|------|
| id | `monster_xxx` |
| 名称 | 怪物中文名 |
| 等级 | 1~10（对应炼气~大乘，详见妖兽等级体系 L7 §4） |
| 分类 | 普通妖兽/精英/Boss/魔物/特殊/野生灵宠 |
| 五行 | 金/木/水/火/土/无/多属性 |
| 出现区域 | 越国/乱星海/大晋/秘境… |
| 刷新类型 | 固定/随机/条件/副本/Boss限时 |
| 刷新间隔 | 如「5分钟」「24小时」「击杀后不刷新」 |
| 基础属性 | 气血/攻击/防御/速度（相对值） |
| 掉落 | 材料ID + 概率（如 `beast_blood_mid (30%)`） |
| 技能 | 1-3 个技能描述 |
| 行为AI | 主动/被动/巡逻/伏击/召唤同伴/逃跑 |
| 社会关系 | 群居/独居/首领随从 |
| 备注 | 原著来源/游戏原创 |
