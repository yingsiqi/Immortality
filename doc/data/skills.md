# 技能表

> 基于 7 层规则书 L2 行动原子、L4 技能体系与功法体系，定义游戏内所有技能与功法的具体数值。

---

## 一、分类体系

### 1.1 总览

```
技能体系
├── 功法（被动养成）
│   ├── 主修功法 —— 1 个槽位，决定境界突破方向与属性成长曲线，100% 效果
│   ├── 辅修功法 —— 2 个槽位（结丹 +1 / 化神 +1），50% 效果
│   └── 机制：周天运转 → 后台 buff，影响三围、战斗属性、五行抗性、修炼速度
│
├── 主动技能（战斗释放）
│   ├── 攻击技能 —— 单体 / AOE / DoT
│   ├── 防御技能 —— 护盾 / 闪避 / 反伤
│   ├── 辅助技能 —— 治疗 / 增益 / 控制 / 位移
│   └── 神通 —— 技能树隐藏节点，需特殊事件/条件解锁（归入技能，不独立）
│
└── 被动技能（始终生效）
    └── 技能树节点解锁，常驻加成
```

### 1.2 与 L2 行动原子的关系

功法与技能在 L2 行动原子体系中对应以下原子：

| L2 原子 | 本表能力 | 说明 |
|---------|---------|------|
| §2.3 运转功法 | 功法切换、周天运转 | 主动操作，切换后进入 CD |
| §3.5 释放技能 | 主动技能使用 | 消耗真元 / 灵力 |
| §4.2 学习技能 | 技能学习与升级 | 消耗技能点 + 灵石 |
| §5.1 拜师 | 门派传承功法获取 | 宗门贡献兑换 |
| §5.4 传功 | 师徒功法熟练度转移 | L6 师徒系统 |

### 2.5 熟练度获取方式

| 方式 | 速率 | 说明 |

| 主动修炼 | 标准 | 在线时消耗真元运转功法，每秒 1 点 |

| 境界突破 | 一次性 | 突破时当前装备的功法获得 `境界层级 × 2000` 点 |

## 三、字段结构

### 3.1 功法条目字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识，小写+下划线 |
| 名称 | string | 功法中文名 |
| 品阶 | enum | 黄/玄/地/天/圣/仙 |
| 五行 | enum | 金/木/水/火/土/无/异（需注明具体异属性） |
| 来源 | string | 具体获取途径 |
| 获取条件 | string | 触发条件/任务/副本 |
| 境界门槛 | string | 最低修炼境界 |
| 灵根要求 | string | 灵根纯度下限，无属性="无" |
| 效果[6] | array | 6 级熟练度对应 buff 数组 |
| 隐藏被动 | object | 炉火纯青级解锁的被动技能 |
| 隐藏技能 | object | 出神入化级解锁的主动技能 |
| 相生功法 | array | 搭配后触发额外效果 |
| 冲突功法 | array | 不可同时装备 |
| 描述 | string | 原著背景介绍 |

### 3.2 主动/被动技能字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| 名称 | string | 技能中文名 |
| 类型 | enum | 主动 / 被动 |
| 分类 | enum | 攻击 / 防御 / 辅助 / 神通 |
| 所属流派 | string | 金剑 / 木灵 / 水月 / 火烈 / 土罡 / 通用 |
| 技能树层级 | enum | 基础 / 进阶 / 专家 / 大师 / 隐藏 |
| 学习条件 | string | 境界 + 技能点 + 前置技能 |
| 真元消耗 | number | 每次释放消耗（被动为 0） |
| 冷却时间 | number | 秒 |
| 效果 | string | 技能效果描述 + 数值公式 |
| 升级消耗 | string | 每级所需技能熟练度 |
