```
├── 功法（被动养成）
│   ├── 装备于槽位，周天运转 → 后台常驻 buff

│   └── 详见 → [功法](./gongfa)
│
└── 术法（主动释放）
    ├── 消耗真元/灵力，战斗中手动释放
    ├── 分为攻击、防御、辅助、控制、位移等子类

```

## 设计理念

- **功法 = 养成线**：装备即生效，决定"你是谁"——剑修、体修、法修、魔修
- **术法 = 操作线**：手动释放，决定"你怎么打"——远程、近战、控制、爆发
- **功法锁死流派，术法决定玩法**
