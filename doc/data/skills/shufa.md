# 术法

> 主动释放类战斗技能。战斗中消耗真元/灵力手动施放，决定"怎么打"。

---

## 一、流派体系

### 1.1 流派总览

基于 L4 §3.5 的流派策略，将术法按六个战斗流派组织，每个流派的技能树自成一系。

| 流派 | 属性倾向 | 定位 | 武器 | 普攻连段终结效果 |
|------|----------|------|------|------------------|
| **锻体流** | 土·体修 | 近战坦克 / 单体爆发 | 拳套 / 重斧 | 击退 + 硬直 0.5s |
| **剑修流** | 金·剑修 | 远程穿刺 / 破甲 DPS | 飞剑 | 破甲（无视 10% 防御） |
| **火烈流** | 火·法修 | AOE 爆发 / 暴击 | 法杖 / 灵幡 | 小范围爆炸（半径 2m） |
| **水月流** | 水·法修 | 控制 / 防御 / 隐匿 | 法珠 / 玉笛 | 减速 20%，持续 2s |
| **木灵流** | 木·法修 | 持续伤害 / 治疗 / 召唤辅助 | 灵木杖 / 药鼎 | 恢复自身 2% 最大生命 |
| **召唤流** | 无·御灵 | 灵宠大军 / 阵法 / 傀儡 | 灵幡 / 虫笛 | 灵宠追击目标 2s |

### 1.2 双修与隐藏流派

| 流派 | 解锁条件 | 说明 |
|------|----------|------|
| **双修流** | 两个流派技能树至少各解锁 5 个节点 | 可同时装备两个流派的部分术法，技能槽 +2 |
| **毒修流** | 暗灵根 ≥ 50 或完成「万毒窟」奇遇 | 毒/诅咒系术法，独立技能树 |
| **阵修流** | 完成「阵法大师」奇遇 | 阵法系术法（L7 世界系统联动） |

## 二、通用设计规则

### 2.1 技能树层级

```
基础层（入门即解锁）
  ├── 进阶层（筑基期 + 基础层已解锁 ≥ 3 节点）
  │     ├── 专家层（结丹期 + 进阶层已解锁 ≥ 3 节点）
  │     │     ├── 大师层（元婴期 + 专家层已解锁 ≥ 2 节点）
  │     │     │     └── 隐藏层（化神期 + 特定奇遇/条件）
```

### 2.2 术法等级

每个术法可升级 1~10 级（受境界限制）：

| 境界 | 术法等级上限 |
|------|------------|
| 炼气 | 5 |
| 筑基 | 10 |
| 结丹 | 15 |
| 元婴 | 20 |
| 化神 | 25 |
| 大乘及以上 | 30 |

每升 1 级：伤害/效果 +10%，真元消耗 -2%（上限 -30%）。

术法等级 N 所需熟练度 = 100 × (N ^ 1.8)

### 2.3 术法槽位（L4 §3.4a）

| 境界 | 主动槽位 | 被动槽位 |
|------|---------|----------|
| 炼气 | 3 | 1 |
| 筑基 | 4 | 1 |
| 结丹 | 5 | 2 |
| 元婴 | 6 | 3 |
| 化神 | 7 | 3 |
| 大乘 | 8 | 4 |

### 2.4 数值基准

所有伤害数值以「攻击力 × 系数」表示，便于随装备/境界动态缩放。

| 层级 | 伤害系数范围 | CD 范围 |
|------|------------|---------|
| 基础 | 0.8 ~ 1.5 | 5 ~ 15s |
| 进阶 | 1.2 ~ 2.5 | 10 ~ 30s |
| 专家 | 2.0 ~ 4.0 | 20 ~ 60s |
| 大师 | 3.0 ~ 6.0 | 40 ~ 120s |
| 隐藏 | 5.0 ~ 10.0 | 120 ~ 600s |
