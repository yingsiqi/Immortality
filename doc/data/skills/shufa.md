# 术法

> 主动释放类战斗技能。战斗中消耗真元/灵力手动施放，决定"怎么打"。

---

## 一、分类体系

### 1.1 总览

```
术法
├── 攻击术法
│   ├── 单体 —— 高伤害、高暴击
│   ├── AOE —— 范围伤害
│   └── DoT —— 持续伤害（灼烧、中毒、流血）
│
├── 防御术法
│   ├── 护盾 —— 吸收伤害
│   ├── 减伤 —— 百分比减免
│   └── 反伤 —— 反弹伤害
│
├── 辅助术法
│   ├── 治疗 —— 恢复生命/真元
│   ├── 增益 —— 临时属性提升
│   ├── 控制 —— 眩晕、禁锢、沉默
│   └── 位移 —— 闪避、瞬移、遁术
│
├── 神通（隐藏）
│   └── 功法出神入化级解锁的高阶术法
│
└── 禁制术法
    └── 探索/社交用术法（天眼术、传音术、控神术等）
```

### 1.2 五行流派

| 流派 | 属性 | 风格 | 代表术法 |
|------|------|------|----------|
| 金剑 | 金 | 高爆发、破甲 | 巨剑术、千刃术 |
| 木灵 | 木 | 持续伤害、回复 | 长春回春 |
| 水月 | 水 | 控制、防御、隐匿 | 冰冻术、水罩术 |
| 火烈 | 火 | AOE、暴击、灼烧 | 火弹术、泰阳焚天 |
| 土罡 | 土 | 护盾、反伤、控制 | 地刺术、流沙术 |
| 通用 | 无 | 人人可学 | 天眼术、御风决 |

### 1.3 术法学习方式

| 方式 | 说明 |
|------|------|
| 门派传授 | 入门后消耗技能点学习基础术法 |
| 技能树解锁 | 消耗技能点沿树逐层解锁 |
| 功法解锁 | 部分功法出神入化后自动获得专属术法 |
| 奇遇掉落 | 副本/任务概率获得稀有术法书 |
| 自创 | 高熟练度 + 感悟 消耗自创 |

## 二、字段结构

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| 名称 | string | 术法中文名 |
| 类型 | enum | 主动 / 被动 |
| 分类 | enum | 攻击 / 防御 / 辅助 / 神通 |
| 所属流派 | string | 金剑 / 木灵 / 水月 / 火烈 / 土罡 / 通用 |
| 技能树层级 | enum | 基础 / 进阶 / 专家 / 大师 / 隐藏 |
| 学习条件 | string | 境界 + 技能点 + 前置术法 |
| 真元消耗 | number | 每次释放消耗（被动为 0） |
| 冷却时间 | number | 秒 |
| 效果 | string | 术法效果描述 + 数值公式 |
| 升级消耗 | string | 每级所需技能熟练度 |
