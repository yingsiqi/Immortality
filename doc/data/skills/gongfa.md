# 功法

> 被动养成类核心系统。装备后提供常驻属性 buff，通过周天运转持续生效，决定角色修炼方向与战斗风格。

---

## 一、分类体系

### 1.1 槽位机制

| 槽位 | 解锁条件 | 效果系数 | 说明 |
|------|----------|----------|------|
| 主修功法 | 炼气一层 | 100% | 决定境界突破方向与属性成长曲线 |
| 辅修功法① | 筑基期 | 50% | 提供次要 buff |
| 辅修功法② | 结丹期 | 50% | 结丹后额外解锁 |
| 辅修功法③ | 化神期 | 50% | 化神后额外解锁 |

### 1.2 来源分类

| 来源 | 获取方式 | 实例 |
|------|----------|------|
| 入门赠送 | 角色创建 / 新手任务 | 引气诀 |
| 门派传承 | 宗门贡献兑换，逐层解锁 | 黄枫真解、月华心经 |
| 秘境奇遇 | 副本掉落 / 洞府遗宝 / 渡劫奖励 | 青元剑诀、大衍诀 |
| 拍卖交易 | 拍卖会竞拍 / 玩家交易 | 淬骨诀 |
| 自创融合 | 多本功法圆满后消耗感悟自创 | 梵圣真魔功 |
| 转世继承 | 转世后保留（需前世已修炼至融会贯通以上） | 全部 |

### 1.3 五行分类

| 五行 | 偏向 | 典型功法 |
|------|------|----------|
| 金 | 锋锐、破甲、攻击加成 | 青元剑诀、太白剑诀 |
| 木 | 生机、恢复、修炼速度 | 长春功、黄枫真解 |
| 水 | 柔韧、防御、控制、隐匿 | 月华心经、玄冰决 |

| 土 | 厚重、生命、护盾、肉身 | 淬骨诀、百脉炼宝诀 |

| 异属性 | 异灵根专属（雷/冰/风/暗） | 疾风九变、玄阴经 |

| 方式 | 速率 | 说明 |

| 主动修炼 | 标准 | 在线时消耗真元运转功法，每秒 1 点 |

## 二、字段结构

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识，小写+下划线 |
| 名称 | string | 功法中文名 |
| 品阶 | enum | 黄/玄/地/天/圣/仙 |
| 五行 | enum | 金/木/水/火/土/无/异（需注明具体异属性） |
| 来源 | string | 具体获取途径 |
| 获取条件 | string | 触发条件/任务/副本 |
| 境界门槛 | string | 最低修炼境界 |
| 灵根要求 | string | 灵根纯度下限，无属性="无" |
| 效果[6] | array | 6 级熟练度对应 buff 数组 |
| 隐藏被动 | object | 炉火纯青级解锁的被动技能 |
| 隐藏术法 | object | 出神入化级解锁的术法 |
| 相生功法 | array | 搭配后触发额外效果 |
| 冲突功法 | array | 不可同时装备 |
| 描述 | string | 原著背景介绍 |

## 三、原著技法归类分析

基于原著世界观文档 `design/world/cultivation/techniques`，原著中提及的技法按游戏体系应做如下归类：

### 3.1 归为功法（被动养成，装备槽位生效）

原著中的「XXX功」「XXX诀」「XXX经」「XXX大法」等自成修炼体系的功法，应归为功法：

| 原著名称 | 归入功法 | 原因 |
|----------|----------|------|
| 长春功、玄冰决、凝元功、归元功 | ✅ | 明确的修炼功法体系 |
| 正阳劲、象甲功、霸王甲、龟息功 | ✅ | 武学/炼体类功法 |
| 罗烟步 | ❌→术法 | 虽然原列功法区，但本质是身法术法 |
| 青元剑诀、太白剑诀 | ✅ | 完整的剑修功法体系 |
| 三转重元功、真阳决、天罗真功、浩然正气决、一气化三清 | ✅ | 正道顶级修炼功法 |
| 六极真魔功、天刹真魔功、托天魔功、玄阴大法、紫罗玄功 | ✅ | 魔道修炼功法 |
| 元磁神光、泰阳决、金真功、玉丹功、葵水魔功 | ✅ | 乱星海修炼功法 |
| 三梵圣功、化龙决、明王决 | ✅ | 妖族/佛门传承功法 |
| 血炼神光、黑煞修罗功 | ✅ | 玄阴经衍生功法 |
| 千浪决、柔风决、血车真诀 | ✅ | 功能性修炼功法 |
| 素女轮回功、化春决、缠玉决、青灵功 | ✅ | 特殊/女性修炼功法 |
| 逆情断天大法、冥河天尸决 | ✅ | 极端魔道/尸道功法 |
| 颠凤培元功、阴阳轮回决、引龙决 | ✅ | 双修功法 |
| 灵暝决 | ✅ | 预知感应类功法 |

### 3.2 归为术法（主动释放，需消耗真元）

原著中的「XX术」「XX咒」「XX步」「XX遁」等一次性释放的技能，应归为术法：

| 原著名称 | 归为术法 | 原因 |
|----------|----------|------|
| 火弹术、冰冻术、流沙术、地刺术、冰晶术 | ✅ | 五行攻击术法 |
| 天眼术、御风决、驱物术、传音术 | ✅ | 辅助术法 |
| 匿身术、敛气术、引气术 | ✅ | 隐匿术法 |
| 控神术、风缚之术、土牢术、水罩术、风墙术 | ✅ | 控制/防御术法 |
| 罗烟步 | ✅ | 身法术法 |
| 巨剑术、千刃术、眨眼剑法 | ✅ | 剑术术法 |
| 血影遁、木遁术、雷遁术、金遁术、化影遁、缩地术 | ✅ | 遁术 |
| 大衍决、惊神刺、失神刺、分魂术 | ✅ | 神识术法（大衍诀功法附带的术法） |
| 禁神术、封魂咒、惊魂咒、锁心咒、暴血咒 | ✅ | 禁制术法 |
| 阴魔斩、梦引术、离魂术 | ✅ | 魔道秘术 |
| 乾蓝冰焰、黑水冰焰、风离冰焰 | ✅ | 火焰神通 |
| 傀儡机关术、驱虫术、虫甲术、通宝决 | ✅ | 驱使术法 |
| 换形决、化劫大法、解体化形、巨大术 | ✅ | 炼体秘术 |
| 弄焰决、化火凝液 | ✅ | 火焰技巧术法 |

### 3.3 结论

原著中约 60% 的技法归为功法（被动养成），约 40% 归为术法（主动释放）。功法数据见下方完整列表，术法数据见 [术法](./shufa) 页面。
