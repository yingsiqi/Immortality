项目开发指南
开发流程
  1. 启动文档服务，访问 https://yingsiqi.github.io/Immortality/
  2. 在文档系统中记录和讨论游戏设计，包括：
    • 开发计划与路线图
    • 系统架构设计
    • 游戏系统（地图、时间、角色、战斗等）
    • 游戏设定集
  3. 完成文档内容后，根据文档开始实际开发
文档系统使用指南

VitePress文档系统特点：

  1. 访问 https://yingsiqi.github.io/Immortality/ 查看文档
  2. 直接编辑 doc 目录下的 Markdown 文件
  3. 支持热更新，修改文件后自动刷新页面
  4. 无需数据库，基于文件系统管理
提示： 所有文档内容都保存在 doc 目录中，直接编辑 .md 文件即可更新内容。
